# 三、项目流程

## 1.前期准备

抖音、快手、小红书、视频号等平台各两个，主要是抖音和快手。

一机一卡一号，不要一个手机下载一个抖音又下载一个极速版，这样是不给推流的

## 2、养号

前三天都是养号的时间段，主要就是养号，账号必须是新号，老号请注销，重新注册，注册过程请使用流量，注册好后请不要修改任何信息，保持原状态进行养号。

搜索：搭子、交友、游戏搭子等等进行观看、点赞、评论、关注

## 3、作品制作

养号三天后，也就是第四天就需要发作品了，下面是如何制作作品

准备设备：小红书、剪映、抖音、去水印

### 方法一：

第一步：抖音搜索搭子，把他们的视频原声全部收藏下来

第二步：打开小红书搜索【行车风景照片】

![](img/aefeca89d9e519da009446bd8bc80501.png)

第三步：用去水印下载原视频

第四部：导入剪映，添加原声，添加文字

这是我做的，做成这样发布抖音快手就可以了

每天2-3条，如果爆了，可以每天多发几条

### 方法二：

小红书搜索【搞怪动漫图案】

![](img/95e323d340fd2c8e9d3748c0989e16cb.png)

去水印 下载下来，添加剪映，添加原声和文章即可

两种方法都是比较热门，比较好上热门的素材
