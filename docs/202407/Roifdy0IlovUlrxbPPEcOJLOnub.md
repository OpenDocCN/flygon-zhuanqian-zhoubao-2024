# 开窍需要足够的刺激！一次熬夜发现的小商机！

一直忙着做项目，很久没有写这么长的内容了。

说起来也蛮有意思的，这个生意是我在一次熬夜的时候，意外发现的小商机！

我组了小团队从0开始模式，跑通以后到现在已经稳定盈利6个月了，业绩还有飙升的趋势。

熟悉我的圈友都知道，我擅长的业务是做小红书获客，21年就开始做了，一直做的都不错，服务过3000多位老板。

去年的11月份我刷到一位博主的短视频，她在镜头面前碎碎念，说自己打算放弃掉抖音10万粉丝的情感账号。

打算把这个几千粉丝的号好好运营起来！因为前段时间她经过测试，卖爆了一个脖套，业绩200万＋~~~

哇！我用考古家看完她的后台数据，瞬间就被刺激到了。这个爆发力也太强了吧..........

后头再看当成我发现的那位博主，她们现在已经是行业头部了！利润同样吓人~~

## 小商机是怎么发现的！

![](img/7e92a2a17e50e32066b63fb9ad937404.png)

![](img/ef4ede668d538f1d6343f216ee332ed8.png)

我是22年有在抖音做个人IP，差不多做到了3万多粉丝。我很清楚，抖音的流量和变现没有多大的关系。

而单月200万的业绩，靠直播的话，需要2-5个人才能配合着完成。

一个挂车的视频，可以做200万的业绩，利润还不错。这个不是暴利嘛？

而且还是有概率的暴利，基本功过关的话，拍100个视频就能爆2个【现在不行了】

22年的时候，我微博上有个互关的美食博主，叫，秒变食神。

她是做那种创意的小菜的，主打就是一个画面感，好看不好吃。

后面是怎么变现的内容，就是靠挂车，卖调料的。200多万粉丝的体量，每个月可以赚30万左右。

这个对比下来，就能感受到，这个短视频带货的暴利了。

那会做这个的也多，后面我觉得这个风口过了，没有想到的是呀。

23年的又起来了，如果不是这次熬夜意外发现这个博主的内容。

如果她没有讲到具体的业绩，我可能会和平常一样，动动手指就划过了。

但因为多停留了2分钟，这个项目到现在，给我带来了100万＋的净利润。

### 第1个模板，3秒可视化模板

*   这个模板是很好用的，文案第一句抓人很难，但用前面2秒抓人就简单多了，很适合新手。

一个产品足够猎奇 就要在开头想一个特别吸引眼球的前三秒

前三秒就能吸引人 牙刷 开头用的都是引起好奇 面部抖动弧度太大 日用品拍出不一样 ，用户就会继续往下看 三秒可视化模板 适合所有价格产品运用

![](img/dba2358ee5894fb79443d3dded29f8bb.png)

都知道 短视频停留数据特别重要， 前三秒留住了 对你有好奇 这个产品 。就能卖的动 三秒抓不住人 爆品在手也买不起来 但这个三秒怎么刷 怎么用 就要用卖点 或者是使用放大 来做到吸引眼球了

第一个爆款，找到的真的好巧合 ，是我文员刷一个博主 在那边卖牙膏

她说，发现那个s牙刷特别有吸引力，但通过后台看，是没多少人在卖的。

我和她讲了，她立刻找了商家寄样 ~~~看看时间线，她竟然是凌晨去对接的。

s牙刷拿到手之后我 我就在想怎么拍 能够吸引人

首先吧 是特别奇怪的品 那么开头一定要表现的特别震惊 ，给人那种从来没见过世面那种感觉

利用黄金三秒 两个夸张动作 吸引眼球 。我们拍的这个画面很夸张，有用户会好奇，这个牙刷为什么是竖着刷的。

然后文案是：我自己也没见过 拿出普通牙刷对比 让大家想到自己用的牙刷 点出普通牙刷到里面打圈的弊端 再介绍s牙刷卖点 竖着刷也好刷

提出第一次用牙刷的质疑 讲出设计这个牙刷原理 一套模板下来 打消人的疑虑 再总结其他卖点 让客户绝的猎奇 于是就能把货卖出去 这是我玩自然流量直播 运用到的勾子框架

最最后流量跑完，一条视频赚快10w 这个还蛮有意思的。

这个模板很好用，我们5月份又卖爆了一个品，单个品业绩100万＋！

### 第3个模板，三秒秀痛点模板

*   这个带货视频是怎么设计的呢？

就是文案和产同步，我们这品利润不错的。

视频结构是这样的：发明这个梳子的人，真的是配享太庙。

这句话讲完，第3秒钟，刚好梳子上的头发掉下来。

伸缩梳的卖点就是头发瞬间掉落， 梳头发的痛点就是要捡头发 ，这个就是痛点前置。这个画面设计的很好，品也爆了。

塑品的时候，洗梳子找到痛点后就能通过行为动作和语言 卖爆 只要有头发 就肯定需要梳子， 只要头发掉的多 就需要这品， 评论区保留网友整体性刺激评论的问题

就会有很多人在下面聊天 自然而然流量数据就做起来了 那么成交数据就会跟上了 适合所有抓人眼球日用品卖爆 或者只需要空镜展示痛点

![](img/3a0c632570a9c48c76ce805579d58558.png)

小白都可以模仿我这个模板 ，因为不用真人出境的。 难度不高 只需要一个简单的模板开头 就能够卖动 产品 更接地气。

最后我给大家讲一个偏方，效果也不错的。

就是很多人拍短视频，开始的时候会很有营销感，爆款我们开始也是这样的。

那么人在什么情况下，是不会有营销感呢？

就是在犯困和很想睡觉的时候。

就像那个让我拿到10万佣金的爆款，拍摄的时间实在凌晨，就是困的不行了。
