# 工具是什么

## 正常做

我们创建一个思维导图通常是，先给一段文本给gpt，然后转化成markdown格式，再将markdown的文字复制到xmind中进行编辑。这个你还需要找到对应的模板，然后去编辑，一行行的粘贴复制文字还要做边距对齐。这期间很耗费时间。

# 工具能做什么

## 内置了几种常用模板

### 第一种模板是常用的生财模版

这个也是做思维导图的新手模版，能够让你快速的实现一些简单的思维导图的样式，这种思维导图的样式对于一些简单的内容来说基本上已经可以了。我自己的常用模板之一。

这种模版的特点就是子标题不能太多，不然布局就乱了。

![](img/7ef19e630f774be400a37cb2576cf8e1.png)

### 第二种模版是复盘或者读书笔记

这类模板应对的是内容较多的情况，比如读书笔记章节很多，那就需要能容纳更多节点的模版。

这种模板的子节点可以达到七个，也就是书的目录后面还能在加上章节的总结提炼。

![](img/dacd4047f603c193ee61e61e3daf4935.png)

### 第三种自我介绍模板

这种模板是固定的，在用思维导图做自我介绍的时候，就可以直接使用这种模板。无需做任何修改。

这类模板的特点就是比较专一，不能进行扩展。

![](img/dedcdb1948c91f635cfab3af0dce17c3.png)

## 模板转换

我们有时候做出来常规的思维导图，觉得想换个样式，这个时候基本上做不到，官方也不提供模板转换，这种转换指的是将普通的思维导图，转换为竖屏思维导图。

比如：

![](img/2c117f26efe1e5c5047d79a2c8f0fca1.png)

![](img/0594f2e78598ab8f2b8320c6ece6d539.png)

目前这种转换基本上实现不了。

我这种能够实现的前提是你都是在这个系统上实现的，比如你是用markdown生成的普通思维导图，然后可以直接将markdown复制到对应的主题模板下面即可实现模板之间的转换。

# 未来提供的功能

## 直接进行文本输入

我们现在的做法是把要生成的内容先喂给gpt，然后让gpt给出对应的内容，但不是每一次内容都恰到好处，我们需要不断的去调整适配，这期间也是需要消耗时间的，所以就在调研怎么解决这方面的成本。

举个例子：

每个模板对于生成的markdown肯定是不一样的，通俗点说就是子标题的数目，有的多有的少，这方面肯定是需要花时间调整的。

如果这个问题解决了，那用户直接输入文本，就会和对应的模版结合形成最佳适配。

## 对文件的处理

大多数的信息源可能是视频，音频，ppt，word等等媒介，现在还是得找先转换文字版本，再转换成markdown格式，就和上面对文本的处理一样了。
