## (9 赞)Agent 是否可以创造出有商用价值的应用？

作者： 拔刀刘

日期：2024-05-24

大家好，我是拔刀刘，上次发的《不懂技术，我是如何用 Coze 手搓一个 Agent 来提升学习效率的》，得到了很多圈友关注，原文：

这段时间又手搓了 N 个 Agent，其中用自己照片生成学生照的 Agent，已经有 20+人体验了。

其实除了娱乐需求，有些场景还是有商业价值潜力的。比如：专属头像、生成证件照、电子写真、甚至是电商图，这些传统收费的应用，现在用 Agent 就可以免费实现。

我发几个图，大家感受一下。

对于标准化产品来说，虽有潜力，但是商业价值不大。价格由供需决定，你很难做出一款只有你才能做出来，其他人做不出来的 Agent。当市面有个产品开始赚钱，会吸引 N 多人杀入，产品供应量会增大，价格自然就会下降，最后大家都不赚钱。

在我看来，Agent 提供的最大价值，就是业务人员可以根据自己的需求，0 代码开发工具，提升效率，而不用麻烦技术。这类小的开发需求，是技术最不愿意接的，既体现不出水平又写不进去考核。往常我们只能靠自己堆工作量解决，现在在 AI 的加持下，流程化的工作可以让 Agent 解决。

在 Agent 出现之前，这类问题我都是通过 RPA 来解决的，但是上手难度来说，RPA 还是大于 Agent 的，效率可能也没有 Agent 高。

所以，当下 Agent 最大价值应该是：流程化、定制化、业务化的场景，这里头可能有商业机会。
