对于有编程经验的圈友来说，整套配置下来需要 2h；

对于无编程经验的圈友来说，可能需要 5h 以上；

（一定要逐字逐句按文档步骤配置）

整套脚本配置分为三个部分：

1.  Python 代码部分：爬取对标账号文章并使用 GPTAPI 生成文章、存入本地文件夹

2.  飞书部分：整套脚本的控制中心，包括各项参数调整、收益计算等

3.  影刀部分：控制悦互联浏览器，负责从本地文件夹将文章发布到公众号平台
