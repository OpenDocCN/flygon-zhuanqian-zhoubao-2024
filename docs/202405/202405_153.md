评论区：

牛高 : 你好，冒昧问一下：这个难道不是和淘客一样的？有什么区别呢
牛高 : 1.淘客有几百几千个微信群（店），一年也有几百上千万的收入。
2.新手做淘客，一个月也能赚几千块钱。
3.淘客的利润，一单也是几块到几十块。
干货哥 : 我们创业者关注三个点（核心竞争力）：
1.稳定的价格带来稳定的消费群体
2.稳定的利润带来稳定的创业群体
3.稳定的供应链带来稳定的现金流

以上这 3 点淘客这种模式给不了的。
牛高 : 价格、利润、供应链，这三者淘客都有的呀。
来对比一下：
淘客的价格都是商家为了销量搞促销，一般只会更便宜。
淘客的利润一单几块到几十块，不比这个低。
淘客的供应链，阿里巴巴是最大的供应链，根本不缺品，要什么有什么呀。
干货哥 : 几个问题：
1.你说淘客一年也有几百上千万的收入，有数据见证吗，如果有，这个概率有多少呢?
2.如果有那么高的收入，怎么解决财税安全问题?大部分都是三级分销平台，收入怎么做大的呢？
3.我提到的三个稳定，你真的理解吗?
淘客整合的是其他供应链，这个怎么可能有稳定的价格，稳定的利润呢
标品的背后是价格战，没有自己的核心定制款，光这个价格战问题都解决不了，就不可能有稳定的消费者和创业者，这才是底层的逻辑。
牛高 : 1.这个逻辑是按照你文章里写的逻辑的，如果上千上万家店。淘客几千个高质量宝妈群的话，收入只会更高。做过淘客的应该都知道这个数据是合理范围。
2.你文中提到一单只有几块钱，然而淘客出一单的话最低也有几块，高是几十甚至上百块。
3.你说的好像有一点牛头不对马嘴，淘客虽然是整合的供应链，但是利润明显更高，这个我刚才第 2 点已经说过了。

最后仁者见仁、智者见智吧，如果我上面说的语气有冒犯的地方，请见谅。我只是根据你的文章提出一些疑问。
牛高 : 再回答一下你说的第 2 点问题，淘客几百上千的收入，生财里至少有不下 5 个是这个收入，这个很好论证，可以翻一下身材之前的淘客精华贴。
干货哥 : 这个不见怪，就是交流一下。
如果两个对比，确实差异化太多了
比如：
1.平台对比，我们这个平台只做产品供应链，做到正品低价，量大价优，薄利多销，复购很高；
2.渠道对比，因为平台只做产品供应链，所以我们的客户和团队都在自己的手里，你塑造的是自己 IP，不是平台的强大，它是否强大其实跟我们没有关系的，我们又不是股东或者投资人。
3.我们是纯私域，所有的定制款只允许在自己的分销群里出售，客户要买东西只能找你，不能自助去下单哈，这样就跟你形成了非常强的粘性，那客户不是冲着价格而是冲着你的人设来的。
4.我们是现金流，我们的模式就是传统批零赚差价的实体生意，只不过互联网化了，客户找我们零售商买东西，零售商是先收钱扣除自己的利润然后把自己的进货价再去向批发商进货，所以每天卖多少单都是日结算，无需去平台提现，这就给了我们很多每天稳定现金流，可以足够我们支撑运营的开支了。
还有其他不考核不掉级等等差异。

我们的模式更加风险可控，

剩下就是快速开遍千城万店的问题，
