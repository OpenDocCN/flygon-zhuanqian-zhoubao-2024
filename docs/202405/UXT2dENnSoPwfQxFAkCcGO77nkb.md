# 二、设备如何选择

### 1.安卓

推荐买小米机型，小米8以上的机型，或者红米note9以上，因为小米手机刷机是最简单的，也可以进行账号的备份.

备份：

是指将真机的全部数据备份到电脑上进行养号，等需要使用的时候再恢复下来的一个方法。

比如你刚登录上的一个新号，如果你想要打创业的内容，直接发那肯定是容易违规，这个时候，你把他备份到电脑里面，一个月后再恢复到手机里面来。

这样的话你这个号就是一个养了一个月的号了。它不会占用你的手机，不然如果所有号都在手机上面养的话人效特别低。

现在备份玩法也越来越没那么神乎了，所以保险起见，备份时间越久越好，一般小米8备份一个月以上当老号用，小米11，也建议15天以上。设备越好，备份时间越短。

所有备份出来的号，恢复回来去打的时候，都建议你拿真机或者电脑比特去养号半天左右。

适合小团队工具截流，或者做自热

### 2.苹果：

如果是苹果手机的话，推荐买苹果7p以上，什么渠道都行，但是切记，一定要问清楚，能不能刷机还原，封了很多次号后，就需要刷机，不然就会封设备，不能刷机还原的手机会便宜很多。

电商渠道上很多店铺他也不会和你说，不能刷机还原的话，你这个手机如果封设备了，基本上就死了，一个手机死几个号就废了。

适合做自热的团队

*   自热：自己在平台上发内容靠纯自然流来引流

*   截流：在别人的内容下去引流

适合做自热的团队

# 二、账号如何选择

然后再说账号，账号的种类太多了，每一种我都详细说明一下，方便大家来选择。

### 1.接码平台1毛号

这种号，基本上没有人用，除非你确实玩的特别的明白，这种号大部分都是虚拟号，最垃圾的号。

但是胜在便宜，也有人拿这种号来做截流，但是非常的糟心，实在没必要。截流也大部分不出评论，点关注也没有反应。（你引流的动作别人看不到，被平台屏蔽了）

截流和自热都不适合

### 2.四方号

就是指用四方接码平台接的厅卡号，一般接码的时效是3天左右，在这几天内可以重复接码上号。

可以绑定微博或者QQ号来无限死号重开，一般一个号顶3-4次都没问题，价格在2块钱到4块钱之间。

这种账号用来做截流就好，别拿来做自热，偶尔哪怕能过推广，也是运气好，总之就不要用。

适合做截流

### 3.拦截号（仅作科普，不准使用，违法）

就是别人不小心给某些软件了短信权限，或者街上乱扫别人的二维码导致手机的短信被拦截，然后通过拦截平台接码使用的号。这种号只能收一次验证码，如果退出登陆了的话，号就没法找回了，价格在5-7块钱之间

拦截号和厅卡号地推号一样，只不过地推号是当面用他的号，拦截号是拦截了别人的短信

拦截号我们在拦截号平台上可以筛选，但是筛选是需要钱的，大概是4-5分钱筛一次，现在市面上拦截老号很少了，很多平台筛一次老号，筛号费用都要好几块钱，折合下来一个拦截老号也差不多10元。我们筛选出来的老号可以用来做高危类目的自热。

很多做贷款，复刻，国学，创业赛道的都是用的拦截号，这种号新号随时都有号，长期专业做小红书打粉的基本上都是自己备份养号，但是直接买老号肯定更好但成本高。

适合做自热

### 4.地推号

最贵的，但是很多人拿其他各种垃圾号来凑数，切记地推号只有正常时间有，如果找人买地推号他凌晨两点给你接码上号的，那肯定是拿其他号骗你是地推

*   如何判断是否真地推号？

直接打个电话过去，看有没有人接。

地推号的买号渠道没有平台，一般都是找地推团队，或者地推的群。地推新号可以直接打低危的赛道和模板，如果要打创业这类的赛道基本上都是高危，建议用老号，或者自己备份养号

### 5.外国号，香港号

用的最多的就是美国老号，这种号价格很低，基本上都是3-5块钱这个样子，但是现在市面上有了29关注的美国号，和做过点赞评论等业务的美国号，价格比突破老美号贵一点，但是确实会好一点，但是实际上也好的不多。

安卓手机就不建议用美国老号了，打美老的几乎全部都是苹果手机，安卓手机操作特别麻烦，听人劝吃饱饭

打外国号，玩法就是电脑挂上vpn，然后买外国节点，挂到比特指纹浏览器上面去，用电脑发布作品（这篇主要是讲打粉准备工作的，大家关注一下我，后续会更新内容出来）

小白做自热用外国号比较麻烦，但是也不难，可以学，我这有份教程，需要的话给大家。外国号和港号基本都是打高危类目和模板的专业团队在用。

外国，香港老号适合做自热

# 三、网络环境如何选择

网络环境这里其实可以分出来手机号卡和IP地址两大类，但是除了注册卡外，任何手机卡，如果你用它的流量的话，其实也就是自带一个IP地址了，但怕大家混淆，我还是拆分开来讲。

## 1.手机卡

*   厅卡

厅卡就是我们在线下营业厅办的卡，IP是干净的，适合用来正儿八经做内容涨粉长期运营。

一个人还只能办理15张。也就是对应15个账号，当然可以用微博，qq等第三方平台顶号，注册新的，但是说到底，也就是有限的账号数量

PS：团队的话，可拿营业执照叫运营商上门来办理，一次性可以办满，如果是去营业厅办理的话只能1张1张办

*   流量卡

流量卡就是网上办理的那种19元，29元，1.2百g流量的那种大流量卡，和厅卡区别不大，也是适合用来正儿八经做内容涨粉长期运营。但是流量卡线上只能办理10张，10张以后只能到营业厅去办理厅卡。

PS：不建议办理广电的手机卡，注册的账号权重天生比三网（移动联通电信）低

*   注册卡（绑定）

注册卡顾名思义就是用来收验证码注册各类账号的卡，也就是俗称的虚拟卡，它不能接打电话和用流量，一般都是工作室用来注册软件，或者绑定手机号的。

PS:很多注册卡是不能注册微信的，如果你要注册微信，记得提前问一问能否注册

*   物联卡

物联卡就是几年前就很流行的只能用流量，不能接打电话和发短信的纯流量卡，很多人也叫他流量卡。

一般是19.9-60g，29.9-100g这个价格，如果比这个价格便宜太多的话，一般都是有虚量，实际上没有那么多g的。

1个人可以办10张，后续基本不会跳人脸，也可以买实名好了的会贵一点

PS:物联卡分为普通物联卡和能跳IP的物联卡，适合低危类目自热和截流

# 2.IP地址

*   厅卡的流量

厅卡很多人都说是静态IP，但是据我们团队测试，其实发现厅卡的IP在开关飞行模式后也是会跳的，只不过他只是动xx.xx.aa.aa，地址中间，aa的部分，也就是后面一半，所以说很多人说不要用厅卡，要用全国跳，其实我是持着反对意见的，其实厅卡真的可以用

但不要被卖全国跳的人骗了，告诉你说只有全国跳才能跳IP，厅卡也可以

办理地址直接甩出来点我办理厅卡

万能，怎么用都行

*   WiFi

连上WiFi，lp地址就变不了了，都是同一个。

适合做内容，不玩废号

*   物联卡流量

物联卡这里很容易踩坑，因为物联卡分能跳IP的和不能跳IP的，不能跳IP的物联卡买回来IP也是同一个，还不如用流量卡

能跳IP的的物联卡分全国跳和省内跳，自然是全国跳是更好的，如果需要买卡可以先问清楚

## 如何跳IP？

插上手机卡，激活充钱后，确保能够正常使用流量，然后开关飞行模式，你的IP就跳了，有些差一点的物联卡IP可能会跳不动，多开关几次即可。

开关飞行模式后，可以在这个网址查一下是否变了IP地址：ipinfo.io

PS:但是现在随着用物联卡的人越来越多，物联卡的IP地址其实也没那么好了。如果是打低危的赛道用全国跳基本没啥问题，但是如果是打创业，国学这种高危的赛道的话，那还是建议买下面这种纯净的IP。

适合做低危类目自热以及截流

*   买IP节点

如果以上的网络环境都不能满足你的要求的话，那就建议你直接来买IP节点。我这边对接了一个比较不错的渠道，如果需要的话，微信89418，可以找我对接

截流团队可以买共享的IP节点，它比较便宜。但是如果你是做自热的团队的话，那还是建议买独享的纯净IP节点，这样的话你的网络这一块就一定没有问题，如果账号不过推广，没有流量的话，那就只能是账号，设备，模板这三个其中之一出了问题。你找问题也更简单

如果你是用国内号的话，就买国内的IP节点。如果你是用的外国号就买外国IP节点

*   软路由

具体的解释就自行百度吧，只要知道它软路由适合大型的团队，它能够把一个工作场地开出几十个WiFi，也就是几十个不同的IP地址

# 四、强调部分

## 1.具体情况具体分析

其实以上说了这么多，很多地方可能会说的比较绝对，但是只是考虑到这篇文章可能会有很多刚入行的朋友在看，我如果不说的绝对一点，肯定很多人就会有疑问，到时候又在设备啊网络啊帐号啊这些事情上耽误太多的时间。

就比如我在上面说，苹果手机用苹果7p以上，但是真的就一定是7p吗？5s不行吗，可以的。但是你设备太差了，小红书又不是傻子，你拿苹果5s在发内容，大概率就是不会推流

设备越高级越好，账号越老越好，IP越干净越好，它没有一个度说，低于这个档次就是不行，很多人用私人老号，连WiFi照样能打出流量

但是秉承着真诚利他的角度来说废话，还是希望大家不要老在准备工作上花太多的时间。干起来，才有结果，想起来都是问题

## 2.出现问题学会筛查

如果你发的内容没有过推广，那么你就要知道肯定是设备，网络，账号，模板这四要素出了问题

设备上号前是否双清刷机了？

同一个IP下是不是设备太多被检测到了？

买的IP是不是不纯净？

账号是不是之前有过违规？

养号有没有养好？

模板是不是淘汰了，要换？

自热批量克隆软件是不是没调试好？

等等......

这也是为什么我上面说设备越高级越好，账号越老越好，IP越干净越好，你拿着苹果15发的内容，IP地址是买的独享纯净节点，账号也是拦截老号，发个随拍能过推广，那最后得出结论就是你的模板有问题，那就换一个模板
