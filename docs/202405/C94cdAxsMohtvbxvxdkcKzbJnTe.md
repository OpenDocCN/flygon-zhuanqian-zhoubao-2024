# 如何破圈？我目睹大佬10位数余额背后的故事！

> 来源：[https://w91h2gmtrn.feishu.cn/docx/C94cdAxsMohtvbxvxdkcKzbJnTe](https://w91h2gmtrn.feishu.cn/docx/C94cdAxsMohtvbxvxdkcKzbJnTe)

大家好，我是乔帮主，今天跟大家聊一个破圈的话题。最近在上海，跟一个大佬聊AI+营销解决方案，用AI来做艺术类IP内容提高影响力。

大佬说，他不缺钱，每天赚的钱都用不完。这不是妥妥的凡尔赛嘛，我上一次听这话的时候，还是在抖音里看到马云在撒贝宁面前一本正经的说的，结果撒贝宁忍住憋笑硬是在抖音上憋了几年。

当大佬把他基金账户余额点开给我看的时候，冲击和震撼感挺强烈的。6开头，9位数的余额，还是美元。合计40亿的人民币现金流，10位数的余额。

以前我做技术总监的时候，也接触过很多企业老板、企业高管，其实遇到的亿级别身价的老板的也挺多的。但大多都是身价和资产，而见到几十亿余额的，这还是首次见，挺震撼的。

接着大佬说，现在组建了一个13人的团队，在搞IP在探索这个事情。招的人平均月薪开到了10万，最低的月薪也有6万。听到这话，我瞬间不淡定了，有种很想跟大佬打工的冲动。

而且大佬搞这事，是因为对艺术类纯属兴趣爱好，只是闲着想在这方面做些事情，并没指望靠这个赚钱吃饭。说到这，真是应了那句，我们要赚有钱人零花钱，不赚穷人的吃饭钱！

而关键是，我们要怎么遇到这样的大佬，去赚他们的零花钱呢？接下来跟大家聊聊如何破圈获取优质人脉和资源，这背后的底层逻辑和可执行的路径。

所谓圈子的重要性，这个认知问题，不需要我过多强调了。就如同我前面说的大佬，当你身边认识的人都是这种级别的时候，是不缺商业机会的，赚钱也只是时间问题。

但我遇到很多人，包括我自己，经常在自己圈层待久了，自己也就麻木了。当你想去做个什么事情，或者想去创业，你会发现，身边的亲朋好友都在劝你不要做“傻事”。他们没办法说服你，同样你也没办法说服他们。

你的圈层，决定了你身边的人的认知、行为想法的层级。所以当你尝试想去做另外圈层的事情的时候，对他们来说，不理解，也很难理解，也很难接受，最终大多数情况是谁也说服不了谁。

所以我经常跟很多人说，当你有个想法的时候，最好别跟自己身边圈层的人交流。最好跟一些比你认知维度高的，比如去跟投资人去聊聊你的想法，哪怕别人否定你，也会在一些关键点上给到你一些实质性建议。

而所谓遇贵人，也是多半由于机缘巧合遇到一个不属于你圈层的大佬，正好大佬帮了你，改变了你人生轨迹。我们常说的破圈破圈，就是为了去遇见更多的人脉和资源，遇见更多机会，当然这里面很有可能会遇到你的贵人。

那如何破圈？之前我在一篇文章《付费思维，让你人生弯道超车的5条捷径》中提到过，付费是破圈及积累人脉最快的捷径，这也是你触达你自己圈层以外圈层的最好方式。

付费链接，这是我们破圈的第一种方式。

要触达大佬，是破圈、混圈及向上社交的第一步。如果你都没触达，连微信都没加，都没产生交集，那什么都是空谈。只有链接了，才会有可能。都没链接上，那什么可能都没有。

付费混圈链接，更多像是一种花钱投流的营销模式。虽然链接上了，但是距离真正破圈还是有一定距离的。因为这时候，你认识大佬，但大佬不认识你，更别谈让大佬心甘情愿把他零花钱给到你，或者更别谈大佬愿意帮助你、提拔你等春秋大梦了。

除了付费混圈，还有一种就是付费请大佬为你服务。大佬的私教课，训练营，几万、几十万的你都报名，为大佬的时间付费，进而让大佬的资源为你所用，这是一种极其简单、直接高效的破圈方法。

付费请大佬为你服务，这是我们破圈的第二种方式。

如果有这些钱，那问题就很简单了。但钱都不是大风刮来的，这对我们很多普通人来说不现实。所以核心还是回到本质上，你能为大佬提供什么价值？

为大佬提供价值，这是我们破圈的第三种方式，这也是破圈的本质。

很多时候，我们都是想去认识比自己厉害的人，为了去解决自己的某方面的问题或者困惑，但很少有人知道，别人凭什么要帮你解决，而你又能给他带来什么价值，这更多是大佬真正所关注的。别人不在乎你所在乎的，别人只在乎他所在乎的。

那究竟如何跟大佬提供价值？接下来展开聊聊圈层的逻辑，主要分学历、职位、家境、颜值这四条线：

这四条线中的每个级别都代表着不同的圈层，所谓的破圈的逻辑，其实都是想往上走，认识比你级别高的人。比如学历线，往上想认识博士、想认识博士后。

比如在职位线，想认识总裁、董事长、大佬。再比如在家境线，肯定是想认识家境好的，而颜值线，肯定都想认识好看的等等。

对很多人而言，普通学历、普通职位、普通家境、普通颜值。那究竟要如何跟大佬提供价值破圈？很多人说要靠情商，会做人，会跟大佬提供情绪价值。

比如，我认识一个年轻人，把大佬公众号文章全部看一遍，然后每一篇都写几百字的感悟心得，这情绪价值提供的满满的，不过我认为这只是外在“招式”。

解决问题，还是要解决根本。破圈的本质，还是得看自身的能力。说的更具体点，普通人想在学历、职位、家境、颜值这四条线破圈成功，更多要靠专业能力。

比如一个中专毕业的一个设计师，特别会设计别墅，然后就会有很多大佬、博士、明星来找他设计别墅。比如Tony老师设计发型特别厉害，自然很多有头有脸来找他设计发型。

比如，我以前在阿里云做云架构师，经常跟一些明细电商公司、及国企政府单位做云架构方案，自然会认识一些明星、企业老板、政府领导等。再比如律师，医生，教师等本质也都一样。

所以只有某一领域专业能力牛了，你才可以无视学历、可以无视职场关系、可以无视家境是否好坏、可以无视颜值等。往往专业能力牛的人，可以无视很多规则，哪怕他们不怎么社交，却依然有很多大佬找他。

所以求人不如求己，努力提升自己，做一个对他人有价值，这才是破圈的本质！
