法律咨询项目：5 人小组单月变现 50 万（从 0 到 1 全流程分享）

5 人小组，三个月，从 0 到 50 万。成绩不算很好，但逻辑清晰，希望能给你做项目带来一些思考。

首先感谢生财，干这个项目也是因为去年在社群里看到圈友   发了一篇法律赛道维权类的文章。哇，还能这样玩。刚好那时候我公司的直播带货业务处在放弃的边缘，我主动联系学习了下收获满满，又发动身边的资源拜访请教郑州的律所、法律咨询同行，每天就是付费、拜访学习，周期大概一个月左右，我觉得项目能干、公司搞直播运营的这几个同事能力也匹配（因为门槛看似高、实际就那回事），就开始搞了。

我看社群里目前还没人分享，我就斗胆献丑，把我们团队法律项目压箱底的实操方法分享给大家，希望能帮助到圈友们，给大家提供一些思路。

公司隔壁楼上就是全国互联网律所的头部，毫不夸张应该是经济纠纷赛道的第一，一年几个小目标。 我们小卡拉米团队就不晒数据了，有点寒颤，直接给大家上实操。

做得是细分赛道——维权退费，我们小团队主要做的是经济纠纷赛道（民间借贷、货款、工程款等），市场太大了，拿出来一个小小的细分赛道就是非常大的市场。

我坚信，利他是极致的利己。如果感觉有收获记得给我点个赞，哈哈。
