## (386 赞)普通人、创业者和投资人的一些思维差别

作者： 亦仁

日期：2024-04-24

2.  聊到了普通人、创业者和投资人的一些思维差别

越是赚到大钱的：

对不确定性的忍受度越高，对风险的忍受度很高，甚至是主动去拥抱和创造不确定性。

开放度非常高，非常积极主动的学习和分享，很容易认可别人而不是证明自己是对的。

对自己时间价值的判断非常高，能花钱解决的尽量不花时间，不在省钱上浪费时间，绝大多数事都是找找专业的人来干，即使自己可以干也找别人来干。

拼图思维，整合思维，尽量减少自己从 0 到 1，但是支持和鼓励更多人从 0 到 1，把各种资源整合思考创造新的价值。

尽量合作共赢而不是单打独斗，大胆的分钱，不怕被占便宜，敢于吃亏。很多刚起步的人，非常害怕合作，总是想自己一个人，或者自己的团队干所有事，结果越干越小。
