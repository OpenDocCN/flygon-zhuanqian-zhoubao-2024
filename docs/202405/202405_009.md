日期：2024-05-23

思无邪与正心诚意

昨天发了关于得到的内容，其中提到的成甲老师下午私信我一段话，大概意思是，当年我推荐他去得到讲书，成为他人生中一个重大的转折点。后来许多重要的人生节点，也多少与这次推荐有关。但他一直对我没有什么回报，内心很惭愧。

当时推荐成甲给得到，我并没有想到过从他那里获得什么回报，也没有想过有什么利益交换。我只是单纯地觉得，成甲是得到需要的那种讲书老师，这件事对得到有益处，我就推荐了，如果没有益处，即使是我的挚爱亲朋，我也不会推荐。

这就是所谓正心诚意，你做这件事只是觉得该做这件事，而不是想了许多利益关系。

比如有些咨询公司，他们可能跟某些媒体签了合作协议，如果从他这里卖出去广告，他们可以获得一部分佣金。那他在给客户推荐广告媒体的时候，很可能就会优先推荐这个媒体。

这个推荐，其实就有利益计算在里面了。

昨天有某 IP 负责商务合作的人给我了一些合作资源，我转头去问了一个客户，他们很感兴趣，我说我给你拉群你们聊。对方其实还是有点吃惊，他说你可以走你们公司的业务，我明白他的意思，就是我们可以获得一部分佣金。但我推荐给客户，是真心觉得这个东西对他们有好处，而不是为了佣金。

这才是正心诚意，你收了客户咨询的钱，你就真心为了客户好，你做的所有的动作，都应该是为了帮助客户，而不是别的。

我想到另一个词，叫思无邪。

孔子评论《诗经》时说：
《诗》三百，一言以蔽之，曰‘思无邪’。

大概意思是说，诗经的诗篇，都是真心话，没有惺惺作态的东西。

做一家公司，你真心希望员工好，和你为了让员工帮你赚钱而对他们好，那又是两回事。你真心对待朋友，和你因为他有利用价值而交往，那也是两回事。

很多人嘴里说着正心诚意，心里却满满的利益算计，这就走偏了。你可以说我们就是利益交换，互惠互利，那也比口是心非好。

真心诚意地的对待客户，员工，合作伙伴，以及朋友。

这才是思无邪，正心诚意。
