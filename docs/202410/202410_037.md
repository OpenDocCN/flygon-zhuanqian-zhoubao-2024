## (28 赞)想写好 IP 人设文可以从写日记、写回忆录开始

作者： 漫漫 mansi

日期：2024-10-23

hi 大家好，我是漫漫～是一个小红书 34 万关注的博主，也有自己的公众号、视频号，也开始创业做运营项目和教育机构啦。之前发过很多小红书 IP 创作、创业、变现的内容，但我最有力的，还是内容，我的小红书关注只有 34 万，但内容的点赞收藏量有 303 万哦～

我特别擅长写 IP 的文章，今天想和大家分享一下，我认为 IP 文最重要的一点：**IP 文最重要的是讲好自己的心情、感受、观点。**

**IP 文的目的，其实是让人认识你、了解你，从而喜欢你。**

很多人设文章虽然需要有一些结构，但就是常用的总分总、时间顺序就行了，

IP 文不需要太华丽的词藻，但需要**精确** ，你需要把你的**感受清晰的表达，是怒还是气，是烦还是燥的情绪** 说出来；讲故事的时候要把当时你做的事情**客观地还原，提炼** 一下**xx 时间做了 xx 事情。**

很多人就说，IP 文很难写呀，很难带着情绪地去讲故事，刻意地往里面加情绪很做作。

所以呀，其实咱们就把心态扭转一下，**你不需要去编故事，你要做的是把自己的故事讲好。**

我给大家一些可以讲成故事的话题，大家可以思考个 30 秒：

（1）你觉得自己今年过得最开心的是哪天？

（2）你觉得前两年过得最开心的是哪天？

（3）你遇见的印象最深刻的老师是哪个？

（4）你父母说过那一句最让你感动的话？

（5）哪个陌生人给过让你感动的鼓励？

听上去感觉是很无聊的一问一答对吗？

都是我曾经热门爆款的切入点。

不过我的那些内容是这样写的：

《我把一群优秀的人聚集到了一起，我好高兴！！》，讲的是我做了我的社群和产品，让很多优秀的人在这里互相学习互相鼓励

《我所有的自卑，都终结在了在月入 4 万的那个月。》

《最好的学习方法，老师们早就教过了》，写的是现在很多人急于求成，疯狂寻找更好的学习方法，学习学习这件事，而不是学习知识。

《我的父母把我教成这样，不是让我为了男人伤心》，写的是分手之后走出失恋的故事。

《我收到了粉丝的来信》，写的是我做自媒体的发心。

都是大爆款哦。

你们看，其实**去捕捉自己生命中最有感触的一些瞬间最有意义** ，因为你是**实实在在有真实的感受，有真实的触动，所以绝对能写出真实的文字、还原真实的细节、描述出真实的情绪** 。

**想写好 IP 人设文可以从写日记、写回忆录开始。** 把你过去那些最打动你自己的经历写下来，你的开心、震撼、触动、改变、烦恼、清晰、感动，都写下来。哪怕只是写一句话，写 50 个字都行。

（因为我最初做小红书其实也是从写日记开始的，我以前发过复盘文章讲过我自己是不小心火的，当时就是考过雅思之后，发了一篇上岸成功的兴奋笔记，然后很多人让我出攻略，我就复盘经历，讲我之前备考多么焦虑、考试没钱怎么攒钱等等，后来上学期间也是当日记写呀，写我每天学了什么、为什么学，我非常热爱学习、想要进步、也不断在打怪升级，日记/小红书当时就是我的实实在在的记录，人人都能看得到，既真实，又有力，大家当然关注我啦。）

那么我想再给大家一些思考的空间，你可以先思考这些问题，然后慢慢地就会有很多灵感～

（1）引起你情绪的事情都有哪些？

你平时最多的情绪是什么？悲伤？焦虑？忧愁？沮丧？还是感到幸福？温暖？

对应这些情绪的，平时你都是发生什么事情会产生这些情绪

（2）通过你在意的人

你身边的好朋友让你最不满的地方，你父母让你最不满的地方，你的另一半让你最不满的地方。

你为什么会有不满，你觉得的应该是怎样的。

试着幻想着给它们写一段简短的表达你的观点，尝试去说服他们的文字。

（3）通过你在意的人

你最喜欢的你的好朋友的闪光点，你能从父母那儿获得的温暖，你的另一半带给你的快乐，

试着去告诉全世界，我好爱这些特质，因为他们也让我过得更好。

IP 的故事文，其实是表达性子写作。

目的是：抒发你的真实情绪、表达你的观点、你的感受、你的思想。

所以要写好的 IP 文，第一步是先把你真实的感受、心里想说的话、真正认可的东西吐露出来。

尝试按照我上面这些切入点，先写写你自己的故事吧～

如果不想被人看到也没关系的～我在自媒体上也几乎不露脸，也还是做到了大家记得“漫漫”这个名字～
