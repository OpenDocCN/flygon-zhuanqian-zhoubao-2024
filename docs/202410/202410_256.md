## (6 赞)参加首届知识 IP 品牌节，我得到了什么？

作者： 天堂地狱

日期：2024-09-24

对于这次的 IP 品牌节，我看群里的评价褒贬不一，有说颇有收获的，也有说 IP 老师的咖位都不错，但是没有干货，尤其是听过老师单独小场讲解的，觉得老师没有拿出实力来讲。

我觉得大家说的其实没毛病，也很理解，这样的大场，如果足够干，时间上估计就要 3 天起。

也许是期待太高的原因，我自己其实在听的时候也是有这种感觉的，有些水。

但是回来以后仔细回味，我发现每个老师都在试图“藏私”的情况，又抱着真诚分享的态度，试图给我们讲清楚 IP 创业过程中的至少一个或多个**“核心价值点”。**

**我个人认为，把这些听懂了，融会贯通了，悟透了，就值回票价了。**

就好像武林大会，武当掌门，少林掌门，都出来给大家拆解一个高招，那就够了，这种场肯定不能给你来一套镇派秘籍把？

如果真想深度学习，我们去尝试链接老师就是了，毕竟都是知识付费相关的老师，不难链接。

好，别的不多说，那既然是一招半式，那么去掉介绍和暖场，总结起来应该也不长，我就尝试根据我的理解给大家拆解下，看看能不能让大家感受到，我感受到的价值。

一来自己也复习一遍，二来给没有参加的朋友分享一些收获。

**下面，我用五个部分，来讲下我对五位老师分享内容的理解，分别是：**

1.  1\. 赫畅老师的三个认知+一个隐藏大招。
2.  2\. 张晓楠老师的一套心法:"别瞎花钱，别下牌桌"。
3.  3\. 张琦老师的"多元讲解"做 IP。
4.  4\. 崔磊老师的“矛盾论应用”做内容。
5.  5\. 吴世春老师，投资人中的段子手。

如果有说的不对的地方，请狠狠的喷，我来学习改正。
