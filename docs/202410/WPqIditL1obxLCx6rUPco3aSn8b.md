# 初入生财

3月28号，加入生财，之后一直处在很兴奋的阶段，

期待着航海的到来，每天做着自己的发财梦。

中间啥也没干，就偶尔刷个贴，

觉得自己好牛逼。

# 牛刀小试

7月航海，如期而至！！！

因为新手群完成任务，可以免保证金参加一个项目，

本着便宜不占白不占。

选了最贵的视频号带货，又交199参加了闲鱼高客单

怕自己精力有限搞不定两个，专门把闲鱼手册好好做了功课。

是的，牛逼如我~

闲鱼两个号，都在第一天出单

一个小单9.9，一个大单900+

视频号嘛就更简单了，小菜一碟，

在经过了7天的奋战后，

直接放弃

在航海期不停纠结，

闲鱼不喜欢当客服，好麻烦，

视频号，怎么看不懂，但是好挣钱，啊怎么办？

希望你不要和我一样，总是想着怕错过赚大钱的机会

在还是小白的时候，挑个你看得懂的，能做的项目，

直接干，别被其它信息干扰

闲鱼，在航海结束后放弃了，

因为之前看过一句话，挑你喜欢的项目

就因为这个，我都没有考虑到，我的首要目的是先赚钱，而不是挑三拣四。

# 第二次航海

就这样到了8月，参加了视频号口播航海

看了之前的手册，和高手领航，觉得哇塞好赚钱！

我也可以！

在航海开始前，按照手册教的去拍百货，发

航海开始后，才发现又踩了坑，视频号的规则又变严了

单纯的这种，特别容易违规。

航海刚开始的时候，

觉得好难啊，这怎么拍？

按照教的，各种类型都拍了看哪个数据好

就定哪个赛道

思维类的出单了，但是觉得这个拍起来太费时间

采访类数据也不错，拍着剪着都顺手，就去做采访了

心想，这次一定要坚持，拍了20多条，都不爆！

又想，是不是形象不适合，3个号，其中一个号换形象

拍了条爱国，一下爆了，后来就这个形象继续拍，数据一直不咋的，期间是非常焦虑的

要不要再换赛道，万一是我拍的不够多怎么办？

怎么样才能快速涨粉，快速赚到钱呢？

后来纠结中，换了思维赛道，不得不说自己确实适合这个方向

7天爆了2条10W+播放，涨粉1000+

但是到了带货文案，自己就不会，又不知道怎么办。

# 第三次航海

后来10月份，新的航海来了，

直接放弃了视频号，没有在更新过。

新航海开始前，一直纠结报哪个？

AI写作，AI特效直播，小红书虚拟资料，coze工作流，都好有潜力

报哪个呢？哪个能快速变现呢？

最后选了coze和小红书虚拟资料，

又贪心，看到K12，普通人也能月入过万的好项目

红利期，自己一点都不了解

期间coze又放弃了，根本没精力去看，

小红书虚拟资料，觉得好难啊，又要做笔记，笔记咋做，啥都不会。

朋友圈怎么经营，还要引流，还要建资料库，话术怎么聊，一次就赚一点，能赚到钱吗？

这就是我当时的心情，非常的焦虑。

既想赚大钱，又什么都不会，还想抓红利。

凭什么是你呢？

航海结束，没有跑通，因为不停地告诉自己

我不喜欢引流，不喜欢这种盗版笔记

没有一点信念。

航海航结束后，整个人没有一点心力，

什么也不想做，觉得自己完蛋了，钱也挣不到，

项目还看不懂学不会，啥也不想干

每天晚上熬夜刷帖子，

期待来一个赚大钱的项目来救救自己。

然而并没有，

翻了往期航海，看到一个早睡航海，

觉得不能这样了，最起码，我早点睡，养养身体

我宝宝那么小，我要完蛋了，她不是太可怜了。

坚持早睡了3天，哈哈又不行了。

后来，想了个办法，

每天晚上，必须让老公晚上9点

把手机没收到他屋里。

不然，我就记他一次，没收他一次

这点他做的还是不错的，没有让我得逞。

到今天为止，

我已经早睡19天了，状态好了很多

# 出现转机

刷星球的时候，看到梨云老师的精华帖，觉得太有帮助了

刚好有私教课程，直接报名了。

私教课，觉得天哪！

我要退款，跟我想象的不一样

我想象的是，每天解锁一课，每天布置一个实操的任务，

课程学完，项目也跑通的那种，有人监督

这么多课我要怎么学。

前三天硬着头皮，要不给她说说，让她给我退吧。

没这个勇气啊，她一定不会给我退的，

那么多钱，不能浪费，好吧，学！

我每天开始看课，实在迷惘，咋办啊，

课也看不懂我没救了。

去问云云老师，她让我先看怎么学习那一课的视频。

我开始每天看课，第一个小目标就是把课看完，让老师带我，看看她怎么带。

3天晕头转向的大脑风暴后，终于看完了，

然后开始让我选赛道，我真的不知道要做什么。

后来老师推荐了一个。

定了赛道就是去装修账号了，

不知道怎么弄，就重新回到课程里面跟着去操作。

找到了点感觉

接着就是制作第一篇笔记，

妈呀，这怎么做呢？

反正就是，遇到困难，就是找相关课程，

跟着课程去做，一点一点，一步一步，慢慢的越来越有感觉

还记得第一篇笔记，花了6个小时。。。

后来越来越轻松

从早睡开始，我的生活状态变得越来越好

现在会关心自己的身体，合理安排时间

不再是一团糟

# 总结

我想写一篇帖子，把这次小红书的学习经验写下来，

以后的项目也能用上

1、做项目先把所有的课过一遍，有个了解，知道大概讲了什么

2、开始做，做的时候比如不会写标题，就回到课程里，去把跟写标题相关的课全部认真听一遍，记笔记，跟着做

3、不定期的再回去看课程，我现在会每天听一节，每次都有新的收获。

昨天刷星球的帖子，

发现那些项目跑的不错的小白。

他们也并不是一帆风顺的，他们也焦虑，也数据不好，最后他们都有一个共同点，那就是坚持！

学习有三个层面

1、第一层：停在信息表面，

比如有个香蕉，别人告诉你它很甜，很软，是黄色的，你知道了。

2、第二层：

你去把香蕉拿过来，拨开，尝一尝，哦原来如此！

3、第三层：信念

我就是相信，我这辈子是能通过自己的努力改变人生的，而有的人，他从心里面不相信自己可以赚到钱

前面两次航海，之所以能跑通，

一方面就是听话照做，另一方面心里坚定的认为，自己就是可以做到。

没有赚到钱，是因为，我压根不相信，我会做这个项目，所以早早地放弃。
