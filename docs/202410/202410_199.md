# 当下创业方向：聚焦非标产品，避免价格战

> 原文：[`www.yuque.com/for_lazy/zhoubao/nh8r0fh6gmyqxklp`](https://www.yuque.com/for_lazy/zhoubao/nh8r0fh6gmyqxklp)

## (70 赞)当下创业方向：聚焦非标产品，避免价格战

作者： 黄海

日期：2024-09-24

当下这种环境，还剩下什么创业方向？

这篇文章不来谈什么宏大叙事，就分享一下踏踏实实创业挣钱机会的思考。

今天在中国商业领域，我们经常听到以下的关键词：产能过剩、过度内卷、价格战。如果听到哪些行业跟这些词汇相关的话，要赶紧躲开。这种行业机会都属于巨头，离创业者是很远的。实际上，所有需要卷价格的行业都不适合创业者去参与。

当下合适的创业方向，用一句话概括就是：做非标的产品，最好这个产品是有点门道，需要你的讲解去协助消费者做决策。

那什么属于「非标品」呢？非标品相对应的是标准品，标准品可以直接比价。很多人现在习惯了在拼多多上买东西，觉得性价比高。我们在拼多多上买的绝大多数产品都是标准品。能很清楚地比较同款商品的价格高低。拼多多商家处境艰难，甚至到了去围攻总部的程度，固然有平台政策严苛的原因，但也和标准品的属性分不开。

那与之对应，不好直接比价、产品和产品之间可比性低的就是非标品。

市场上有不少的非标品生意非常赚钱。最典型的行业例子就是珠宝玉石，每一款玉石彼此之间都是不一样的，它是彻头彻尾的非标品。在直播间里主播给你推荐，这是一个珍稀宝物、世界上独一无二的，直接给你卖个几万块钱，是会有消费者上钩的。

珠宝玉石这个行业里面门道实在太多了，消费者很难分辨什么是珍稀宝物，什么只是破铜烂铁。所以很多成本只有几百块钱的产品，在直播间里面卖个几万块钱，还是销路很高。

在一段时间里面，抖音上最大的直播间并不是大家熟悉的东方甄选、董宇辉、罗永浩这些人，而是一个卖新疆和田玉的直播间。这个直播间单月销售额在 5 亿以上，一年 GMV 几十个亿，可以称得上是赚的盆满钵满。

不仅如此，在我们做这个研究的时候，东方甄选这个大家耳熟能详的头部直播间，仅仅排在抖音的第四名，前三名都是卖珠宝玉石的。珠宝玉石是非标品，而且一般人很难了解里面的门道，需要非常强的讲解跟内容能力去引导下单，所以一个善于讲解行业门道的直播间就能赚到超额利润。

除了实物商品之外，另外一类非常典型的非标品就是服务类的产品：包括知识付费、社群，还有文化旅游产品。消费者买知识课程，买的是一种想象力，因为你在下单之前根本不知道这个课程里面会有什么。它的不确定性很高，信任门槛也就极高。

大家都听过樊登读书会，樊登所讲的书本身是一个标准品，但樊登的讲述节目就变成了一个非标品。因为这是樊登作为老师，他自己消化过一次再结合自身经历个性化演绎的，哪怕其他人讲解的是同一本书，产出的内容是完全不一样的。

樊登演绎讲解能力强，因此他可以把「互联网讲书」这个行业做到一年收入几十个亿。樊登这种知识付费的业务，它本身的硬性成本并不高。所以等他收入几十个亿的时候，企业利润有多么可观，就可想而知。

多说一句，樊登的著作「低风险创业」阐释的创业方法论，大体就是对一个商业问题通过提供非标解决方案获得利润，推荐大家阅读。

另外一个典型例子是新东方。新东方是做教育课程起家，最近两年他也在进军文旅行业，开始组织老年人的旅行团。在中国，刚刚退休的初老一族可以称得上是整个社会上消费力最强的一群人。

新东方瞄准这个群体，去组织带有历史和文化色彩的旅游团，它比一般单纯的旅游又高了一个境界。这种没有办法标准化，带有很强的文化历史属性的服务，它背后的毛利率也是可观的。旅游产品文字讲不清楚，新东方通过视频和直播间来生动呈现祖国的壮丽山河，有钱有闲的初老一族受到感染自然就下单了。

总结一下，做一个需要讲解的非标品，为什么是当下适合大部分人的创业方向？

第一，非标品无法比价，利润率通常比较高。内卷是消灭利润的最主要原因。在中国过度内卷有两大因素，第一大因素就是平台的算法，第二个因素就是供应链的产能过剩。这两个因素共同导致了信息极度透明，产品也特别容易获得。你做了一个好的标准品，可能在一周之后就会有比你更便宜的模仿产品上线。

创新者的成本永远是最高的。跟随者看到你的套路之后，他去模仿的成本是更低的。

第二，在当前的社交媒体环境之下，做一个非标品，跟用户去进行直接沟通的方法是前所未有的丰富。以前的互联网行业，大家买东西就只能想到淘宝和京东。同样是卖珠宝玉石，如果丢给消费者一个淘宝链接，打开去看详情页，消费者是很难被打动的。但是在直播间里面，主播绘声绘色地给大家描述个 15 分钟，用户可能就乖乖下单了。

第三，非标品龙头企业、大公司很难兼顾——大公司都要瞄准最大的市场。非标品总会给人一种感觉，难以做很大。在这种情况下，大公司就顾不上你，只有大厂看不上的地方，才是创业者活得最好的地方。

七天 : 总结下里面的犀利观点，对个体创业选品类很有帮助~
1.非标品是高利润领域：在产能过剩和价格战激烈的市场环境下，创业者应当避免参与标准品的竞争，因为这些领域往往已经被大企业占据，并且利润率低。相反，非标品由于难以直接比较价格，通常能够保持较高的利润率。
2.知识与内容创造价值：对于非标品而言，消费者决策过程中需要更多的信息支持。因此，那些能够提供专业知识、个性化服务或独特体验的企业更有可能成功。这包括了知识付费、文化教育以及定制化旅游等服务。
3.社交媒体促进沟通：现代社交媒体平台为非标品提供了前所未有的机会，使得创业者可以直接触达潜在客户并建立信任关系。通过直播等形式进行产品介绍和服务展示，能够有效提升转化率。
4.避开巨头竞争：大型公司倾向于关注规模更大、标准化程度更高的市场。相比之下，小众而非标的业务模式则不太可能吸引到这些行业巨头的兴趣，从而给创业者留下了生存和发展空间。
5.利用信息不对称获利：非标品往往涉及到一定程度的信息不对称（例如珠宝玉石鉴定），这意味着拥有专业知识的一方能够在交易中占据优势地位。善于利用这一点的创业者可以通过提供咨询服务等方式获得额外收益。
