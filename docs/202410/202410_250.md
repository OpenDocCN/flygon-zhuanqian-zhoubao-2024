## (16 赞)采访分享，利用身边资源轻创业

作者： 胖虎

日期：2024-09-23

大家好，我是胖虎。

上周去沧海哥公司聊天，虽然行业不同，但同为初创者也有很多想法上的相似之处，相互吸取了各自的经验和能量，也决定面向沧海哥的会员用户做一次分享。

说来惭愧，最近很少写些什么，正好借这次机会，也能让我自己梳理记录一下最近的想法。

下边主要是问答环节。

# **提问 1：**

跟大多数职场人不同，胖虎，刚毕业 2 年就选择开始创业，当时自己怎么考虑的？创业的方向是什么，可以给大家分享一下？

# **回答 1：**

我在大学期间就做过创业项目，虽然现在看起来比较业余，但当时好像点燃了心中的一团火，一团想做点事儿的火。

毕业后朋友也邀请我说一起做点事情，但“人贵在有自知之明”，我知道当时自己几斤几两，时机不到，还是想着沉淀一段时间。

在两年的工作期间，我的成长是挺快的，所以就离职跟朋友一起来搞点事情，刚开始其实也没明确的方向，正好又有机会跟周围的中小学进行合作，共建人工智能示范校，收取培训费用，自己又对这部分业务有一定了解，碰到机会就先干，先下场做。

# **提问 2：**

都说创业不是人干的事，创业苦创业累，胖虎你创业 1 年半以来对你最大的收获是什么？

# **回答 2：**

创业的确苦，除了体力上的辛苦外，还要独立深度思考，吃脑力的苦；点头哈腰求人，吃尊严的苦；持续进化迭代，吃自律的苦。还难在苦吃了，可能没什么结果。

但在创业过程中，慢慢觉得这些好像也不重要，咱们想，如果把创业当成玩儿一款无限游戏，升级、打怪、换装备、了解新玩法、解锁新地图、结交同行者...

在玩游戏的过程中，我们能知道了自己的成长属性怎么样，更喜欢玩战士还是法师，想当是先锋还是军师，难关怎么过，BOSS 怎么打。这样可能会有趣很多。

创业后视角发生转变，看东西的角度跟打工不一样了，这不但是做业务和商业上的感觉，在生活和与人交往中，会更了解自己、了解人性和一些社会的运转规则，整体比以前更通透了。

# **提问 3：**

对于很多自由职业者，很多想要轻创业的朋友，很多人是没有方向的，胖虎你当时是怎么找到自己创业方向的？

确定这个方向的时候你做了哪些准备？

# **回答 3：**

提到方向，我想先聊聊当时做副业的经历。22 年 7 月份开始，听说好像抖音能赚钱，在当时局限的视角下选择了拍 vlog，想着涨粉接广子，结果是五个月 VLOG 涨粉 80 多个，变现十几元，结果上看是失败了；后尝试视频号书单号，有了搞钱意识，锻炼了视频剪辑能力，10 小时出单，每个月都能持续变现几千元；接着做小红书个人 IP 内容，包括自媒体运营、面试技巧等，因不专业和体量小流量不佳，也只有几百块的变现。

2023 年初 ChatGPT 火了后，开始分享其产品、功能等，通过引流卖账号及课程分销快速赚到一万多。后面开了实体的线下水果捞店铺，然后也开始正式创业，做很大程度上也是因为这些经历。

开始创业的项目选择的是科技教育，觉得熟悉的领域更有把握，毕竟初创时减少一切投入，做有把握的事情很重要，而且当时自己对于项目视角和信息是很有限的，没有太多选择的余地，方向不是选出来的，是干出来的。

无论失败还是成功，每个项目都能积累可复用的能力。我们需要积累小成功，每一个小的成功都会增强你的信心、突破认知边界。

# **提问 4**：

很多人会觉得自己是 I 人不适合创业，创业就是要不断的跟人打交道，

# **回答 4：**

社交能力是商业市场下，非常核心的一种能力。

我目前做的主要是线下生意，与线上不同，需要不断的去见人谈合作，并且很多还有是政府体系内的领导，这更需要进行不断的打交道。

比如去年跑浙江省各教育局谈合作，一个月跑遍各地级市及部分区县教育局，30 天喝了 20 次酒，我酒精过敏却不得不喝，因为在很多线下资源性生意中，喝酒似乎成了必要环节，年轻小伙这样做才会被认为有冲劲、拼劲而受前辈青睐，虽然不擅长但也会尽量去尝试。

在这一年多的沟通合作中，见了形形色色的老板，发现外向和内向与是不是和创业并无本质关系，重要的还是踏实做事，清晰表达，尊重利益，尊重人性。

# **提问 5：**

对于很多初级创业者自由职业，想做好个人事业，是很孤单和寂寞的，我知道你也有合伙人，当初你是怎么寻找合伙人？怎么找到你的创业搭子的？

# **回答 5：**

现在的合伙人，我们从大学创业时就是一起的，有足够的信任积累，这很大程度上减少了一些不必要的矛盾。虽然在沟通过程中，会有不愉快和争吵，但目的都是为让业务更好发展以及让自己成长。

我们在做事风格和性格上互补，他气场强大，善于攻克关键节点，去疏通关系，而我更注重业务发展与我们自身是否匹配，看重团队协同和复盘迭代。

找合伙人，就是要各取所长，并肩战斗，增加做成事的概率。

# **提问 6：**

从少儿编程到研学，是什么原因让你有了这个跨领域的创业方向的转变？

# **回答 6：**

首先是市场原因，之前和跟中小学合作少儿编程这块儿，全职教师的成本太高，因为有时候课是派不满的，兼职老师又不稳定，后续想要做到更专业，要在教研上投入的资金和精力比较大。再加上教育这一块查的很严格。

研学也是很早就进行了测试，高校研学这个方向也是我们愿意去做的，业务前景还可以，有对标浙大研学能够模仿，杭师大校友也能不断争取学校支持，而且是现金流的生意，收钱干活儿，把这几笔账算好，就投入开始做。

整体上都是教育领域，细分一下代表的是其中的素质教育类型，我自己是山西县城的，来到杭师大以后，才感觉到教育水平的差异。目前的我们做的教育内容中，更在乎孩子的综合能力，

# **提问 7：**

对于现在创业的小伙伴，大家都更聚焦在轻创业上，压力小，胖虎你的创业起步在杭师大，你是怎么利用身边的资源开始轻创业的，你都用了身边的哪些资源？这些资源给你解决了哪些问题？

# **回答 7：**

作为异地的毕业大学生，刚开始创业能用上的就是你的同学朋友，学弟学妹，老师和母校。

以我自身经验为例，大学时创业就与学校和学院的很多老师熟悉，毕业后也有一些联系，开始创业后前辈们也理解创业的不易，由于是校友创业刚开始的办公室，房租减半，还拿到市政府补贴。通过与学院谈合作、答辩及申请，获得了校内办公场地。

刚开始做科技教育时，找上课老师，学弟学妹也会很相信你并且帮助去发动。不要小看大学生，一些工作技能上同学是很优秀的。

重心放在研学后，之前合作科技教育的学校是有需求的，我们和校长熟悉就更容易切入谈合作。之前聊过我们拜访了很多区县教育局和教培机构，也是我们研学合作的对象。

由于是师范大学，有些学长学姐和学校老师们会帮忙推荐，学院的老师和书记接触的都是很多企业老板、学校领导，这样对接下来合作更容易达成。

做过的事情都是算数的，没有人脉，没有资源，先从身边入手，不断积累。而且我们的出发点是做成事情而非单纯谋利，争取的是学校、学生和我们三方的共赢。

一所大学和大学里的老师，是拥有很大能量的，把这些身边现成的资源进行整合和利用，会对自己创业过程中有很大的帮助。

减少一切不必要的投入，仔细想想如何借助现成的资源，增强做成事的概率。

# **提问 8：**

创业就是生存实践，要想尽一切办法活下去，活得好。在你创业这 1 年半的时间，你做到最有成就感的事是什么？有没有让你很难忘让你长教训的案例，方便跟大家分享？

# **回答 8：**

成就感其实还没有，看到的越多只会觉得还有很长的路要走，只是觉得公司还活着，挺好。

踩过一些坑跟大家分享一下：第一点，做事上，切记自嗨。小团队创业，没有利润都是扯淡，初创团队搞流量大于搞产品，如做研学时，投入大量精力和成本做产品体系，设计更专业的课程产品，做出了也没有获得真实市场的认可。放下完美情节，边做边优化，流量才是关键，有了充裕现金流才能更好调控。

第二点，识人上，擦亮眼睛。与人合作不要看他说的什么，要看他怎么做的。现在张嘴就来的老板很多，口气很大的，暑期跟一个渠道合作时，签了五千人的招生协议，我们来对接落地和管理，后因对方招不到生导致团队闲置，跟学校各部门的合作信誉受损，经济受损，所以合作要慎重，不能依靠他人，自己要有足够能量应对潜在风险才稳妥。

# **提问 9：**

胖虎，你创业路上也遇到了贵人，比如你的投资人，你是怎么找到投资人的，他投资你的原因是什么？可以分享一下？

# **回答 9：**

我们常听这样一句话，读万卷书不如行万里路，行万里路不如贵人相助。所谓贵人，他一定是认知眼界和经济实力远超于你的。

目前的投资人是大学时就认识的老大哥，起初因对少儿编程感兴趣去他公司做兼职老师，后来与他有了一定合作关系。毕业时老大哥虽建议去他公司上班或安排到其他地方，但我选择了陌生的互联网行业，想自己试一试，当时他跟我说过一句是，"要让自己变得更有价值。"

经过两年工作积累扎实技能和能力，变得更有价值后，与老大哥以合作形式开展创业，他进行了前期资金和资源的投入，自己有价值，贵人才愿意为你的价值进行孵化和培养。

其次，上一家公司的老板条形马老师，也是我的贵人。虽然我当时是用户运营岗，但我我还做过直播运营的活儿，了解人事行政的制度，还有一些公司的盈利数据也会交给我去分析整理，这极大程度上锻炼了我对全盘业务的理解能力，贴身学习时，他的做事方法和逻辑让我收益很多，如苦练基本功、重视文字的力量、培养迭代思维等。

离职创业后，更加体会到他的厉害之处，遇到困难会想，如果是马老师会怎么做？回过头来当时的一些我不理解的决策，原来是我太菜了。这些都对我的创业有很大帮助。

# **提问 10：**

创业让人兴奋也让人痛苦，没有伤痕累累哪有的皮糙肉厚，创业是很消耗人精力的事情，当你消沉 没有心力的时候你是怎么度过的？

# **回答 10：**

以前不知心力为何物，当业务开展不下去、团队不知道怎么管理、发不起工资，一个人在办公室发呆的时候，接感觉心很累，这时候我 才理解到，心力就是一个人内心的强大程度，它是动态变化，需要不断经受考验，不断磨练加强的素质。

心力憔悴是好事情，这个时候往往就意味着自己要成长了，痛苦意味着改变，改变意味着适应，适应后就成长了。

而且要和自己和解，也不用把自己逼太紧，累的时候会看书、运动，若连这些都不想做，就放肆一把，出去旅游或在家打游戏，以此调整状态，只要清楚目的是快速调整状态，找到适合的方法去尝试即可。

# **提问 11：**

对于现在想要轻创业的小伙伴，是大胆去做自己想做的还是真的要具备一定条件才开始？很多人总是觉得自己是小白，不敢迈出这一步，你有哪些建议给到大家。

# **回答 11：**

我不鼓吹所有人都要创业要干副业，但希望在当下的环境中，我们脱离公司平台后，要有能独自面对市场赚钱的能力。

低成本做项目，失败成本并不高，而错过的成本却很高。还是像之前提到的，把它当成游戏，结果可控的情况下，即使失败了，大不了重开一把。

在过程中，不要过度担忧结果，要取得阶段性成果，从小成功不断获得正反馈。

庸人总说“如果”，普通人总问“结果”，高手只分析“因果”。

**躬身入局，把手弄脏，降低预期，都是惊喜。**
