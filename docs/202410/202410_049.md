评论区：

盐宝儿～ : 思考： 聪明是否意味着“能看到各种机会点” 智慧是否意味着“能一切得当的做出选择、判断，并做出行动。一切都恰合时宜”（这样看，智慧确实有点难[偷笑]）
不知道亦仁老大有没有时间能稍微具像化一下这种“智慧”？

亦仁 : 比如，聪明是可以快速的在生财的各种项目里面穿梭从 0 到 1，智慧是谋定后动，看准一个项目 allin 进去，不管遇到什么困难，只要最底层的判断没有被颠覆，就会一直坚持下去，从而最终赚到大钱，当然，智慧不可能靠只看不做而获得，而需要大量的实践之后才可以获得。
比如，你看生财的很多人，很聪明，很容易出小成绩，但每隔几个月找他聊，就会发现他换项目了，这就叫做没有智慧，分不清到底什么是最重要的要聚焦的地方。

盐宝儿～ : 1.不是我要什么，而是我不要什么，只要什么 2.谋定而后动+长期主义 3.只要底层判断没被颠覆，就要相信相信的力量 学到了！谢谢老大！🌹🌹

卢志敏 : 怎么弄清楚最底层的判断有没有被颠覆？感觉这个问题就难道一大批朋友了[流泪]

亦仁 : 不断的提升认认知，如何提升认知？ 读好书，做大事，见高人。
