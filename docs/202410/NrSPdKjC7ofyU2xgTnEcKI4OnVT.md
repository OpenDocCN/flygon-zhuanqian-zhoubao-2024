# 五，小绿书目前的玩法分享

目前小绿书的玩法主要有两类：

## 1，达人分销

如果是达人分销，那其实入局的玩法就非常简单，你只需要开通视频号橱窗，然后缴纳100元保证金，就可以开始发图文带货了。

文章爆了，你就能够出单，卖的越多，自然你就赚的越多。

优势：成本低，方便批量起号带货，流程简单，门槛低，实现自动化之后可以非常高效发布。

劣势：你不是自己的品，自己接触不到厂家，只能在选品中心里面选，对整个链条接触会比较浅。

避坑点：在选品带货的时候，要看商家是否会及时发货，我们之前有一个朋友自己做，文章爆了，出了几十单，但是商家有超过一半没发货，造成退款，那这个损失还是比较大的。

后面才了解到原因是商家国庆这几天放假，不发货，而你作为达人而言，是没有什么话语权的，如果你是自建小店，你还可以更换货源，换个厂家发货。

这里给圈友解释一下，为什么说自建小店，基本上就相当于是在自己打造供应链——

因为你有了自己的店铺，就需要去选品，去上品，去对接快递物流，去对接厂家货盘，去对外销售，这不仅仅就是简单带品发文那么简单，这是一系列的组合。

所以在后文，我都会不断提到，自建小店，其实就是我们在打造自己专属供应链的一个过程——

你销量越好，和快递谈的运费价格就越低，和厂家可以谈的拿货成本价，就越低，继而你的销量优势就更大。

# 六，我的经验和技巧分享

## A，达人分销技巧和建议

### 1，没有流量怎么办？

这个基本上是大部分在做的朋友，都会遇到的问题，每天在发，但是基本上没流量，然后自己就很焦虑，会怀疑自己的努力是否有回报。

很多人中途干了一段时间，没有效果，就直接放弃了

没有流量，其实是一件非常正常的事情，作为普通人，我们能做的就是两个字：怼量。

一条发1条肯定是不行的，日更10条是基础，日更20~30条，再确保你发的每一篇都有爆款因素，这个是猛人，不可能不出结果。

### 2，选品很关键

选品这个部分，分两块，一是选实用特色的品，相对应产出爆文，出单才好；二是选这个产品背后的商家，有的商家不靠谱，那就很麻烦。

微信小店虽然有选品中心，但是不代表里面什么品都可以上.

比如，就会有人带货出了单，但是商家一直不发货，那这就很难受了，最后超时未发货，用户直接退款，出单了也是白忙活，所以找到合适的商家做后端，也非常关键。

### 3，对标爆款，复制是最好的路径

我看见太多人抄作业，也会抄错的，明明对标了一个爆款标题，但是经过他的手，再写出来的标题，就变得非常干巴巴，压根就不想点进去看了。

那如果有这种情况，其实会极大的影响最终的结果，不出单就是非常正常了.

这个时候，直接抄对方的标题，或者只改微小的一部分，其实就是最好的策略，然后在内容上优化和变动即可，小绿书目前对原创的要求不是很高。

## B，自建小店经验技巧：

### 1，0元保证金试运营

如果你选择自建小店的话，目前视频号官方是有福利政策的，可以申请开通0元保证金试运营，这样你不需要保证金也可以开始上品卖货。

但是你卖的钱不能提现，以及最多只能卖200单，然后你只有把保证金补齐了，才能解除限制。
