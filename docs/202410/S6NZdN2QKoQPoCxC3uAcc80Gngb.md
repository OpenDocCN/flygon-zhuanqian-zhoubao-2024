# 一、拼多多开店的优势与劣势

优势：

1.门槛足够低：与淘宝、天猫相比，拼多多的门槛相对较低，拼多多只需要缴纳1k元的保证金【最近会降到500保证金】，而且对于制作商品图片和补单的要求不高，只要店铺运营得当，就可以被动承接流量~

2.开店起步快：如果大家可以掌握文章介绍的“运营”部分的知识，能够很快的去起一个新店铺，并且拿到正反馈！最重要的是持之以恒！

【根据我的测试结果，只要运营得当，可以让完全不懂pdd的小白大概率至少拿到一个店一个月2k的成绩，当然，话不能说绝对，这是我的测试效果】

3.同行竞争少：相比于闲鱼的同行而言，拼多多的卖家还没有达到白热化的阶段，在拼多多庞大的用户基础上，卖家数量远远没到达到闲鱼与淘宝的那种地步~

4.时间成本小：前期工作铺垫好之后，只需要每天打开直通车等询单即可，一次聊天平均在2-3分钟，前期询单数少，随着店铺权重积累，会逐渐变多，单子普遍使用GPT10-30分钟就能完成

5.询单稳定：所有的动作都有迹可循，因为我们是用钱去买流量，所以不存在说没有询单的情况，询单只会根据店铺权重一天比一天多，然后到达一个临界值，最重要的还是持之以恒！

劣势：

1.稿费水平一般：由于拼多多的用户比较注重性价比，这会导致一种拼多多上的价格竞争比较激烈，对于演讲稿等通用稿件的价格一般会在20-30/千字左右的价格容易成交，【这个价格转化率高，并不意味着所有稿件都是这个价格，我身边谈单能力强的伙伴，也可以谈到拼多多千字100-200，算是比较特殊的】，鉴于这种情况，我们大多数人是小白和普通人，目标建议以千字20-30元总价为目标。基于这个前提，建议可以自己一条龙服务，保证自己的利润！

2.拼多多对代xie监管十分严格：

由于拼多多的规则非常严格！所以在拼多多的日常运营中一定要谨言慎行，由于我们做写作单子需要和客户沟通字数要求和交稿日期等一些内容，而在这些内容中会有很大的可能性含有敏感词语，所以在这个过程中一定合理的去尊重和规避平台规则。

【这点不用担心，我接下来会告诉你怎么在平台的规则下游走，让你手把拿捏】

3.聊天频率快：拼多多的客户很匆忙，基本上你一会不回这个人就走了，聊天频率基本上2-3分钟一个客户能不能谈下来就知道了，有的客户想做预算低，也可以尝试接下来，因为单子很简单，抱着练手的目的去做就好。

# 四、重点：违禁词规避

## 1.聊天

### 1.1正常沟通流程

放在最开头：

在聊天的时候时刻注意违禁词的出现！！！一旦不注意，轻则有限期封禁，重则无限期封禁店铺！

拼多多对于代写的管控非常非常严格，所以大家一定要注意对顾客进行筛选，把眼光放长远，不要捡了芝麻丢了西瓜🍉~

店铺有顾客询价时可以按照这个模板:

多少字数？什么时候要？是否有通知文件？

ps：字数在5k字以上一般都是lw，可以追问一下查重率，确认一下，然后给出自己的报价~

报价可以给出不同价位，给顾客提供一个选择~

然后发出消息后看看消息是否迅速被已读如果已读，迅速回复消息说:您看怎么样?

给顾客进行逼问，让他回复你才能有商量的余地。

不然消息发出去了也不赶紧追问会很大程度上的造成顾客流失~

如果顾客点名要原创的话，可以给顾客说免费出查重报告，让顾客感觉占到便宜，这种文章一般大多数都是需要查重报告的，他说出来和我们说出来的效果就不同了文章写完后尽量都去降重，免费的查重网站给出一个报告就好了，这样也能够拉高店铺的评分。

在店铺前期尽量成单，及时回复客户消息，三分钟回复率是很重要的，大家尽量好好控制自己的睡眠，不要占用8:00-23:00，虚拟店铺数据掉下去还是比较难刷出来的，尽量重视，成系统的去做。
