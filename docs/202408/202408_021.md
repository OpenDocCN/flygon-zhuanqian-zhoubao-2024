## (32 赞)利用 AI+agent 一天生产 50 条伪原创口播视频

作者： 张一凡

日期：2024-08-21

声明：本人只是对 agent 感兴趣，这个工作流是某天想到的，于是把他做完了，没有在自媒体平台上做过验证，仅作为思路供参考

工作流程：

1.利用插件爬取某个抖音账号的所有视频

2.筛选出点赞量高的内容，利用 coze 提取出文案

3.上传自己的内容文档，ai 模仿你的风格去改写爆款文案（也可以用其他主播的风格或者名人，比如雷军）

4.制作自己的数字人分身，上传爆款文案，生产视频

5.二次剪辑，生产字幕+导入素材
