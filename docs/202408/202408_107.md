如何平衡好主业和副业？

01

这个问题，每一年，都有好多圈友在星球问。也有不同阶段圈友的不同回答，我也回答过这个问题好多次，答案在不断改变。

现在，我对这个问题看法如下：

1/ 你一定要搞清楚一件事：你的主业是在为你的副业做准备，你的副业是在为你的事业做准备。尽量把主业和副业看成是一件事，不是去平衡，而是去衔接。

2/ 我强烈认为，每个人，都要有自己的事业，或早或晚。

3/ 有一件事，可以让你全身心 100% 心无旁骛投入进去，哪怕失败，又何尝不是一种全新的独特体验？人生在世，就是为了获得足够多不同体验。

4/ 我说主业是为副业做准备，并不是说让你在主业里摸鱼，恰恰相反，你应该在主业上像创业一样疯狂投入和尝试。

5/ 因为这是一个赔率巨大的事情：尝试成功了，碰到不算太差的老板，会给很多回报和机会。尝试失败了，学到了经验和技能，有老板兜底，也不至于破产亏钱负债。

6/ 在主业中尽量学习副业中最需要的能力：如何从 0 到 1，其中更重要的是如何进行低成本尝试，这是我认为赚钱最重要的能力。

7/ 如果你能近距离接触老板或者操盘手，看他们是如何从 0 到 1 做成一个业务或者一件新事情，那你是幸运的，请认真学习。哪怕经常被骂。

8/ 赚到钱，核心就是因为控制住风险和成本，让自己不要出局。只要不出局，就可以不断尝试，直到成功。

9/ 如果副业已有起色，但考虑到收入问题还舍不得立刻放弃主业，请把副业收入的一部分拿出来招线上助理，不要舍不得，哪怕你空出时间来休息养精蓄锐，或者陪陪家人。

10/ 助理最好从同学、家人或者朋友开始，他们会更相信你。因为这个时候你根本不知道怎么从 boss 直聘上，大多数人也会把你当骗子来处理。

11/ 但你最好有两个预期：第一，你的助理并不会帮你干的多好，你会觉得自己亏了，这个钱花的不值；第二，如果你的副业很赚钱，你的助理最后很可能会变成你的竞争对手。这些没关系，大家都经历过，先解决眼前的问题，未来的问题未来再说。用资金换时间，用时间换空间。

12/ 关键是做好取舍，不可能副业也好，主业也好，休息娱乐一个不少。我还没见到谁能做到，赚到大钱的人，背后基本都有巨大付出，只是有些付出不是你能想象到的维度。

13/ 当我们问如何平衡主业和副业时，是不是在问如何保持休息和娱乐时间都不少的情况下，副业和主业都能获得增长？想什么呢，凭啥是你？

14/ 和老婆商量下，周末挑一天出来，出去找个付费自习室，拼命干一天，基本等于你每天下班回来干一小时干半个月的效果。记住了，及时的把一些进展分享给你老婆，未来你可能获得两天的全职副业日。

15/ 不要长期熬夜，保重好身体。

16/ 争取副业干三年赚 100 万，让副业光荣转正。

17/ 加油，来到生财有术，你已经赢在了起跑线。
