# 项目拆解流程：GMV=流量*转化

## 一、流量端

流量是直播推荐 + 搜索流两个；接下来分别说一下这两个流量如何做：

1、搜索流流量来源是产品链接，所以直播间要挂满200个商品，爆款商品自带搜索流量；

2、直播推荐流量来源是直播间停留互动 + 订单密度 + GPM数据递增破流量池做出来的，所以需要起号打标 + 破流量池操作。

3、尽可能长的开播，开播的时间越长，收益越高。

总结：流量端的核心动作就是挂满200个商品 + 直播起号操作 + 长时间开播

## 二、转化端

如何保证直播间进来的人能够产生高转化呢，这个就是直播间内容的模块了。

1、消费者为什么会在这类直播间下单，本质是产品性价比高。搜索流1.0玩法是利用达人券补贴消费者提高下单率；搜索流2.0玩法是通过选出高性价比的商品，现在精选联盟里面有很多品极致性价比，价格低，给达人5-15%的佣金，产品符合抖音“价格力”策略，有更多流量；

2、搜索流本身是精准的，所以只需要选出有搜索流的品200个 + 直播间话术承接就可以；

3、直播推荐流量在起号阶段是不精准的，需要打标签做数据才能精准流量，所以起号很关键；在起号的过程中，要配合主播拉流话术，200个商品要有主次，自然流容易出单的产品往前排；

4、整个直播间要营造低价、清仓、批发、甩卖等氛围，更容易留住消费者，促使其下单，具体表现在直播间画面、主播话术；

总结：转化的核心动作是直播画面吸引 + 打标起号 + 主播话术 + 组品排品（如果放弃直播推荐流量，仅仅拿搜索流流量，就不需要起号打标，组品排品等操作）

利润计算：净利润 = 产品价格*佣金率*90%（90%是技术服务费）

# 起号操作

以上工作完成，直播间就可以正式开播了。抖音直播间从0-1是最难的，尤其是自然推荐流量打标拉流的过程，所以在开播前要确定自己的起号方法，以及运营操作，接下来是起号的重点内容，也是直播间成败的关键！

## 一、搜索流

搜索流是因为产品优质，消费者直接搜索产品进入直播间下单的，所以没有拉流、打标的操作；搜索流大小的操作很简单；

1、直播小黄车产品200个填满，选品选择近期爆款，且竞争对手不是很多的产品链接，我们自己测试下来一个链接同时被3个直播间挂都是有流量的，最先挂的流量最大；

2、直播间的200个产品可以更换，基本一个链接超过3场一单不出就更换掉，搜索流的核心就是测品；

3、直播间话术，按照上面讲的话术循环讲解，承接流量就可以。

总结：搜索流是很容易出来的，只要选品测试循环做，挂满200个爆品，第一天就出单是很轻松的；

### 二、直播推荐

直播推荐流量是系统根据直播间的标签和流量层级推送的，所以直播推荐的流量想要拉起来，就需要有精细化的运营能力；给大家用我们最近新号的起号数据，给大家讲讲如何操作；

1、抖音底层逻辑概括：直播间想要系统推流，首先要解决两个问题：

第一是流量精不精准？系统是通过标签匹配用户和直播间的，简单来说，就是直播间有目标人群互动、下单了，系统就识别了标签，下单的人越多，标签越准。

第二是流量多不多？系统识别了标签，知道推送什么样的用户了，但是到底是推荐100个，还是10000个是流量层级决定的，流量层级如何提高呢，搜索流2.0玩法直播间就是靠订单密度拉高流量层级。从我的观察看，这类搜索流直播间大部分账号的在线人数就是100多人，就是4-6层级。

流量层级在线人数阈值：5、30、60、80、120、180、230、260、1000人。

直播推荐流量起号的过程就是，打标签，破流量阈值的过程。

2、第一场操作

第一场操作，挂满200个商品开播后，通过点对点成交的方式打标签。这个时候进人很少，每个人进来就用点对点话术洗一遍，进来就走掉的不是目标用户，进来有人提问的才是目标用户，尽可能成交他。

第一场流量走势图，刚开始在线是0-5个人，点对点成交，等到成交几单后，系统识别基本标签后，推流会相对精准，在线会进入5-20人左右，在这个阶段不断洗人，提高下单率。

![](img/0982e1aad9735e29bfb37af8b491942d.png)

第一场基本是搜索流先出单，直播推荐流量后出单，但是一定要区分两个渠道的出单，搜索流大量出单，直播推荐不出单，是拉不起来直播推荐流量的。

![](img/5aee8454f72cefff7b643faac831cddc.png)

### 3、第二场操作

延续第一场的操作，还是要注意话术拉流，互动话术、关注话术等互动话术，直播推荐流量有人互动，有人下单才能拉高流量。第一天的操作一般比第一天的流量上升一个层级，也就是在线10-30人左右，如果出现途中这样的高峰，说明，前面1小时数据做的不错，系统在给更多流量，破流量层级。

![](img/0d512ff01a38564c609e672a9ab55ab4.png)

直播推荐流量占比达到50%左右，整体的订单量、GPM数据都有增长，订单密度增加。

![](img/a7d8f9bbeef3c44592a48190ae7697c5.png)

### 4、第三场操作

如果数据继续增长延续第二场操作，如果数据没有递增，第三场要做调整，这是要分析是哪个流量没有涨，或者下单，如果是搜索流问题，就换品，给直播间添加新的产品

如果是直播推荐流量，就调整产品排序，主播话术调整，重新精细化分析对标账号的话术。

正常第三场的在线是可以破30人的，也就是说账号基本打标完成，系统推荐的流量更加精准，播同样的时长，下单人数变多，GMV增加，GPM稳定。

![](img/530dc6c205b98c7cba6f124973f3818d.png)

### 5、第四场操作

如果数据继续递增继续之前的操作，如果数据没有递增，还是找问题，确定是搜索流问题还是直播推荐问题，然后做操作。

![](img/9bf45e84a13d396a24804b2aa8f37523.png)

![](img/aa857064e4e63a3c8588f7764dda2d42.png)

总结：直播推荐流量起号这一步是比较考验运营能力的，我上面说的第几场不一定就是第一场，如果你第二场没有做好，第三场还是延续第二场的操作。如果直播推荐流量一直拉不上来，也不用太担心，专心有话搜索流，GMV也不会很差。

这种类型的直播间起号大概就是7-10天，超过10天直播推荐流量没有起来，就放平心态，不用刻意管直播推荐流量了。专心播搜索流。如果有小伙伴做这个项目，碰到具体的问题也可以问我~

# 项目总结

1、本次项目是非常适合个人全职去干的，整体营收一个月1-5W左右，但是需要拉时长8-16小时，夫妻干最合适。单小时GMV我们测试下来是500-1000元；看多个类目同行的数据也差不多，利润单小时50-100元；

2、这个项目的成功率比较高，原因是有搜索流，搜索流本身属于保底流量，只要上满200个商品，再加上测品的操作，一天卖几千没问题；

3、搜索流2.0玩法里面加入了直播推荐的流量，更可能做大流量，但是直播推荐流量波动较大，所以在流量做起来以后，尽量每天固定时间，固定时长，断播3天就会掉流量层级；

4、搜索流2.0玩法把发优惠券的操作也去掉了，基本不会亏钱，就是比较费时间；

5、搜索流2.0玩法尽量做真人出镜的直播间，不要录播，近期平台查录播很严，录播直接扣12分关橱窗；

6、搜索流2.0玩法的背景画面都是静态图片，图片类直播间存活更久，背景不要做视频循环，容易违规；

7、搜索流2.0玩法直播间除了多品模式，还有单品打爆模式，也可以多去研究下对标；

8、如果想要把它当作项目去做的话， 一定要做矩阵，多矩阵运营收益更稳，尤其是所在城市人工成本比较低的话，可以用兼职主播模式放大。
