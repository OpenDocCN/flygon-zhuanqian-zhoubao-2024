# 20 岁倒闭了 3 个工作室-公司后我的深度复盘

> 原文：[`www.yuque.com/for_lazy/zhoubao/po55shi0xaxd3igf`](https://www.yuque.com/for_lazy/zhoubao/po55shi0xaxd3igf)

## (精华帖)(76 赞)20 岁倒闭了 3 个工作室-公司后我的深度复盘

作者： 高尚坏孩

日期：2024-08-13

生财的挚友们，好久不见，我是坏孩，一名 20 岁的持续创业者，创业三年累计变现 200w。

就在两天前，我正式遣散了新工作室的合伙人们，哪怕曾经这个屁点大的工作室也有过日产千粉的光辉历史，哪怕这个工作室开着，能够让我在外吹牛逼说坏孩手下好几个人。都不好使，我在管理制度上的各种问题，以及工作室整体目标不清晰导致整个后半个月工作室一潭死水。

**倒闭 7 大坑**

**1.没有摸清楚变现方式就盲目扩大**

这个错误出现在这次创业，原先开工作室的原因就是觉得自己跑通了整个创业粉流量的闭环，依靠矩阵和一些科技软件的搭配做到了一天内 1000 个客资，每天回消息也回不过来，就想着招几个合伙人一起来。但是后来发现人来了后，如果单纯给流量分成的办法的话，整个团队的压力就顶在我一个人身上，因为合伙人的收入来源是帮我打的流量，但我没有好的变现路径。以至于**很多时候很急，越急就越容易出事儿**。

**2.业务上不亲力亲为**

时常关注坏孩的人就知道，前段时间我们发售了一个付费进群的项目，男粉逻辑，一开始我全套跑通了后，就下放给了合伙人，后续我就自己经常跑出去，为流量这边的事情看后端业务，然后付费进群就全部交给合伙人了，等我回来后看到项目就玩的差了不少。

**3.和下属过于亲近**

坏孩三次创业其实都犯了同一个错误，就是和合伙人，员工们住在一起，这其实很不好，虽然初创团队就应该节约开支节约成本，但是人家的那种**初创团队的构成，一般是多个干实事的合伙人一起**，而这次创业，虽然大家都是合伙人，但是小白居多，很多人还没有成长起来，没有独当一面的能力，再加上我经常外出，导致合伙人们之间就走的很近，赚了钱走的近可能还好，但是没赚钱走的近呢？

那可能就是大家一起为后续找出路（跳槽），或者互相诉苦，他们可以和同伴沟通，就是不和作为老板的你聊，这不怪任何人，哪怕你的员工是自己亲哥亲妹，但你两的身份就决定了关系的疏离。

**4.开局大场地，重买设备**

这个错误出现在去年 7 月份开公司的时候，当时开公司是因为一个直播项目，想的很好，我们团队当时有 7 个人，就觉得人也不少，再加上，当时坏孩这个 IP 刚在互联网圈子写出了（19 岁，我赚了一百万[`t.zsxq.com/9DWWj）`](https://t.zsxq.com/9DWWj%EF%BC%89%E8%BF%99%E7%AF%87%E6%96%87%E7%AB%A0%E6%B2%A1%E6%9C%89%E5%A4%9A%E4%B9%85%EF%BC%8C%E9%80%BC%E6%A0%BC%E8%BF%98%E6%98%AF%E6%9C%89%E4%B8%80%E7%82%B9%E7%9A%84%EF%BC%8C%E8%A7%89%E5%BE%97%E4%BC%9A%E6%9C%89%E5%BE%88%E5%A4%9A%E7%BD%91%E7%BB%9C%E4%B8%8A%E8%81%8A%E7%9A%84%E5%A5%BD%E7%9A%84%E6%9C%8B%E5%8F%8B%E4%BC%9A%E6%9D%A5%E7%BA%BF%E4%B8%8B%E5%8F%82%E8%A7%82%EF%BC%8C%E5%B0%B1%E6%90%9E%E4%BA%86%E4%B8%AA%E5%B8%A6%E8%8C%B6%E5%AE%A4%E7%9A%84%EF%BC%8C%E6%83%B3%E7%9D%80%E8%87%AA%E5%B7%B1%E4%BD%9C%E4%B8%BA%E8%80%81%E6%9D%BF%E8%A6%81%E6%9C%89%E7%8B%AC%E7%AB%8B%E6%80%9D%E8%80%83%E5%A4%A7%E7%9A%84%E7%A9%BA%E9%97%B4%E5%B0%B1%E6%90%9E%E4%BA%86%E4%B8%AA%E8%80%81%E6%9D%BF%E5%AE%A4%EF%BC%8C%E6%83%B3%E7%9D%80%E8%80%81%E6%9D%BF%E6%80%9D%E8%80%83%E7%9A%84%E6%97%B6%E5%80%99%E6%80%BB%E4%B8%8D%E8%83%BD%E7%9C%8B%E7%9D%80%E4%B8%80%E5%A0%B5%E5%86%B0%E5%86%B7%E7%9A%84%E5%A2%99%E5%90%A7%EF%BC%9F%E6%80%9D%E6%83%B3%E4%B8%8D%E5%BC%80%E9%98%94%EF%BC%8C%E5%B0%B1%E6%89%BE%E4%BA%86%E4%B8%AA%E5%B8%A6%E8%90%BD%E5%9C%B0%E5%A4%A7%E7%AA%97%E7%9A%84%E3%80%82%E8%99%BD%E7%84%B6%E5%85%AC%E5%8F%B8%E5%8F%AA%E6%9C%897%E4%B8%AA%E4%BA%BA%EF%BC%8C%E4%BD%86%E6%98%AF%E5%85%AC%E5%8F%B8%E6%80%BB%E4%B8%8D%E5%8F%AF%E8%83%BD%E4%B8%8D%E6%89%A9%E5%A4%A7%E5%90%A7%EF%BC%9F%E5%88%B0%E6%97%B6%E5%80%99%E6%90%AC%E5%9C%BA%E5%9C%B0%E5%8F%88%E9%BA%BB%E7%83%A6%EF%BC%8C%E5%B0%B1%E9%80%89%E4%BA%868-12%E5%B7%A5%E4%BD%8D%E7%9A%84%E3%80%82)👈生财有术直达链接，这篇文章没有多久，逼格还是有一点的，觉得会有很多网络上聊的好的朋友会来线下参观，就搞了个带茶室的，想着自己作为老板要有独立思考大的空间就搞了个老板室，想着老板思考的时候总不能看着一堵冰冷的墙吧？思想不开阔，就找了个带落地大窗的。虽然公司只有 7 个人，但是公司总不可能不扩大吧？到时候搬场地又麻烦，就选了 8-12 工位的。

最后的最后，一个在长沙一个月房租 1 万的写字楼就在我的合同上定了下来，盖上了手印，结局自然可想而知。

这次倒闭工作室，又有几十台手机存着准备卖出。。。。。。

**5.带亲属创业**

这一点我相信很多人都踩过的坑，作为亲戚，你道义上要对他多有照顾，但是你作为老板，你又势必要顾全大局，那么很多时候就会产生矛盾，钱扣还是不扣呢？骂人是骂还是不骂呢？亲戚借钱借还是不借呢？借了不还怎么办？

坏孩一共带了 3 个亲戚创业，3 个现在都混的不错，至少要比上班强很多，但是有一个赚了钱后走错了路，现在联系不多。现在想来一个深深的感悟就是：**初创小团队老板，本来就年轻，本来就好面子，本来就很多时候不肯开口谈钱，有时候对员工都要组织好语言才肯开口，何况和亲戚？很多时候就是吃哑巴亏。可能人家在你这也呆的不自在，走了后还要唠你。**

**6.过分依赖付费学习**

**付费学习本来是个需要想清楚再去做决定的事情**，但是有了团队后你就有了个每天烧钱的地方，你没有业务，或者遇到了卡点没有办法解决，心里想着每天都没事做，就容易产生“赶紧找懂的人付个钱搞个业务过来盘活团队”的想法。

**现在这个时代本来就镰刀横行，有实力的当了镰刀，没有实力的赚不到钱也当了镰刀带别人赚钱，有实力的这个项目发挥失常也带你赚不到钱**。你付费一个项目 80%可能性都赚不到钱，那么在现在这个行情下，你**付费更得小心谨慎**。**带公司如此，个人创业亦如此。**

**7.团队没有复盘**

在我看来，**老板和下属之间天然就带着一些隔阂**，那么在这种情况下，你们团队又没有复盘，那整个团队上下全是迷糊蛋，老板不知道团队情况没有办法给出下一步的方案，员工看老板不主动说话，迷迷糊糊的在下面装样子，等着月末汇报的时候甩锅给老板（你，你，老板，你先你别骂我，你自己没给我安排的，难不成让我打工的，自己来动啊?那我为啥不自己当老板？）

以上就是我 3 次组建团队最后都已倒闭告终的部分经验，希望对大家有帮助和思考，如果觉得写的还可以的话，那就感谢圈友们的点赞，都是我接下来前进路上的动力
