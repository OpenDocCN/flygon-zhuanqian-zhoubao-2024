# 二、项目介绍

抖音矩阵掘金项目，是多么高大上的一个名字，让很多人都觉得这个项目一定很不错，比较带有dy的项目都是收益比较大的。比较抖音的日活流量还是十分恐怖的。

先给大家看一下外边发的宣传图片：

![](img/c289e056967dae05e7dc3abb31d86b7a.png)

这个项目的售价已经炒到了1088元这个价格

当你拿到项目就会发现，其实就是简单的短剧/影视推广项目

收入有两种： 第一种就是赚取播放时的广告收益

第二种就是用户购买影片 进行分销

# 三、项目准备

手机：安卓手机、苹果手机都是可以操作的

抖音账号：这个办理副卡 或者申请5元月租的就可以做

营业执照：自己申请或者淘宝代办就可以 几十块钱就可以，经营范围随便

在这里说明一下为什么办理营业执照，抖音不开通企业号是没办法挂载小程序的，有一定的粉丝数量才可以，但是有了企业号就可以0门槛挂载，当然你设置的员工号也可以直接挂载

然后打开快手

搜索影视作品名字，我拿【七月之后】举例子

![](img/61fc04ba2e0a7e0379e67cf8c4a4b706.png)

筛选最多点赞的作品，然后用 去水印 下载下来

![](img/b749ef57962415ab5fd6857281b65af0.png)

做好的视频直接选择作品 挂载即可
