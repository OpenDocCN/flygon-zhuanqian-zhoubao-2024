# 二、自热环境要求

## 自热四要素：

首先自热要从设备、IP、账号、模板，这四方面基本要素入手。也是做自热的基本要求。

## 、设备干净：

首先从设备开始，如果是安卓设备就记住一点：设备要干净！就俩种方式，刷机或者改串。那咱们如何刷机？https://onfix.cn/rom 这个网址是安卓设备刷机的刷机包大全，苹果用爱思助手。你可以在里面找到你相对应的刷机包，直接按照教程刷就行。

那如何改串呢?改串就更 简单了，你可以直接在淘宝或者咸鱼买设备相对应的改串包就行。还有就是你可以去线下实体店，去解决这个问题，设备也可以找我。

## 、IP 网络：

IP 网络我建议大家只选择，使用全国跳+纯净ip。提前插一句全国跳运营商在个别区域跳不动，建议购买之前提前咨询好，也可以找w0222168。

## 耗选择：

这里首选是推荐使用营业厅办的电话卡自己注册，其次是备份老号。

## 模板：

自热模板，目前已更新，四十多个赛道详细的 SOP 模板精细分析。授人以鱼，不如授人以渔。希望你在自热详细分析模板里，能学到得更多的是打粉自热的方式和方法，而不是一味复制和无脑前行。
