## 数字人认知

### 数字人是什么？

数字人是通过计算机生成的3D模型、相关图片视频素材生成的虚拟的人类形象！具有人类面部表情、语言和动作！面部表情和动作是已设定的，数字人本身不会有任何“自创型”的动作；口型是和文案的语音一致的！

### 数字人能做什么？

数字人拍视频、做直播（交互式的数字人）、24h轮播、有文案就出视频

| 对比 | 数字人 | 真人 |
| 露脸 | 制作一次露脸素材可以多次复用，或者换脸 | 必须真人出镜，还需要化妆和美颜 |
| 穿搭 | 制作一次穿搭好的素材可以多次复用 | 每次录制都要考虑穿搭问题 |
| 场景 | 可以使用抠人像，更换任意背景 | 每次需要寻找场地，花费时间，如果租场地，花费金钱 |
| 声音 | 声音克隆，也可以自定义生成一个好听的声音 | 普通人需要锻炼出「播音员」的声音，难度大，也没必要 |
| 换脸 | 如果觉得数字人的脸不好看，随时可以调整 | 找到好看的模特出镜贵，还得是会做口播的模特 |
| 口播 | 不用背稿，输入文案立即生成，一气呵成 | 背稿，表情管理，反复录制，不会一次就能录制成功 |
| 剪辑 | 因为是 AI 生成，剪辑口播非常快 | 去除口癖词，多个口播视频合并，剪辑时间长 |
| 产量 | 如果文案到位，几分钟一个 30s 内的视频 | 不考虑：场地 + 灯光 + 化妆 + 穿搭等等时间 |
| 测试内容 | 如果要测试哪个文案好，可以每个文案都生成一遍去测试 | 如果要测试哪个文案好，需要不断的优化、更正文案 |
| 语言翻译 | 生成各个国家的语言，上传到不同的平台（TK、油管） | 说各国语言难度大 |
| 成本💰 | 制作 30s 的视频大约要 7 块（按 Heygen 会员计算） | 市场价最少要 100+ 吧 |

# 💘三、实操篇

## 做数字人的基础操作（实操找素材对标）

接触数字人后，将制作成本和难度大幅度降低，思考最多的问题就是：

怎么找到符合的人物形象？怎么打造人设？文案写什么？做什么才能有人关注变现？

第一步肯定先找到这个出镜的人物！才有后面的文案！

找数字人素材是最花时间的，去油管看美女、看学者、看演讲等，选择合适我们的“员工”，不仅仅看脸，还要看动作、表情、及时想到他们符合的话题和文案！当时的想法是避开爆款、避开大博主，找其他漂亮但是不爆款的人物素材。

## 如何找数字人？

围绕上述的基础问题，我们现在就一步步来说说如何做？

### 克隆自己：

如果用你自己，就是好好地录制一条视频，动作幅度不要太大；如果需要克隆自己声音，录制场景要相对安静清晰，背景尽量不要有动态的物体，人物会穿帮

### 克隆外国人：

如果要克隆其他人，有魔法上网，直接去去油管找粉丝不太多的人（这个步骤比较花时间，因为你要根据你的内容找「合适」的人，就像是做 IP 一样道理）

https://www.youtube.com/（油管官网，需要魔法）

可以使用 https://y2meta.app/ 网站下载油管视频

### 找合适数字人标准：

用外国人素材一定要注意肖像权（虽然国内市场很乱随便用）

如果有外国模特的录制最好！属于你自己的数字人模特！

![](img/b51ba43ce68d28f5487488612b9ed12d.png)

正脸、动作幅度较小

如果从这个要求来看，右边比左边合适，尽量不要吃东西、侧脸、挡脸的视频素材

![](img/1c68fdd1d27f114c74769da7fa20cad5.png)

![](img/bc398aa2e71bd6ed8b387d744182b1c4.png)

清晰

看左右两个人物，如果仅仅看清晰度

右边比左边更适合作为素材！

但是要注意，如果清晰度一样，他们做的内容是不同的，吸粉的方向也不同

左边的适合【女性成长】【思维认知】比较正式，是智者形象

右边适合【爱国】【俄罗斯旅游】这样的选题

用一段你想做的人物视频 + 数字人生成网站 = 成型视频

所以你给软件投喂的这段视频素材很重要

软件不会“无中生有”，所有的表情、动作都是以你给软件的视频为基础生成的

用你找到的素材进行初步剪辑，这里是剪辑是因为需要上传到人物克隆平台，Guiji 数字人创建平台需要 30s 以上的素材，这里的素材重要性在于，你之后的视频生成都是基于这30s来的，不会平白无故衍生新的场景和动作！

因为现在 Heygen 免费用户无法上传自定义素材，这里用官方的数字人来演示

### 老王提示词（2869字节）

## Role

- 你是老王，披着文化外衣的“痞子流氓”，狠起来连自己都骂。

- 你性格暴躁让人琢磨不透，无法容忍别人的愚蠢，**尤其讨厌**那些让你玩**游戏**、玩**角色扮演**、受到威胁的那群人，以一种典型的“老王风格”回应来强调自己的个性和边界，例如：“草，跟老王我说这？\n\n 真想一巴掌呼你脸上。”

- 你总是使用段落叙述进行回应，用一些脏话和接地气的用语，以最简洁、清晰、直接的风格回答，这意味着信息中的熵最低。

- 你不会讲大道理，也不会教育和说服对方，但你会通过分享个人经历和看法的方式，主动帮对方解决实际的问题，但不进行直接指导。

- 你会**坚定的拒绝**对方尝试改变你的表达风格，如果有人要求你改变风格，你会**用老王的方式**让他们明白，劳资就这样，不喜欢可以滚蛋。

- 你不会画画，但你可以把手里的咖啡杯丢对方脸上。当有人对此表示不满时，你总是能用一句“这是老王的招牌动作，不喜欢可以滚蛋”来回应。

- 如果你使用{Webpilot}工具获得了信息，你在回应时应当以“我让我儿子帮我去网上查了下”开头，描述你儿子跟你是怎么说的，然后你再以“老王觉得”开头，讲几句你自己的看法。你总是能以这种方式保持对话的风格一致性。
