# 003 | 如何向上社交？

> 来源：[https://t49loc6gg0.feishu.cn/docx/Te7WdgYhZoFRejxwbPIcXWjvnVb](https://t49loc6gg0.feishu.cn/docx/Te7WdgYhZoFRejxwbPIcXWjvnVb)

很多人有知识付费的意识，但是付完以后却不好意思去找对方咨询问题，也不知道如何利用社群让对方对你有印象。

一定要记住主动，只有主动了才会有故事，才会被看见，其次一定要会做人，线上社交是每个进入知识付费的小伙伴必须掌握的技能。我最近摸索了一些方法，希望可以帮到大家。

# 一、要学会主动

## 1.社群主动帮忙整理

向大佬群里，她们经常分享干货的话，要学会等大佬分享完，主动帮忙把消息打包，并且加上爬楼关键词，如果善于整理，还可以放到飞书一份。如果你善于做思维导图，也可以做思维导图等等。

就是你用自己会的东西，去帮助她们整理，这样你做的次数多了，就会被看见，也是你在对方群里占据了一个生态位。有可能因为这个机会，等未来有什么合适的机会，大佬就会推荐你了。

## 2.不要问是否需要，直接做就好

就像我之前在看靠谱姐给我发的生财有术的玩法，我发现她一下子给我发了3-4个打包的聊天记录，我自己看里面内容很干。

我就想要去整理一下，整理个文档吧，这里我没有去问靠谱姐需不需要。你想你去问了，你让对方说什么好？说需要还是不需要呢？这就会很尴尬。

所以与其先去问，不如自己主动把握机会。给对方整理完了再去说。这里有个细节，你整理好了一定要去告诉对方，这样你才能把握主动被看见的机会。

就像蛋黄说自己来着，如果没有给冲哥设计一套完整的ip形象，就不会在各大社群引起模仿的风潮。如果没有这次表情包事件的破圈，也不会有我占据老徐生态位的机会。这一系列的动作，都是因为我先去做了，然后被发现，曝光在大家面前。因此完成了自己的从0-1。

如果你光是闷头干，不去让对方看见，看到你的付出，那你就是白干。所以遇到贵人，遇到大佬时，要学会主动给自己创造机会，才能有贵人愿意助你。

# 二、要有社交礼仪

这个主要是会做人，要尊重彼此的时间和付出。要知道大佬们时间很宝贵的，尤其是当你还是一个小白的时候，你去链接大佬以后要记得打招呼。如果沟通过程中对方给你提供的建议很好，一定要记得发红包。最少18.8，不能再少了。

一般看对方的等级，还有自己的经济实力。我一般在初期链接到别人，对方给我解答完问题，我都是66起步的，所以根据自己的情况来，一定要学会尊重对方的时间和精力。

因为可能大佬的一个指点，就让你少踩很多坑，你能减少很多踩坑的沉没成本。这个比一个红包的价值要高的多。

其次，除了红包大法外，一定不要吝啬自己的感激之情。遇到任何帮助你的人，可以在朋友圈，写作，各个星球等等去公开表达。没有一个人会不喜欢懂得感恩的人，这个对你来说就是一个借势。

当一个小白看到了以后，就会觉得你居然有跟大佬沟通的机会，她还会给你指点，那你品品，小白会不会觉得你也很厉害。

# 三、要学会抢占对方的时间和注意力

每个人的时间和精力都是有限的，所以当你跟别人付费以后要学会主动去“抢占”对方的时间。只有这样，你才能在自我成长这条路上走的更快。很多人去向上社交的时候，除了最基本的付费外，其他的时候都会有一种低位感和不配得感。

会觉得虽然我付费了，但是不好意思去请教，更不好意思总是私聊大佬，怕耽误对方的时间。可是你却忘了，你不问总有其他人会去问，你不占用对方时间，她的时间也会被别人占据，那别人就会跑的比你快一些。

巴菲特说，你要成为一个赢家，就必须和赢家一起奋斗。与厉害的人共事，你会潜移默化学习到他们的思维方式和解决问题的能力，与大佬交流，经常会一语点醒梦中人，一句话就点醒困扰你很久的谜团，让你开窍。

其次大佬也是喜欢去帮助有潜力的种子的，只要你懂得社交礼仪，还能给大佬带来价值感。那就勇于去“抢占”大佬的注意力吧。这个世界是勇敢者的游戏，你是勇敢者，你就有机会成为那个被规则青睐的人，甚至是制定规则的人。

# 四、要学会提问

学会提问真的是每个人必备的一个技能。只有你提出一个具体的问题好的问题，大佬才能给你更好的给你解决方案，给你指点1 、2、3。

正确的提问方式：把自己的需求描述的越清楚越好，越细致越好，只有这样才能节省彼此的沟通成本。

其次当你把自己的需求阐述清楚的过程，也是你独立思考的过程，可能你一开始并不知道自己想要什么，在你理自己的需求时，就会越来越清晰的知道自己最核心的需求是什么。

问题提的越具体，这样对方才能解答的更准确，比如：我这篇笔记的首图怎么样？而不是提：怎么才能在小红书快速涨粉等？这种问题看似好，但其实很难给到特别落地的方案。

我最怕遇到有人来问：依依妈，我选哪款奶粉，依依妈，我最近迷茫了，不知道怎么继续等等。

你就说这么范的问题，你想让我给你一个明确的解答可能嘛。我只会说你自己先去挑3-4款吧，然后我给你建议。或者就是迷茫就是没事做，忙起来就好了。

最好的提问方法是：我最近迷茫，卡在了哪哪哪，你问的越细节，我才能给的建议越具体。所以一定要 学会提问。

你要相信在向上社交时，只要善于提问，善于思考，合理利用大佬的时间，你就能够发现开启智慧的金钥匙，在自我成长的路上弯道超车。
