# 和陌生伙伴，每周六早晨的分享复盘会，让我收获价值千万

> 来源：[https://t054j8sfmp.feishu.cn/docx/Is4kdkt5lo9ospxZTq6cvP0vnSm](https://t054j8sfmp.feishu.cn/docx/Is4kdkt5lo9ospxZTq6cvP0vnSm)

经过了一个月的西藏之旅，从高原回到了地面，恰逢刚开完早分享复盘会，还在休整期，刚好也是全年复盘和规划的时候（其实已经做完了，从u西藏回来之后有些调整）。这次主要分享每周的分享复盘会对我这么几年带来的价值，和搞钱看似没有很明显的关系，内在又千丝万缕密不可分。

和生财的小伙伴们相比，我做的真的太土了，超土超土的传统行业。得益于早起复盘会，开始转型互联网。虽然按照爆文写作的要领，应该用具体的数字来达到冲击力，我从中的收益早已经不能用具体的数字来衡量了。时间紧的伙伴，可以只看有颜色的、重点标注的、其他可以说都是废话。

项目选的好不好，做的怎么样，基本都是看带头人。我选择也多，再加上是处女座，圣母情怀严重，就极其容易焦虑。从22年的7月开始，回顾这540多天，每周的周六早晨都有分享复盘会。

包括在做爬宠直播的时候零点2点下播，也是6：30起床开会。今年的大年初一都没有停，一起互道早安，一起分享复盘。在这个过程中，我渐渐清晰了思路，开始有“时间”去做自己想做的事情，生活不再只是搞钱，还取得了很多额外的突破。

开会的这些伙伴们，其实我并没有见过，虽然有些都在北京，我们也并没有见过面。大家是通过另一个组织建立的联系，这540多天里，也充满了烦躁、愤怒和无数次想放弃的念头。但相信我，任何一件小事坚持下来都伟大且有意义的，哪怕是每天按时吃饭这样的小事。

## 起因

说白了我当时的觉知告诉我需要。

我是一个没什么脑子、靠体力挣钱的人。虽然知道复盘的重要性，在工作中也复盘，但仅限于赚钱方面。生活不只是搞钱，还有很多其他的。对于我自己大的人生规划更多的是处在想一想的阶段，或者说是一直焦虑但一直是想起来就想一下，和复盘这个词根本没啥关系。所以我需要利用团队里的力量让我想明白关于我人生的规划。

# 对我带来了哪些好处？

搞钱就不用说了，不再像以前一样忙碌，有了众多的时间尝试更多的项目，聊点基础的。

## 1、改变了很多认知

从知道到真正去践行，至少花了3年的时间。

当时我认为搞钱最重要，这些都和项目无关。直到意识到我总是在处理紧急事件，对未来发展并没有时间去判断。

当时我认为早起会降低效率，直到2周后，如果没有早起就会让我感觉一天忙碌且没效率。

当时我认为和这些不相关的人开的分享复盘会完全没有意义，直到我看到记录中的我在不停的改变。

## 2、极高的心理舒适度

我有个朋友特别有意思，说如果自己觉得烦闷，就听听别人的惨事，总有比自己惨的，这么一看自己的事也就不算啥了。

听到其他小伙伴分享工作中的乌烟瘴气、家庭里的烦心事，看来大家都一样，别人也没比我过的好哪里去，遇到这些问题基本上都是这么解决的。

即从贴近生活、具体的案例，让自己内心得到平衡。又在这个满足了窥探的过程中，慢慢的更专注于自己的生活，更少的受外界影响。

## 3、最重要的是身体好了

有位老师讲：智商不在一个水平的人根本构不成竞争，最后拼的都是体力。

智商是动态的，所以朋友们完全不用担心，仍然有可能和雷军等一线大佬竞争。前提就是先要身体好，能够干的到那时候。

我第二次创业做项目的时候，就仗着年轻用力过猛，在高峰准备冲击全国市场的时候面临了继续干还是保生命的选择，那个时候我才明白老师讲的那句话的意思。

很多时候开完早会，时间也算早，就去楼下跑跑步，就这样身体慢慢好转。

# 其他奇奇怪怪的问题

## 为什么一定是早会，而不是晚会或者白天的某个时间段？

马云说过一句话“睡前道路千万条，醒来还是走原路”。

晚上夜深人静的时候，看似美好，实则适合小酌一下，感慨下人生。感慨和复盘的关联性也不大，白天发生的各种事情或多或少的都会影响心情，从而带着更多的评判思维影响判断。如果恰逢心情爽，激动的规划完以后激动的睡不着觉也是常有的事情。但是一觉过后，大概率都不能很好的执行昨晚的计划。

白天的某个时间段更难坚持，时时刻刻会发生各种念头和事情打断，比如电话、微信。可一般早晨不会有，有可能你想和别人打个电话都不知道打给谁，就算和别人发了微信，也极小的可能迅速获得反馈。

所以，在早晨，没有别人打扰，所有的事情也经过一夜的沉淀，心情影响指数降到最低，更适合思考问题和做决策。制定了方案以后，白天可以尽量安排时间去推进。

## 早起不了怎么办？早起一次，没精神一天怎么办？

我也不是晨型人，而且极爱晚睡，并且觉多。就算早期，睡得也不早，但是中午会补觉。

刚开始确实挺痛苦，坐起来得缓个半小时，结果又睡着了。百分之90的时候，就算睡着后又醒了，也是比别人早，这个时间段也是清净的好时间。习惯成自然后就好了！

## 为什么要和别人一起开，怎么开？

我也不相信我自己能坚持那么久，就肯定需要小伙伴的力量。像现在很多的训练营都是这个逻辑。也是陪伴社群的一种，只是链接很弱，纯粹的弱链接。

复盘和分享都是自己讲自己的，不对他人进行评判。流程后有自由交流和分享时间，如果恰好听到感兴趣的内容，可以继续沟通。有时候也会对热点呀聊上几句。我觉得在这个过程中，不评判是非常重要的，就会让人感觉很舒服。像很多年前的私董会，提问、逼问、给建议，如果心理不够强大确实不太容易接受。

更像是自己做着自己的复盘分享，同时听着别人的，还挺能找到生活同感。

# 结尾

大早晨有感而发写了这么多，祝愿生财的小伙伴都能够在2024乘风破浪，生财有术。如果欢迎小伙伴链接一起搞钱，拯救一下土土的我。
