## (53 赞)总结下健康长寿指南

作者： 亦仁的收藏夹

日期：2024-03-14

总结下健康长寿指南:

长寿 = 运动 + 饮食 + 睡眠

运动 = 心肺能力（慢跑 间歇冲刺跑） + 力量训练（深蹲，负重爬山）+  柔韧稳定性练习（普拉提，瑜伽）

饮食 = 间歇性断食（少吃）+ 低糖 + 高蛋白

睡眠 = 昼夜节律（早睡早起）+ 8-9 小时夜间高质量睡眠 + 午间小盹
