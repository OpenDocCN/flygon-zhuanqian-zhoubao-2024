# 一、注册注意事项

## 设备选择

我目前自营选择的是 iPhoneX64G 无面容，600 元左右

设备选择只要不卡能流畅即可，安卓机注意选择矩阵工作室淘汰的机器，因为在小红书无论怎么发都是浏览量 1 如果遇到这种情况大多都是设备的问题更换设备和 IP 即可

## 号卡选择

主选厅卡，广电，移动，电信，联通，一个人能办 20 张，找个营业厅的业务员直接上门办理即可

（有的地区一个运营商三个月内只能办三张可拿家人的办理）

部分注册卡也能注册有流量但是大多数没流量可自行测试

## 注册环境

账号注册不能连续一天一个地方最多注册三个号否则跳验证风控账号发笔记没流量，可换地方换 IP 注册

（例：在 A 地点半小时注册三个账号，换到 B 地点再注册三个账号不会触发风控不会账号限流）

注册后完善资料后发 2 篇互动笔记进行数据测试（不要连续发隔 2 小时）如数据超过 200 可进行开店或实名，如无数据不佳隔天在进行测试如隔天小眼睛还是不超过 20 可检测设备问题或者注册时候的环境问题，如检测无问题继续发爆款互动或者降低小红书版本即可。

**注：**注册时不可批量同地点同 IP 进行注册，需要进行换地点换 IP 进行注册才不会触发风控，如果时间多可同一地点注册一天内注册三个账号即可，每天注册不会触发风控。

## 企业店（个体店）

企业店开通需要缴纳 600 元认证费，但是企业店能够绑定主理人，主理人能绑定 3 个号（需实名）一个企业店加三个主理人等于四个号绑定的三个主理人的号出单都在企业店客服也统一在企业店（便于管理）账号的流量互相不干涉，即使企业店主体被限流，主理人账号流量正常一样可以出单。

企业店认证注意事项：

同一时间同一天同一个法人不能认证超过三个店铺，超过三个会跳风险，需要两个月后才能认证

（一个月内慢慢注册就不会跳风控显示，但是部分地区也会提示风控，最好可以多注册几张不同法人的营业执照）

企业店注册昵称前面可加字幕避免店铺名重复延长审核时间

一张营业执照能注册 2 个店铺第一个店铺审核第二个店铺快速通过

## 个人店

个人店一张身份证可开多家，但一般只开 25 家否则会跳支付风控，个人店一家店需要一个实名，注册过程同上！

# 二、起号注意事项

## 店铺分配

企业店

一组 3 个号，一个运营可管 2 组号

个人店

一组 3 个号，一个运营可管 6 个号

（注：以一个运营每天 8 小时工作量计算从内容产出到内容分发，成熟运营可增加账号设备）

人员充足把剪辑素材分发单独小组化做成标准化的运作流程

（例：剪辑每天需要负责剪 10 组素材，素材需要找到 20 个爆款标题对标账号，分发负责手动发送主页观察数据反馈数据给运营优化并执行优化动作）

新开店铺一般需要经历权重期和提升期再到爆发期一般 7-20 天

（注：有的账号注册后权重高一发都是几千上万，有的账号只有几百，账号基础权重高的账号很快就能出爆款，但是这种号注册时几率很低大多数都需要提升一下权重再带货）

## 账号起号周期

第一天到第七天

账号权重低的账号不宜发太多笔记账号笔记数量控制在 3 条以内，具体看数据是否能超过 200 再慢慢增加账号笔记数量，前面七天是一个不断的测垂直类目品的时期，如果遇到这个账号发的某个品数据高可以多发几条看看，出现爆款的几率高。

前面七天内，尽量选择垂直领域内的品去进行测试不要什么都测，如果什么都测账号不垂直账号是起不来的。

第七天后

前面七天基本测出这个账号适合哪个类目，在根据这个类目的爆品进行测试，及时监控对标异常值和垂直类目的异常值或者爆款及时跟，提升爆款率。

# 三、运营注意事项

## 账号限流

账号限流一般为几种情况

1.同 ip 多账号，被限流      解决办法：换 IP 降小红书版本尝试

2.发布多条骗点击内容      解决办法：等 14 天处罚后恢复正常

3.发布擦边违反平台内容   解决办法：换号

4.突然账号被限流            解决办法：找客服申诉多次申诉 3 次以上上报工单恢复

发布时间：

建议时间错开发布不要同一时间发布，不设定时发送，每个账号爆款时间不同需要测

设备环境：

一机一卡厅卡直接开流量，如果没流量就办理流量卡全国跳 IP，在手机 50 台以上不要把手机都放在一起，有几率会影响账号流量。

**做矩阵账号只是消耗品，算出一笔能够盈利的账即可**
