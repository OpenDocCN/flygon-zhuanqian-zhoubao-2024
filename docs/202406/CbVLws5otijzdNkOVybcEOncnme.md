# 为什么你的小红书一直做不起来？其核心原因是以下三点...

> 来源：[https://hadad3oapn3.feishu.cn/docx/WqzOdQtqKoAbfUx46Ehc4VALnWh](https://hadad3oapn3.feishu.cn/docx/WqzOdQtqKoAbfUx46Ehc4VALnWh)

看过那么多的账号，我发现其实大家小红书做不起来的原因都差不多，都是有共性的，其实主要原因就是以下三点

# 一、太把自己当老板了

在你发现一个新的方法的时候，自己不亲自下场研究策略，一味地让员工去做去试。

这里我说亲自下场不是说让你去做剪辑、做图文这种没有增量的工作，我举一个我自己的例子大家就明白了

我在 22 年6月中旬的时候让我的员工去做小红书，但是半年过去了一点起色都没有，到 22 年年底的时候实在没有办法了，我自己一个人把小红书所有的书全部看了一遍，我用了整整两个月的时间把自己关在屋子里研究小红书的底层逻辑，把小红书的所有后台的玩法，所有整体的玩法，小红书上所有的博主的资料、视频课，全部看了一遍，并且结合自己的思路，做了一套完整的SOP，完完整整的落地下去，结果就是我的自营盘项目一下子就做到了小红书本地同城类目的头部。

在做这个事情的过程当中，我从来没有做过图文，我也没有做过剪辑，但是我有策略，我知道爆款的图文应该怎么做，什么类型的图文是好图文，什么类型的视频是好视频

当时正好是在过年期间，我找了两三个人在我公司来做兼职，带着他们亲自下场，把小红书一点一点给磨出来了。

但是在做这个工作的整个过程当中，大家问我干过活吗？我其实不干活。

做图？我迄今为止，一张图都没做过，剪视频我也是一个都没剪过，我也不会这个东西，但是我能把这件事情能做出来，核心很简单，亲自下场干，并且干的有策略。

我们在做任何事的时候，尤其我们小公司老板的，做什么事情都要有增量有复盘，你如果自己都对平台的玩法、底层逻辑都不清楚你怎么去带领员工往一个正确的方向走呢？员工出现方向性错误，公司又应当怎么办呢？

# 二、等靠要思想，把事情交给员工或者代运营

为什么你的流量看着很大，但是变现却基本为0？

小红书拼的是内容能力，很多人的流量做的好，但是却没有多少变现

其实根本原因就是因为很多人懂要做科普内容而不做转化内容

什么叫科普内容？

就是别人听，觉得你讲的好讲的对，觉得你讲的这个内容对用户现在甚至是未来的一段时

间都有作用，对你的笔记进行点赞收藏评论等一系列互动行为。

但是！

别人从来不会给你私信和你链接，更不会主动私信你给你发消息，想给你产生链接，这种类型的笔记就是科普内容。

如果你做小红书只会出内容，但是没有转化，这种就是内容能力缺失，那么靠普通员工到底能不能解决这个问题？

我跟大家很负责任地讲，很难，几乎是解决不了的。

为什么这么讲？没有一定的社会阅历，很难做出真正的好内容。

话又说回来，什么叫做好内容？

那是你我之间的事情，你愿意为我买单，我做出的内容是愿意让你买单的内容。

但是这个点，看似很小，实则有很大的讲究，哪怕他是一个公司的编导也好，普通的公司的员工也罢，如果他不太懂人与人之间的微妙的联系，不懂人性上的东西，是做不出来好内容的。为什么这么说？

好的内容一定是人性上的痛点营销，它没有一味的去介绍他是卖什么什么产品的，他的产品有多么多么的好，而是换种方式，从用户的角度去讲解，让用户感受是真真实实的是替他去考虑的，拉进和用户之间的心理距离，人与人之间的接触，你接触多了，你就会发现，他的底层诉求可能就一两个，所以你做内容只需要去解决他们这几个尤为明显的痛点，就可以通过笔记吃下他们，带来转化，甚至是成交。

所以说，这就是我们看到为什么很多人说他这个做得有多好，但是你去仔细看他自营盘的时候，你会发现他做不出东西来

因为什么他不具备内容能力，他也只是在人云亦云能说出一些内容来，但是它不具备最重要的内容创新能力。

大部分的员工想问题是一叶障目，而不是说他看懂了一个全面的一个东西，即一件事情的底层逻辑。

你知道大部分的员工是一个好的执行者，如果这个人到一个好的执行者层面，说实话我们就算请了一个好员工。

但是执行者跟有策略的人跟合伙人的高度那是两码事情。所以就是说我们有等靠要的思想去寄希望于员工，把自己太当老板这件事情根本就是错误的。

我最近的工作重点就是推翻以往的公司内部方法论+训练公司所有人的能力，尤其是内容能力，等这件事完成好之后再去开新盘，那样效率快且准。

# 三、不要相信所谓的某方

很多老板非常迷信所谓的某方啊，觉得只要是他们说的都是正确的，严格按照他们的走，但是做了一段时间之后发现效果也就那么回事。

我跟你说核心点在哪里啊？

其实某方也是一帮普通员工，他们给你讲的所谓的小红书的知识点全部是概念性的东西，又或者是你去小红书认真搜一搜都能搜到的知识点，根本就不是实操落地的东西。

我跟大家讲这么一个点，某方那些出来跟你对接的人，他的工资无非就是 8K 到12K。你想想他有多高的能力吗？

他能指导你真正把你的小红书做起来吗？

他有这个水平，他何必去上班呢？

是不是这个逻辑点？

所以不要去过分相信这些。当然我不是要你任何人都不信，只是说你等号要的思想不要太重，不要去迷信于任何人，还是要自己懂业务，不然很难拿到结果。

投流投不出去，很多人调整投流，其实核心是内容出问题了。代投只懂技术，不懂内容（行业内容有壁垒 ）

说个话外题，普通人做账号，刚开始没有流量没有粉丝没有变现，就会产生一系列的自我怀疑，以及做了很多努力却颗粒无收的挫败感，信息会动摇，会玻璃心、负能量全部爆发，这些都是很多人坚持不下去的原因。

你或许会听很多大V跟你讲做自媒体的技术和思维，但是大家都忽略了一个事情，就是没拿到结果之前，保持一个好的心态，有效的坚持。做事的方法和技巧都可以学，但是内心拥有足够的信念才是成事的关键。

当你熬过这段黑暗的时期，你可以迎来光明。但是事实上，能熬过这段时间段的人少之又少。
