# 2.拉高笔记数据

但凡是了解过博主变现模式的，都知道

接广告不难

难的是怎样维持数据⚠️

让自己持续的能接到广告⚠️

## 2.1内容形式

我把适合个人做的笔记类型分成这几类

干货：对专业性的要求高，难写，大多数品牌喜欢，因为干货难被人听懂，所以数据难掌控

硬广：对专业性没要求，好写，品牌喜欢，除非品牌砸钱投流，否则数据难掌控

软植入：对文案要求高，有一点难度，大多数品牌喜欢，数据好掌控，

合集：对文案要求不高，选品要求高，大多数品牌不喜欢，数据好掌控

测评：需要做的准备工作多，大多数品牌喜欢，数据起的慢，但通常不会差

·

每种我都测试过，但现在最常写的就是合集，干货软植入，测评类

因为在不额外砸钱投薯条的情况下，这几个是最好掌控数据的，就像我的数据是不是大起大伏，算是很稳的，至少也能在300赞+，所以只要报价不是太高，被品牌方选中的概率还是挺大的，

而和我同量级的博主，要么因为总是发硬广数据不好，要么总是发不报备的软广，被平台抓到判违规，所以这两年都在降价接单了，而我不光没降价，还从1000慢慢涨到了11500。

所以选好笔记类型对稳定笔记数据，也是很重要的

## 2.2打磨文案

用户喜欢的广告都是软到看不出广告的

（除非是自带流量的明星大v）

so！想要笔记数据好，就千万不能直接复制bf里面的卖点

文案没啥技巧，就跟写作文似的，多看范文，多学多练，

如果跟我一样，从小作文就流水账😅

那就去看100条w赞以上的广告，就是右下角带赞助标那种，0.75倍速看，

我自己用的笨方法就是每次看到别人写的好的句子，就会逐字逐句编辑在文案收集库里，这样下次自己写的时候可以直接拆解别人的文案，套用在自己要写的产品上。依葫芦画瓢嘛。

## 2.3关于拍摄剪辑

### 封面

在小红书，有个封面点击率的指标。用户期待的是点进去的内容

在抖音，是2s跳出率的指标，用户永远期待的都是下一条

所以小红书用户是很容易被封面吸引进去的，即使内容不咋地，也会多看几秒（毕竟，来都来了是吧～）

封面可以多拍几张，多换花字来测试流量

### 我的剪辑思路

先粗剪+美颜美妆（把不卡壳的口播剪出来）

再添加不口播的素材（画中画模式）

识别字幕（再确认好素材之后）

把重点字用花字标注

隔10s左右加个特效（蒙板/转场/表情包都可以）不然一直口播会让人没耐心看

加背景音（背景音音量建议调20左右，口播声音调200）

加音效（音效要多！尤其是以口播为主的视频，基本上3s内就要有个～能让平平的口播节奏变得灵动，可以配合花字或者手部动作的出现，加个合适的音效）

# 3.边接广告边涨粉

我的号10条里都9条都是广告。然而一点都没有影响我涨粉+变现。

一是因为我的笔记数据很好很稳～

二是因为我很挑，对广告很挑

## 可以接的广告

1.有预算投流的品牌（可提前和pr确认，或者去搜搜这个品已经发了的广告数据咋样）

投流是品牌方花钱给你的笔记投放的意思

很香！！很有可能会把原本数据不行的笔记，拉到千赞以上～

2.大品牌

比如说母婴类目的帮宝适，飞鹤

美妆护肤类目的欧莱雅，兰蔻

家居类目的美的，戴森、3m

...

大品牌是自带流量的，不仅会提高搜索流量的概率，而且还能让更多大品牌pr看见你

## 不要接的广告

接1条10000的广告比接10条1000的广告，可要香多了

像这些我都是直接pass的～

❌没几个店铺在卖的杂牌，小牌子

❌图文直发（伤害号易违规）

❌医药类（专业性太强）

❌不符合消费观的（十几块钱一瓶的婴儿水啥的。。。智商税）

❌差评多的（会让路人转黑）

⚠️日系品牌（没原因，就是不喜欢！！）

⚠️脚本已经写好了，但品牌方硬要修改里面提及的缺点的（这个看个人）

⚠️返点要求离谱的

（返点的意思是：报价的一部分需要返还给pr。比如我报价11500，返点20%，就需要返给pr11500*0.2=2300。）

只要报价合理，正常返点10-25%。撑死了这么多！除非是会送大额的产品，比如我一开始提到的6k+的学习桌。返28%我很愿意😬

只有那种很水的买来的号..他们会给30%-40%甚至是50%很高的返点....我们大大方方做内容的，宁缺毋滥😠
