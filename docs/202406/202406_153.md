最近用 Coze/扣子制作的 Agent 智能体，连续两次上了官方热门推荐[呲牙]。

同时还获得了通往 AGI 之路举办活动的“最酷 Agent 奖”！

深刻的感受到，Agent 是大模型的 Plus 版！如果大模型是 1.0 时代，那 Agent 就代表了 1.5 时代。

通过插件和工作流的应用，大幅提升了大模型的能力。大模型能搞的，Agent 都能搞，大模型不好搞的，Agent 还能搞。比如：基于私有知识库的问答机器人；一键提取抖音文案并转成小红书内容并配图。
