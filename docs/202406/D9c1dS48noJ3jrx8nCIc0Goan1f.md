# 花了一个星期时间，用AI制作了一条《AI在武汉》

> 来源：[https://uqobyhlq6i.feishu.cn/docx/D9c1dS48noJ3jrx8nCIc0Goan1f](https://uqobyhlq6i.feishu.cn/docx/D9c1dS48noJ3jrx8nCIc0Goan1f)

大家好，我是西风残雪，一个从事8年的影视后期从业者。

前段时间有看到央视《爱我中华》ai做的宣传片，感觉AI视频是一个大趋势，最近一段时间花了一个星期，每天除了吃饭就是出图，抽了将近2000张图片，做了一个《AI在武汉》。

这一个星期经历过迷茫，担忧，放弃，到最终成片的释然，虽然我有做过各种视频的经验，但是用AI做视频，完全没有尝试过，开始的时候完全是懵的状态，后面在各处找资料，B站 YouTube，按照视频一步步操作，中间有好几次想过放弃，但是觉得AI视频是一个大机会，最终坚持下来了，当然整个成片有很多的问题，包括人物的脸部 腿部等等。

## 出图

将优化后的文案，导入到GPT中，生成画面提示词：

![](img/bfb0d9f382bd8afccd4ea3752db55713.png)

画面提示词同样用了GPT和kimi生成了好多稿，然后挑选对应的文案和画面描述比较好一些的提示词，最终组成上面的画面提示词

用画面提示词在gpt中dalle里面生成图片，用dalle生成图片会比较差，最主要目的是能大概有一个心理准备，大致了解视频想表达的意思，不会导致最后在Midjourney中生图时候抓虾，偏离视频表达的意思。

## 视频

ranway账号可以在淘宝购买，一个星期26元钱，可以无限生成，ranway可以双开，打开两个网页同时生成，如果是共享账号，白天使用runway会非常卡，我是在凌晨3-7点使用runway，避免了使用的时候总是抢不到，用了2个晚上生成了所有的视频。

![](img/9be10ad5c135dc0c8fa7411c0cd10416.png)

### #总结

1.开始做这个视频之前，一直在抽图片，抽了差不多快2000张，最后视频做下来，大部分没有什么用，浪费很多的时间，尤其是有关抗战时候的图片，mj生成不了有关我党的衣服图片，生成出来的图片要么是苏联时期红军图片，要么是德国军队的衣服，或者是国军的衣服，不论是垫图还是提示词，最后效果都是很差，最后只能用雕像代替。

2.由于电脑的显存比较小，带动不了StableDiffusion，制作不了图片中插入文字效果和全息宇宙，另外对这个不太懂，还要再学习然后制作，花费的时间会更加多，因此就没有做这两个

3.先完成再完美，做视频之前，觉得这个很难，mj不会，runway也不会，视频出来后，觉得没有想象中那么难，做了再说
