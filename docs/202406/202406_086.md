慧敏小红书无人直播纯分享
___ 拼命想要挤入电商赚钱的个体户

```
   大家好，我是慧敏，一直想要在电商圈赚钱的个体户，加入生财 3 年，每次看到各位大佬写文章，总觉得太厉害，一到自己做的时候，就感觉这个也不行，那个也不行，只能投去羡慕的眼光。最初一开始是跟着木易师父，做视频号直播，对于最初创业来说，很不错了，20 天，夫妻档，一天播 3-4 个小时，赚了十几万；去年一直在做小红书笔记带货，打出多个爆品，吃到了最初笔记带货的红利期，单条笔记百万流量以上，变现能力也不错，就算是 200 的粉丝账号，变现也有十几万。去年线下见到书豪，和他聊了一下这个，他回去之后很快也做起来了。
   今年重点在打小红书无人直播，如果是个人完全是可以做的，不需要打付费，纯自然流。为什么我一直是个体户，因为觉得管理麻烦，而且小红书的这个项目不适合公司去做，门槛太低，感觉随时会跑路，项目的周期也不知道有多久，主打一个当下能赚钱，就先赚钱了再说。
  在离职创业的路上，踩过很多坑，有说抖音女装流水打 1000 万的，要合作，结果抖音什么也没干，最后骗我的钱租房，还管他一家三口的一日三餐。今天我就纯干货分享，小红书无人直播怎么做，还有注意事项放在最后。
   电商上的数据，是无数个日夜换来的，你说值得吗，看到赚钱就值得，还有多个是枯燥测品阶段。我觉得做电商是个人可以改命的机会，这种玩法已经是轻资产了，低门槛。如果你也在迷茫，可以仔细看下操作 SOP。
   第一次写文章，不太会，有不懂的可以留言，我再补充。
```
