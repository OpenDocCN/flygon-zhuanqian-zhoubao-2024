TOP 销售如何借助工具制定销售策略

```
  作为销售个人，你有制定销售策略的习惯吗？如果没有，建议你一定要继续看完这篇文章，也许它能扩展你的销售思维，用更科学的方法工具，帮助你提升销售效率，TOP 销售就是这么做的。

   首先，什么是销售策略？

   销售策略是企业或销售个人为了实现销售目标而制定的一系列的计划和行动。这篇文章接下来主要是讲我们销售个人如何制定销售策略。

   这里的“一系列计划和行动”，具体指的是什么呢？

  你可以这么理解，就是我们销售个人从“确定总目标—制定策略—拆解出具体指标—明确行动计划”这么一系列的销售动作。

销售策略一旦制定出来，并不是一成不变的，在执行的过程中，是需要不断的复盘和调整优化的，以适应市场的变化。
```

比如说，一名电话销售，制定的策略是每天电话覆盖 50 个客户，但是打了几天，一个商机客户都没有开发出来，那就需要及时复盘，是什么原因开发不出商机呢（商机：指有销售机会的客户）？客户名单质量不高？还是自己销售技能问题？如果是客户质量不高，那就调整策略，先解决客户质量的问题，如果是自己销售技能问题，比如开场就被秒挂，那就赶紧培训自己，提升接通率。

```
又或者说，做知识付费的个人 IP，做社群发售，成交量很低，那就一定要复盘是什么原因导致成交少？结果不好，一定是执行过程中某个环节或者某个方法出了问题，只有找出原因重新调整销售策略，才能获得更好的结果。

所以，懂得制定销售策略对于我们销售个人来讲是有很多好处的，为什么这么说呢？
```

一、明确销售目标

```
销售策略可以帮助我们销售个人设定清晰的销售目标，有助于销售聚焦目标，只干跟目标相关的事。而且实时提醒销售关注自己的年度目标达成进度，不只是光看短期目标，做销售一定要看年度收入，某一个月赚到钱但一年下来没赚到几个钱，肯定是没有年度目标意识。
```

二、提高销售效率

在制定销售策略的时候，我们在不断的思考、识别和筛选销售方法、销售工具，这样就可以减少无效的销售动作，提高工作效率。

```
就比如说：开发新客户，有很多开发方式，有线上的线下的，到底哪一种方法更适合你呢？就需要你不断去尝试和识别，思考哪种开发新客户的方式更适合你的产品，且能最快的链接到目标客户并取得他们的信任。
```

三、提升销售能力

```
通过制定销售策略，我们销售个人可以在制定策略中不断思考和成长，改进和优化销售方法，从而提升销售能力，具体可以直接提升哪方面的销售能力呢？ 比如目标管理能力、销售漏斗管理能力等。
```

四、风险管理

不断的思考销售策略，可以从中发现销售规律，比如说，你发现每天覆盖 20 家客户就可以打出一个商机，或者你发现你的新客户到成交的转化率大概是 5% 左右。这个销售规律就主要体现在销售过程中的一些数据转化，你掌握了这些规律，你就可以做到业绩预判，提前把控风险，及时调整销售策略。

五、个人职业发展

```
懂得制定销售策略是销售管理者需要具备的一项基本能力，不管是在职场做销售管理，还是创业带团队，有了这个能力，你在职业发展道路上也会有更多晋升和发展机会。

   这也是为什么我建议销售个人要学会制定销售策略，特别是我们职场上的销售，千万不要做一个“销售工具人”，只知道听话照做，没有自己的思考力，你确保你的销售经理，他的策略也是完全没有问题的吗？其实很多经理也是稀里糊涂的，所以我们一定要为自己负责，为自己的未来负责，让自己具备销售策略思维能力。

  那销售自己要如何制定销售策略呢？

很简单，直接借助工具，我已经为你设计好了一个“销售策略计划表”，这个工具表是建立在“OGSM 思维模型”上实操设计出来的，我从带团队到自己做个人 IP 轻创业，用了已经 6 年多了，我个人认为是非常适合销售个人使用的。

 这里有提到“OGSM 思维模型”，也简单介绍下，OGSM 是一个做目标分解、策略分解的思维模型，也被普遍应用在很多世界 500 强企业、各大公司。“OGSM”是四个英文单词的首写字母组合而成，分别是 O(Objective 目的）、G(Goal 目标）、S（Strategy 策略）、M(Measure 衡量），                            

   简单来讲，OGSM 底层逻辑就是围绕“目的”的层层拆解，这个“目的”你可以理解成做一件事的初心，比如你为什么今年要做销售，“目的”是存买房的首付钱，或者是“改变自己，让自己变的更强大”等等，这些就是你的初心。

   然后，围绕你的“目的（初心)”拆解到一个具象化的“目标”上，比如今年完成 100 万业绩，再根据“100 万业绩目标”拆解出你的销售策略，每个策略下再拆解出可衡量的具体指标。

  这里可能你会有疑惑，销售制定策略就直接按照自己的业绩目标来拆解制定就可以了呀，为什么非得从“目的（初心）”出发来拆解呢？“目的”是我们销售自己希望、期待、渴望达成的样子，所以，请你一定要静下心来聆听自己内心最渴望的声音，只有你清晰的知道“你到底想要什么”，你才能有持续的自驱力克服销售道路上的重重困难。

```

在使用这个工具表的时候，一定要特别注意几点：

1、“指标分解”一定要具体到“每天可量化”，也就是每天具体做到什么程度，一定要很清晰明确，这样才能保证我们的销售动作不打折不变形，确保最后的业绩目标达成。

2、每一项销售策略，一定要和时间绑定，启动时间和截止时间，可以确保我们做事不拖延，提高效率。

3、在制定目标或者指标的时候，如果不是 100%确定，也不要太纠结，能想到多少就多少，因为目标没有绝对的合理，相对合理就行了， 关键是把 80%以上的时间精力都放在“如何做”上， 就是策略执行上，通过执行不断靠近目标就好了。
