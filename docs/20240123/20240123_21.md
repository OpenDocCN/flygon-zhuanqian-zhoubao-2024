# 年度总结

最近微信公众号推出了公众号年度总结，正好最近也在做微信红包项目。

![](img/574f42b7d67c2a41a6d9c7b637230e91.png)

正好用来给粉丝发福利。

# 微信红包开放平台

这个平台很好注册，建议大家都去注册一个

然后主要关注这个封面图要求。图片尺寸，大小等。

![](img/d89845f4eeb7228b617245fcd5626aa3.png)

# AI 助力

现在市场上 ai 作图工具已经很多了。我还是使用的传统 MJ 去做的。大家也可以使用吐司，创客贴，等作图平台去做。

![](img/f65957cb9314a5f0ae6a17d4bc89a868.png)

简单切图出来之后，就可以直接上传到红包开放平台了

上传之后自动显示效果图。

![](img/f7574a12d8a8844475d75149fa2a25b8.png)

# 证明材料

目前 AI 作图证明材料比较简单 ，直接贴生成图就行了

也就是直接上传这张图。

![](img/f65957cb9314a5f0ae6a17d4bc89a868.png)

# 提交审核

证明材料上传完成之后，就可以提交审核了。

如果证明材料过多记得打包成压缩包。

![](img/2fae2d53c7b3009dbe67dbc603f6fd18.png)

# 引流：

设置领取时间，这样用户就会关注你的公众号，到时间来领取。

而序列号可以放在公众号文章，抖音，小红书等平台。毕竟占便宜，肯定会有很多人来领的。

![](img/2b71b71edf9f4a8651ab87c382688f12.png)
