## (13 赞)如何快速达到 YPP，有什么 BUG 可以卡？

作者： 书情小跟班

如何快速达到 YPP？YPP 有两个核心要求：

1、1000 个粉丝

2、4000 个小时

这里是指长视频的情况

如果你是从 0 到 1 开始做，会出现两个问题

1、粉丝够了，时长不够

2、粉丝不够，时长够了

1、粉丝够了，时长不够

说明你的 IP 不错，观众看到你了，愿意订阅你，这意味着你的片子偏短，这个的话，一般见于短片的情况，比方说一些搞笑类的频道，很吸引人，观众也爱看，但是你的时长很长，看的人虽然多，但是累积的时长不够。

这也是之前描述的一个点，先用短片，把人拉过来，粉丝数凑够，后期再用长视频拉时长，或者是你片子更新频率低了片子太少，增加片子的数量也是一个方式，开直播是挺好涨时长的方式

2、粉丝不够，时长够了

这种情况属于别人看你的视频挺多，但是不订阅你，观众只是对你的内容感兴趣，不见得对你的这个 IP 感兴趣

可以说是选题没毛病，但是你的个人 IP 物质征不明显，可以解决的办法，在内容中加入你的观点，加入你鲜明的个性的东西。

利用平台漏洞，提前加入 YPP 赚钱适用于 2、粉丝不够，时长够了

你可以付费推你的这个视频，在 YouTube 有一个宣传广告的选项，这个 BUG 关键词：**宣传广告是对您的频道在 YouTube
上的自然覆盖面和发现的补充，通过广告获得的任何订阅者和观看时长都不计入****[YouTube
合作伙伴计划资格](https://support.google.com/youtube/answer/72851)****的考量指标。**

我们可以通过付费推广，可以得到播放时长和粉丝，但是不算在 YPP 里面，这里的 BUG 在于**播放时长确实是不记录，订阅的人数是算的。**

这个 BUG 给我们的启发是，前期，我们要更关注自己的播放时长，而不用很担心你的粉丝数，播放时长才是赚钱的核心。

假设你现在粉丝数够了，那么你是立马能赚钱的，这个时候你就要付出一点点成本了，这里给两个案例用于支持前面说的【**播放时长确实是不记录，订阅的人数是算的。** 】

I Bought 1,000 REAL YouTube Subscribers... Here's what happened (YouTube
Promotions)

这种方式是正规的，毕竟你是给油管官方平台付费嘛，但是很明显一个点，那就是粉丝不见得一定是精准的，这种方法只是帮助你快速涨粉达到获利要求。

就是花点小钱，让你早点赚钱，这个操作类似于公众号刷订阅人数，适用的情况也是一样的，你有能力写爆款，但是订阅人数不够 500，最快赚钱的方式就是买 500 粉，你就能立马开通流量主了。

最后啰嗦一点，涨粉的事不建议找人刷粉，这是存在一定的风险的，同样是花钱，当然是直接找油官官方刷啊。
