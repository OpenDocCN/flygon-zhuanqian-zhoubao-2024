# 三、做知乎博主涨粉的三个方法

知乎平台的回报率还是比较高的，打造一个千粉账号后，就有机会接到基础的合作，当然，万粉合作大牌产品的机会会更多。

在知乎涨粉的方法很多，其实各个平台的涨粉逻辑，无非出于这三个方面：干货价值、情绪价值、时间价值。

这里举例3种

### 1、分享人生经验哲理

比如这样的人生道理的题目，很多人关注，浏览量也很高。

![](img/8ddf3f320f88985be535a5deaf244677.png)

我们可以去其他平台整理一些人生哲理，来回答这个问题，文末注明：侵删致歉。

也可以平时收集一些名人名言的金句，比如罗翔老师、人民日报的金句，整合一下来回答这样的问题。

出高赞之后，你就能享受到一天涨粉大几百上千粉的快乐了。

切记：如果用AI创作，一定要注明是AI生成的，系统会查出来的。

### 2、追热榜，写娱乐明星或者时事

以电脑端为例，打开【创作中心】——【近期热点】还有【潜力问题】里面搜索自己可以回答的问题，一定要快，热榜的热度也不是一直有，所以发现是热榜上的问题尽快回答，才能获得更多流量。

![](img/1c1b72cb6585e7d25c4fb4b9f3519cbe.png)

这时做好自己的人设，每篇回答下都带上自己的人设，更容易吸粉。

除了在热榜寻找话题，搜索框也是有热度的问题。具体方法：打开电脑知乎pc端，有一些问题滚动，这也是潜力热榜问题，这类型问题容易得到更多流量，也更容易曝光。

写这方面内容尽量不要太过带入个人情感色彩，避免有人看你观点太过激进招骂（只是提个醒，知乎的整体使用人群素质还是比较高的）

写娱乐明星或者时事方面的内容，在知乎获得的自然流量比较大。

这个方法能涨粉的原理在于： 提供了情绪价值。
