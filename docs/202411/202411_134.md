## (51 赞)流动性涌入与基本面误判：自我合理化的理论体系及其潜在风险

作者： 亦仁的收藏夹

日期：2024-11-12

“人的本能，是把流动性的涌入，误以为是基本面变好。然后不断研究分析基本面，拿基本面中的一些素材来合理化实际上是因为流动性涌入带来的价格上涨。
这个逻辑可能很长时间内会自洽，形成一套说服自己的理论体系，自洽时间越长，对理论的信念也就越发强化。
直到有一天流动性突然被撤掉。对于还沉浸在自我合理化的理论体系的人而言，这将变成一个灾难。”

忧愁小松鼠 : 想象一下，你在一个市场里卖东西，突然有一天，来了很多买家，他们争着买你的东西，价格就涨起来了。你可能会想，是不是因为我的东西质量变好了，或者市场对我的东西需求变大了，所以你开始研究你的东西到底哪里好了，为什么大家都喜欢。
然后你找到了一些理由，比如你的产品质量确实不错，或者市场上类似的东西很少，这些理由让你觉得价格涨是有道理的。你越想越觉得对，就越来越相信自己的这套理论。
但是，实际上价格涨起来只是因为突然来了很多买家，跟你的东西好不好关系不大。如果哪天这些买家突然不来了，你的价格就会跌回去。但因为你太相信你的理论了，当价格真的跌的时候，你就会很吃惊，因为你没想到会这样，这对你来说就像是一个灾难。
简单来说，就是人们有时候会把市场突然变热闹误认为是自己的东西变好了，然后找理由来支持这个想法。但如果有一天市场不热闹了，他们就会很惊讶，因为他们太相信自己的判断了。

YS : 用一个炒房例子来说明： 假设小区里的一套普通二手房： ------------ 1.最初阶段： 这房子本来每月租金 3000 元 市场价值 100 万
年租金回报率是 3.6% ------------ 2.忽然大量资金涌入： 银行放松房贷政策,利率降低 购房者拿到的钱多了,竞相买房 房价涨到了 150 万 ------------ 3.购房者开始"找理由"： "这小区位置真不错啊" "学区房肯定保值" "通货膨胀,买房最靠谱" "租金以后肯定会涨的" ------------ 4.突然政策收紧： 银行收紧房贷 买房的人少了 房价跌回 110 万 ------------ 5.此时仍在强调"价值"：
"这房子绝对值这个价" "等政策好转就会涨回来" ------------ 6.实际上： 涨,主要是因为钱多了 跌,主要是因为钱少了
房子本身并没有什么变化 租金还是 3000 块 ------------ 这就是典型的把"资金推动的上涨"误认为是"价值的提升",最后被自己的解释框架所困住。
