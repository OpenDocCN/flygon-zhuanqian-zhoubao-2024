# 二、公司参观以及了解店铺过程

去了他们公司之后，他们公司分为几个区域，客服区、运营区、美工区、会议及洽谈区。

接待我的销售张辽直接领着我去了运营区，找到了负责麦叮当旗舰店的运营，让他打开店铺后台，看下店铺的流量结构，以及最近几个月的生意参谋首页的销售额报表，期间也将网页刷新和上下滚动了，证明这个网页是真实有效的，实时的，并不是P图的数据。

也让他给我讲解了下这个店铺起店的做店思路，接受的第一步就是统一了店铺的图片风格，采用同一个模特实拍，增加店铺视觉的美观感，然后进行了批量测款，测出好款之后进行了哪些动作，数据的递增、补单、推广。换季的时候如何提前布局，提前测好一批款式，做好基础，在季节到来的时候直接顺势爆发。

其中有一款裙子，在12月份的时候直接上了市场竞争里面的单品销量排行榜，是一款小女孩的连衣裙，后面有一对小翅膀，这个款式是他们运营结合过往的热搜词跟爆款元素重新组合排列生产的独家款式，市面上没有同款，所以一举打爆了。

他们一排是五个工位，五个人负责一个店铺，一个运营，两个运营助理，一个美工，一个美工助理。

一共聊了四组人，看了四个店铺，期间给我的感觉是挺专业的，说出的那些做店过程、运营思路结合我自己过往的实际做店经验，感觉没什么问题，我就说可以聊下怎么合作了，结果销售先给我提了个悬点，他们并不是什么店铺都接的，没有实力的不接，而且要先跟他们的管理层聊过之后，才能同意愿不愿意接你这个店铺。

然后就安排了会议室，让你等一下，过了一会进来了个西装革履的男士，一番自我介绍，说是公司的副总，合伙人之一，以前是银行里面的铁饭碗，刚开始要辞掉银行里面的工作，家里人都不同意，他不管家里人的意见，说跟老板是朋友，比较认可他的公司理念，就加入投钱成为了合伙人，然后就是聊天阶段，问我之前是做什么的，做童装多久了，有没有经验，之前做的业绩怎么样之类的........

闲聊了一个多小时之后，说他是江西赣州的，跟我是老乡，觉得我挺合适的，可以合作，管理层之间要开个会议，完了之后让销售发合作合同给我，让我先回家了，接到合同之后如果觉得没问题就可以直接打款开始操作了

第二天发了合同的PDF文件过来，让我打印出来，没问题的话就签字盖章，然后回传给他们，收到合同的时候，朋友也劝了我，不要跟别人乱签合同，看清楚条款，不合适的话不要做，但是自己当时有点急于求成了，合同虽然仔细看了，但是并没有找杭州那边细聊合同的条款

甲、乙双方合作开始推广的15天后乙方制定首期合作期限内考核指标，经甲、乙双方协商一致正式执行。甲、乙双方做效果评估，如果没有达成销售目标，甲方有权利单方面终止协议或继续合作，且不用支付乙方任何违约金，如果达成销售目标，甲方须继续合作，不得单方面解除合约。

当时自己以为合同里面的这个条款的意思是，不满意的话可以随时停止合作，没有找他们确定下这个意义

# 三、终于开始安排团队运营店铺了

7月28号开始合作了，加了QQ群，事情都是在QQ群里面聊天安排，然后制定了店铺的运营计划，选品、上款、做基础销量评价、测款之类的.......

我自己粗略的计算了下成本，童装的普遍利润率在30%-40%之间，常规价格，因为我们是去档口拿货的组货商，并不是自己生产的，价格方面并没有什么优势，压缩不了什么成本，所以直接在群里指出了销售额计划的不合理部分。

有些走低价的同行，利润率甚至只有15%左右，拿货的价格加个十块钱就直接那样卖，这边发快递要三块钱一单左右，还有运费险的成本，他们就直接一件衣服赚个三五块钱不到，把淘宝当拼多多一样做

后面他们重新制定了一份新的计划表给我，按照这个计划表，能达成的话，应该后面能把前面的亏损赚回来，就同意了他们的营销计划，开始实际操作了

新来的运营重新换了目标，之前的是主打卫衣，现在换成毛衣，毛衣的适用人群广一点，款式也没有那么多花里胡哨的，起来的希望大一点，就按照他的节奏来做了，结果一个月过去，也是大坑一个，亏了个新高三万元

# 四、开启要债之旅

意识到他们不会主动退款了，也去我自己当地的辖区派出所咨询了，我这个事情应该怎么办，派出所的民警告诉我，这个情况属于经济纠纷，他们这边无法介入，建议我去咨询律师，走法律程序打官司。

出了派出所之后也是直接找了下附近的律师事务所，等了一会，来了个律师，我把合同拿出来给她看了，她说这个合同里面说的比较模糊，其中这个停止合作的合同条款比较模棱两可，就是直接打官司的话也不一定能赢，而且律师费需要6000元，这个不管官司的输赢都是需要我这边直接出的。

甲、乙双方合作开始推广的15天后乙方制定首期合作期限内考核指标，经甲、乙双方协商一致正式执行。甲、乙双方做效果评估，如果没有达成销售目标，甲方有权利单方面终止协议或继续合作，且不用支付乙方任何违约金，如果达成销售目标，甲方须继续合作，不得单方面解除合约。

那段时间真的是睡觉都睡不着了，整个人真的很难受，心力憔悴的那种。因为合作的款项，一大部分是找父母拿的，亏损了之后也不敢跟他们说，怕他们受不了，合作了三个月，服务费6万，店铺亏损一共是7万多，合计亏损了13万多。

期间也想过直接跑到他们杭州公司去拉条幅，骂他们骗子公司，直接让他们工作不了，结果还是自己懦弱了一点，没有付出实际行动。

店铺亏损的是已经亏了，生活还要继续过，店铺还要继续做，不能扔着不管了，还有三个月的服务费没有退，想着就当是以后的美工费算了，以后的做图什么的都丢给他们，就算请了个比较贵的美工。

也是店铺没有让他们继续做了之后，跟十月份安排给我的运营聊天之后才知道，他对我店铺没有做起来也比较抱歉，聊了下他们公司的分配，虽然每个店铺都是分配了一个运营团队，但是团队并不是只负责一个店铺，平均每个团队都要负责五六个店铺以上，多的要做七八个，难怪他们之前上款的时候，一天也上不了多少个，还没有我自己一个人一天上的款式多，回复消息也不及时，这下算是明白原因了。

大部分的代运营就是博概率，矮子里面挑将军，手上接几十上百个店铺，有一个成功的那就是他们的功劳了，是他们的明星店铺。做不起来的你们就要多考虑自身原因，是不是不够配合，款式不是独家的，图片也是用的档口图，没有自己的视觉优势，价格优势，不怪他们。
