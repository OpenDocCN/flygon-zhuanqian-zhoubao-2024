# 1、什么是远洋捕捞？

民间定义：指出于利益驱动，部分地区的公安机关违法违规异地抓捕、查封、冻结、甚至划转外地民营企业财产和个人财产的做法。近年来，此类现象多有发生，较大地影响了民营企业的安全感和发展信心。「来源：经济观察报」

# 2、为什么会出现远洋捕捞？

地方执法与司法的暗战：利益首当其中。刑事案件沾边就管，管辖权往往意味着背后的经济收益「罚金、罚款、罚息、追缴违法所得」，地方税局有地方创收考量，其他法检公有财政收入。考核指标也是硬门槛，能否顺利完成今年的立案、结案、定罪、量刑指标，等同于是否尽职尽责。

沾边就能管：“犯罪地”包括犯罪行为地（犯罪行为的实施地以及预备地、开始地、途经地、结束地等与犯罪行为有关的地点）以及犯罪结果地（犯罪对象被侵害地、犯罪所得的实际取得地、藏匿地、转移地、使用地、销售地）

## 4.1、骗取出口退税罪：深圳工厂主被带到内地四线城市、获刑5年多

有，一起骗取出口退税罪的再审翻案，我们1月接受咨询，3月介入接受委托，5月正式递交再审申请书，9月取得意料之中的驳回再审申请结果。直到8月开始兴起远洋捕捞提法，我们才发现这起案件背后是2020年典型的远洋捕捞。简而言之，内地某四线城市经侦大队长，将深圳民营外贸老板带走，经历两年多一审二审，最终由可能缓刑、判处有期徒刑五年多。从骗取出口退税罪的罪名看，涉案当事人并非具备出口退税主体，即没有退税资格，怎么会成为一起骗税罪的主犯呢？其中出现刑事案件意料之外、情理之中的情节：威逼利诱改口供、认罪认罚无辩护律师在场、疑似开始办理缓刑手续突然变成判决五年半，可能还存在一些行贿受贿情节。案件证据也令人大跌眼镜。直呼意外。该市检首次办理骗税案件，被质疑判决不公时，疑似曾向家属硬气表态，不需要参照有经验的广东省案例。

所思所想：

对于刑案再审程序，辩护指数是难上加难。对于具有诈骗属性、与税相关的犯罪，更是一经定罪，难以翻盘，基本为铁案。不同地区办案水平不同，具体到个案，每个案件审判过程中的每个证据异常关键，当事人本人能掌控的证据，核心是口供，这个案件中改了一字口供，刑期变成五年多。不要轻易更改口吻，一字之差，从犯变主犯。

# 6、远洋捕捞与普通刑事案件有何不同？为何谈「远洋捕捞」色变？民企老板为何瑟瑟发抖？

这三个问题要放在一起看，本质上存在因果关系。

一个刑事案件是否能立案、符合起诉条件、得到公平公正判决，往往需要以最严格的法律程序保障，这是刑事辩护被认为律师“掌上明珠”业务的根本原因。但：

1）难不代表必然公正

企业主必须认识到：刑事案件的办案水平一样因人、因地、因案而异，大部分能被看见的不公平判决，往往需要耗费大量的时间精力与金钱才能取得较为彻底的翻盘效果。而这只是总体样板量的极小极少数一部份；

企业主应知应会：让企业免于沾边刑事犯罪，是防范这类风险的关键，出事儿之后精准踩坑只会让人财两空；我们普法这类风险，并非为了业务转化，更大的理念在于有效扶植一家民营企业有序有效存活，使得市场健康，大家好法律服务才好，才有可能实现个体职业抱负与梦想；

2）企业主要防范的是基于企业经营的系统的涉罪风险，不是照单全收或完全漠视

因为所处跨境出海行业，面对的涉罪风险指数更大、情形更复杂、警戒线更高；由于远洋捕捞案件本身并非为了防控犯罪或者维护经济秩序、金融安全，不排除以刑事防范手段解决地方创收问题的可能性，因此办案过程中是否能秉承公平、公正、维护法律尊严与秩序的法治理念与精神，比起一般案件更具备趋利性、对企业主也就更有趋害性；

3）民营企业是「远洋捕捞」重灾地，也是滋生不公正办案的温床

民营企业风控意识太薄弱，老板一个贪念，就葬送自己和企业寿命，得不偿失！但现实是，大部份老板忙于业务，急于解决活下来的问题后，才有精力考虑如何活。这是常情常理，只是需要加多常识的学习与认知，才有可能真正的活得好、活得久、走得稳、走得远。

# 7、丁邱洁律师第一次公开呼吁：保护民营企业刻不容缓，必须严厉打击「远洋捕捞」

## 发刊词：保护民营企业刻不容缓，必须严厉打击远洋捕捞（2024跨境电商太空漫游）

我旗帜鲜明的反对远洋捕捞，成就一个企业要历经千辛万苦，九九八十一难都不够，但毁掉一个企业只在顷刻之间。

直到今年五月份办一起骗取出口退税罪的再审案件，我才意识到，中国的民营企业家、小企业主，出海的创业者们，到底面临着怎样的风险，又应该有着怎样的担当。

民营企业的成长九死一生，资金、信息、资源的竞争，任何一个环节都有可能让民营企业就此死亡。对于出海企业跨境电商来讲，双边乃至多边监管更加困难重重。

企业主要突破的不只是别国的小院高墙，还有本国的不当监管。

我无比痛心的看到，一些企业有些还没走出国门，就在国内被通过远洋捕捞的形式所击倒。呜呼哀哉，不胜扼腕。

作为法律人，我们更多考虑的是合规和风控。想要让这些企业高效的出海，以较低的代价、较小的成本取得较高的成绩和较大的结果，并让企业不断打磨供应链，塑造品牌价值，形成真正具有国际竞争力的企业。

复盘办理的相关刑事案件，研读各地陆续爆出的涉刑案件，我不得不问：这些企业是有多大的罪需要用刑事手段三年又三年的追究打击，是否真的就没有办法能及时化解企业经营中的不合规？如果企业真犯错了、出事儿了、涉罪了，有没有更好的方式能让企业成长？

冻结账户是最初最浅最暴力的手段。一个辖区的派出所拿了一纸公函，就能够让一个十几年的企业大厦毁于一旦。是谁的权利需要被限制在牢笼里？

我过去觉得，我们辅导单个民营企业就能够解决一大群人的就业问题，解决星罗棋布的中国企业在海外的声量问题。

但我现在发现，一家民营企业在经营中出现各类问题太过容易，下一纸立案文件也太过轻易。如果我们仅仅着眼还后知后觉的去解决已发生的辩护问题和难题，那么刑事风险控制企业刑事合规将只是一纸空谈。

我们必须让声音发出来，被看见，让民营企业的举步维艰能够真正被重视。

放眼全球，眼下中国企业尤其是中国供应链出海，横亘在眼前最大的问题和难题已经不再是供应链本身的质量、口碑、生产制造问题，更多的已经上升到国内监管端的源源不断，走刑事似乎成了打压企业的高质量手段。而在国际层面，各类贸易调查、封锁、管制、禁令数不胜数，地方保护主义频起。

一个普通的民营企业哪有能力应对，如何有还手之力？

我再次声明，我旗帜鲜明的反对远洋捕捞。

我坚定的站在民营企业一边，希望为更多民营企业发出自己的声音。只有无数的民营企业，尤其是中小微企业能站起来，才有可能真正的让市场有市场，让市场真的成为市场，让市场回归健康的市场。
