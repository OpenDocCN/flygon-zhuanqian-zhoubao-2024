# 一、揭秘「结婚」和「产品发售」该如何联合，带来更多现金流呢？

九月份的时候去南京跟靠谱姐和理白聚会，他们给了我很多发售的灵感和策略。

靠谱跟我说公众号的文章、朋友圈的内容撰写要更贴切我的身份转变，从女孩到人妻的视角成长，婆媳关系的讲述，女性的独立，都可以多写一写，触达目标客户。

于是我就开始践行这个发售的想法。

# 二、可粒如何以“结婚”为关键营销节点，发售10年新产品，多赚17万

其实不管你发售什么产品，成交用户都是有进度条的，从认识你、认知你、认可你到最后认购你，都有一个过程。

成交的本质就是「信任」，那我们就要倒推用什么动作去提高用户对咱们的信任度，加速用户付款的过程。

最简单的逻辑就是——你要频繁在用户面前出现，让他觉得你的内容有价值，让他对你心痒痒，就忍不住买单了。

我们做自媒体卖产品就像是开一个商店：

有人会进来选购，付款；

也有人看看就走了，犹豫；

也有人还不知道你是卖什么东西的，或者是说有什么产品可以挑选。

那咱们做发售也像是商店里的大促活动、主题活动、周年庆活动，活动多多。

相同特点就是限时发售，限时福利，借势发售，涨价促单，这么一套组合拳下来，用户不买单都怕自己后悔了。

接下来，可粒给大家像素级拆解「结婚」和「产品发售」结合的全过程，分为（婚礼前两周）发售前、（婚礼这一周）发售中、（婚礼结束后）发售后等三大板块。

# 三、通过「结婚发售」，我总结出适合普通人发售搞钱的心法

我见过很多自媒体IP或者创业者：要么流量很多，但全是白嫖粉，就是不买产品；要么只卖的动低客单价产品，只能收小白，一卖贵的就被骂。

本质的问题就是因为不懂得如何做内容营销。

当你懂得把内容当作资产而不是营销手段的时候，你就理解了内容营销的真谛。

内容营销和广告的最大区别是，如果某天你不再为广告续费，它就会从人们的眼前消失，但内容却能成为你的永久背书。

内容创业的基础，是选择一个目标粉丝聚集并适合内容呈现的渠道，在一段时间里坚持积累内容资产和粉丝人气。就像两军交战，选择有利地形修筑工事，这将成为你创业征途的有力后盾。

所以，我一直坚持写优质的内容，自媒体4年，数不清写了几百万字，也为我带来了200多万的增量，但也远不止如此。

更是因为自媒体的慈悲，让我从农村女孩变成拥有20w粉丝的自媒体博主，甚至也有贵人朋友们飞跃几千公里，从哈尔滨、北京、广州、深圳、福州，杭州等地方来参与可粒的婚礼，也收到了千万大佬的红包祝福，能走到贵人的身边和他们并肩奋斗，能感受到大家充分的重视，我都觉得格外荣幸和自豪。

如果没有自媒体，没有做私域产品，我只是一个普普通通的27岁女孩，在适宜的年纪结婚，为人妻，为人母，平淡度过这一生。

这不是我想要的人生，所以我不断折腾探索，去发现人生的更多可能性。

现在我做产品和做业务，更注重于做长期的策略，比如写作课、个人品牌课、发售课、朋友课。

包括我的10年超级个体孵化基地，也是10年交付产品，不追求10倍增长，现在更想把产品做十年，这也是跟粥左罗老师学习的。

希望大家都能尝试做一门可以长期卖、还可以卖给更多人的课程，同时也是在找自己的长坡厚雪，稳稳变富。

## 2、做IP要像明星一样搞钱，跟全世界宣告你的人生大事件

## 

之前有人在我的公众号文章留言，说你结婚怎么预告了那么久。我就乐了，这说明我的结婚发售营销很到位啊，让你记住了我的大事件了。

你们知道吗？有些明星、网红结婚，那都是有品牌赞助和广告植入的，这是影响力真切能变成钱的最好证明。

虽然大家都喊我知识付费届的女明星，但我也清楚，咱们的美貌是尚可，不至于艳比明星，但是在搞钱上，我是不认输的。

万事万物皆为我所用，万事万物皆可为营销素材。

我发售产品和结婚的时间，是巧合，但是给我额外带来了十七万的营收甚至更多，普通人结婚花钱如流水，就算收了份子钱，还是要倒贴，可粒结婚怎么算都是赚的。

![](img/7c0f30994089985b68a3374645a2ba40.png)

我想没有人嫌弃钱多吧，做自媒体真的很好，拥有了上帝视角，把本来单纯的消费，变成了生产力。

那么你的一切活动，小到日常生活大到结婚生子买房，都有机会变成钱，祝福大家早日拥有自己的渠道和生产资料。

我相信，下一个IP明星，或许就是给文章点赞的你。

## 3、把产品嫁接到热点话题或者永恒需求上，增加联想空间

## 

粥左罗老师曾说过，热点话题和永恒需求是相对的，这是最近发生了什么新的事情和大家比较关注的，永恒的就是不管什么时候都是大家需要的。

比如结婚，生娃，生日等等，都是阶段性的热点话题，所以可以阶段性的营销。

永恒的需求比如人性的需求，想搞钱，想成长，把这两个东西嫁接一下，就会使你售卖产品有想象空间，引起用户遐想，从而实现下单。

## 4、内容营销，是最真诚且有效的营销手段

## 

我觉得世界上最好的营销套路，就是顺路，刚好你有一个产品，可以解决用户的一个问题，所以你赚钱是应该的，用户为你的价值和服务方案付费也是应该的。

其实大多数人赚不到钱，并不是因为能力不够，本事不大，而是因为不懂得用内容包装自己，用人设包装自己。

重视内容营销，这是能够帮你穿越周期，持续赚钱的最好捷径，内容为王，所言不虚。

如果总想走捷径，总想暴力引流，暴力成交，那么「暴力」方法吸引来的，注定就是低质低智的客户。

为什么要写私域内容，写朋友圈、写公众号？因为可以免费、多次、触达客户，如果你不写，你的客户就很容易被其他人的营销手段吸引走了，如果你写，你就能筛选出源源不断的优质客户、好交付、好出成绩的天使客户。

记住，正因为交易是低频的，内容是高频的，所以我们要靠高频次内容来吸引用户、建立口碑，带来转化和传播。

## 5、在卖产品之前，先学会做好「交付」，永远不怕没人买

我有个客户曾经问我，10年的产品，能坚持那么久的初心吗？

我说，你如果问我搞钱的初心，我能坚持到die，而且我又不是只想赚你一个人的钱。

我还想多赚几十年，那我就会好好做交付，交付好老客户，口碑稳住了，就能带来新客户，新客户也能带来表客户，这不更香吗？

其实做知识付费，也像是庙里的菩萨，你越灵，越能够帮助大家实现愿望，那你就会香火旺，祝福你我的事业都能一路长虹。

感谢你的时间和注意力，祝我们都能发财被爱。
