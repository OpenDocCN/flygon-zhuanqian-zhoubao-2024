## (精华帖)(338 赞)初中生学习编程的推荐渠道与心得分享

日期：2024-11-11

提问：

初中生想学编程有什么好的渠道推荐么

回答

你是初中生？ 是的话，如果你想学编程，我可以赞助你一些学习过程中可能会用到的付费工具，比如 cursor、v0、claude 的付费版本。
建议去外网，特别是 X 上，搜 AI 编程学习，资料很多，也可以跟着这个月的 AI 编程航海来学习。 网上已经有不少学习框架，可以参考下面一个用户总结的：
过去 48 天，我在 Cursor、v0 和 Claude 上花了 217 小时。 我当时对 Nextjs 应用程序的路由结构一无所知。
但现在我能在 4-7 天内完成一个 Web 应用程序的编程。（前端、后端、登录页面、着陆页） 以下是我的全部心得： 1. 文档是关键！
把 80%的时间花在 GPT4o1 模型上。 你需要为每件事创建文档： - 项目需求文档 (PRD) - 每个页面的布局和结构 - 应用流程文档 - 应用功能文档 一旦有了这些文档，将它们保存为.md 格式并上传到 Claude！ 2. Claude 将是你的主要大脑。（Cursor 上的 Claude 3.5
sonnet 很弱，我怀疑它是不是 3.5 sonnet） 在 Claude 上创建一个项目，并上传所有用 o1 模型创建的文档。 3. 转到 Vercel 的 v0。（v0 将是你最好的前端设计工具） 在 v0 上创建一个新项目。 将每个页面文档（如仪表板页面、应用流程和功能文档）粘贴进去。
现在 v0 将拥有项目的上下文、技术栈、流程以及应用的功能。 开始构建前端组件，确保将每个页面拆分为多个组件，如 - 头部 - 统计 - 表单
这将帮助你在 v0 上细化代码而不破坏整体结构。 4. 回到 Claude 并粘贴每个页面代码到 Claude 项目中。
这样 Claude 就能理解代码和项目需求，这在你使用 Cursor 时会有帮助。 5. 转到 Replit。（Replit 帮助你连接 Cursor、Github 和本地环境） 在 Replit 上创建新的 Nextjs
Repl。（我个人使用 Supabase + Nextjs 的 Starter kit，效果不错） 连接 Github 并使用 SSH 连接 Cursor。 6. 转到 Cursor。 创建 Claude 建议的所有文件夹和文件。 在 Replit 上运行项目，将代码粘贴到指定文件夹，并安装所有依赖项。 7. 创建后端。
我使用 Supabase 来创建数据库、存储和认证。 AI 可以为你编写数据库表，这是最棒的部分。
在 Supabase 上创建新项目，获取你的密钥并粘贴到 Cursor 中的 env.local 文件。 8. 让 AI 帮助你创建后端（数据库表、存储桶、认证表）。
按照 Claude 的指示操作，完成后，你将拥有一个完整的 Web 应用程序，包括前端、后端和登录注册功能。 9. 调试和优化时间。
测试所有页面上的所有功能，使用 Claude + Cursor 进行调试。 10. 设计着陆页。
o1 负责内容和布局，v0 负责代码，Cursor 负责组装代码，Replit 负责部署着陆页。 现在你拥有一个现代 Web 应用程序和现代着陆页。
看起来很复杂对吧？其实并不。 从未有过这么容易编写软件的时刻。 50 天前我还不知道： - Nextjs 应用路由 - 数据库表 - 认证功能 - shadcn Radix - SSH 连接 AI 帮助我学习了编码。现在我可以在最多 7 天内完成一个完整的 Web 应用程序。 过去 30 天，我编写了 7 个项目。
我的计划是，不分享商业计划，而是用 AI 代码构建业务。 然后将项目卖到或卖给客户（我现在就在这样做），然后重复。
我喜欢构建，所以我会继续下去。[`Acquire.com`](http://Acquire.com)
