如何选赛道？想要赚更多的钱，那么建议优先以欧美为主要的客户群休

做得专业一点就上海外住宅 IP，不是非要一定这么做，只是这么做显然专业，不用住宅 IP，多个频道都挂在同一个机房，有一定概率会被识别到。

可以做一个类比，住宅 IP 是真实的人在使用，非住宅类 IP 类似于网吧的 IP，多个人共同，这个网吧人少，确实是有可能也被认定为住宅 IP，较为纯净。

免费的节点是万万不能用，牛鬼蛇神都能用，容易被标记，另外一点免费 IP 一个策略是多个 IP 进行切换，哪个能上上哪个，那在后端的 IP 地址也是乱跳的。

既然是要做海外，那么得做好分析，我们中国文化有什么是出圈的？比如说 China，又名 瓷器，比如说功夫 李小龙，比如说中国的熊猫，中国的四大名著。

之前就有关于外国人沉迷于中国的小说，有些人把国内的小说翻译成英文进行一个收费

一个思路是把这些四大名著配英文，把人物的对话由原来的中文配成老外能看得懂的中文，这是在原来有视频的基础上进行一个创作。

也可以自己原创，题材还是以国外对中国文化熟悉或者说喜欢的点进行创作，比如说以动漫的形式，以搞定的形式，配字幕还是英文来配

当然了，根据四大名著也可以做更多的细分，比如说红楼梦电视剧中的各种服饰道具，对这些服饰道具进行解说，也是一个方向，又或者支持这些衣服的一个定制服务。
