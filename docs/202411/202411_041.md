## 一、工具介绍

这套工具主要有三部分组成：

1.  工作台，主要用 Cursor 开发，可以管理对标账号、对标品、选品关键词以及对自己闲鱼多账号的管理

2.  自动选品，主要使用影刀开发，分为两个部分

3.  针对选品关键词自动搜索并筛选出符合选品需求的品

4.  针对对标账号自动筛选出符合选品需求的品

5.  自动上品，这部分同时使用了 Cursor 开发的代码和影刀，主要使用阿奇索的功能来实现自动上品
