# 怎么制作口播类视频？以heygen为例

## 注册heygen

![](img/a86d4b16d7c71d4b81421bd70a35e777.png)

## 获取heygen积分白嫖

点击右上角的礼物icon获取邀请链接，通过小号不断刷积分，可以刷3个积分

有软件可以刷积分，原理也是这样的，只是减少手动操作

![](img/9cf4749aa0378fb5a87914a36308bde3.png)

## 定制数字人（目的是去重、增加亲和度观众缘、最好自己从TikTok、YouTube找一段很少人用过的、还能减少视频剪辑）
