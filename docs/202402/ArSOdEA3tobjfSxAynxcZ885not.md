# 一、项目背景

之前有参加过小红书店铺运营的航海，关注的是玩具赛道的，所以经常会刷一些玩具类的内容，发现有个玩具最近很火，小红书的数据很好，账号的第一个视频是在1月23号发的，后面都日更一个到两个视频，整个店铺只有一个产品。
