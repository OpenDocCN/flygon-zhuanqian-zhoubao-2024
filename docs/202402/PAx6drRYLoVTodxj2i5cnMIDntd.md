# 6、细节注意

1、直播手机挂着不要动，等做完这一轮，他才可以休息。

2、刷机的话，网上都有教程，各个手机不一样，没办法统一说，大致就是按着音量下键和锁屏键，屏幕亮了之后，松开锁屏键，选择恢复出厂设置，如此反复6次，才能清理干净。

3、一定要用流量，无论主号还是小号，都要使用流量

4、但是还会存在一个问题，就是这个游戏类的，它的钱数是有上限的，就是比如说他是10万到30万，到达这个金额以后，可能这个游戏会出版新的版本或者是。不再转化了

5、开播的手机，最好使用苹果手机，比较稳定，安卓也可以

# 7、复盘总结

第一步、准备好开直播的账号（必须实名）在准备几台做数据的手机和快手小号(快手小号不用实名)

第二步、几个做数据的小号要先关注开直播的大号.养一养粘合度。前期的话建议一个直播的配上四个玩游戏的手机就可以。后期熟练之后 再去追加手机就可以。（手机配置建议华为 6+128）

第三步、双清手机，意思就是每天清理手机 要保证你的手机处于新机状态。收益稳定 高，

每一个品牌的手机双清方法不一样华为的方法是先把手机关机，按着开机键加上音量＋键同事按着手机屏幕亮了之后在按到 3-5 秒就可以了，格式挂手机数据重复三遍 然后再重启手机 在按着开机键加上音量＋键在格式化手机清楚数据三遍，然后在重复重启手机清理数据 注意这个步骤一定要充满回复三遍

第四步、刷完机之后就可以准备开直播了，直播的号开始直播，其他几个做数据的账号先去刷一下财经类的直播间，主要就是看一下商务广告类型的主播间就可以，看半个小时左右就可以（记住这个步骤是每天开播的必要）

然后，再去看自己开直播的账号半个小时，主要目的就是养一养粘合度 前期的话看直播的这几个账号可以送一个灯牌 点点赞 在直播间互动一下然后退出快手重新进 一定要在同城里进入自己开直播的账号去下载小游戏，进入直播间下面有一个小铃铛直接下载就可以，前期的话同城没有就要在关注里进直播间，这个玩一轮的时间建议 3-4个小时，而且一轮之后要开始下一轮的话 一定还要重复第三步骤刷机 一定要重复双清手机三遍。
