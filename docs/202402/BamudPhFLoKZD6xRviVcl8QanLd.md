# 项目背景

几年前听过香帅的一节课，标题是育儿嫂和行政秘书谁更有前途。课程提到育儿嫂和教师一样，都是社交智慧型职业，被人工智能取代的概率是8.4%。

现如今，市场环境也发生变化，之前在一二线才有市场的育儿嫂，现在在三四五线城市也有市场。尤其是两代人的育儿理念发生改变。因为两代人育儿理念发生改变的情况不断增加，育儿嫂的市场需求也在不断增加。

人们对于育儿过程中交流，技能，智力开发越来越重视，从而受过专业培训的育儿嫂更受欢迎，从而也更能拿到高薪。因为高薪，一些年轻人也接受这项职业。育儿嫂培训市场存在着需求，在线培训接受程度也增高。而且一些培训机构也承担着派单的业务。能更好的吸引用户报名学习。

# 流量获取

## 线上流量获取

本次拆解的是抖音上的一个账号，通过日常短视频，直播方式增加订单。名称中有育儿指南关键字，会获得一部分搜索流量。目前有5套课程，单价从99元到2680元不等。最高客单价2680元学浪6794人在学，抖音课程专栏6016人在学。售卖订单6000+。

短视频发布500多条，以口播为主，讲述育儿嫂的理论和方法。听过一场直播，会以育儿嫂涨薪为直播间的标题。操作方式是晒简历，然后说目前工资是多少。和直播的用户说对比，然后晒工资更高的育儿嫂简历，通过培训之后，会做到8000元--50000元/月。

在直播过程中，会讲一些具体的案例，就是日常服务中，擦桌子，拖地这些小细节，通过这些小细节判断做的好不好，值得拿多少薪资。

然后在直播中会介绍和之前学员的对话，也提出派单需求。派单也会在不同的城市展开。

## 线下流量获取

在搜索过程中，找到2023年3月，地方政府举办的公共实训课程。和当地政府合作举办活动，也是一个不错的流量获取方式。

# 后端履约

通过和客服沟通没有获得更多有信息的资料。不过查询关键字会有派单老师和培训老师。做这项业务因为后面有派单需求，可以提升课程转化率，但是后端承接也需要做好。

最高单价课程，1200多评论，中评3个，差评8个。反馈课程没用。每个课程都没有办法满足所有人，后端履约方面做的好能够避免口碑破坏，需要做更多的努力。

或者在售卖时就针对用户提出需求，什么样的用户适合这套课程，提前预防一下。
