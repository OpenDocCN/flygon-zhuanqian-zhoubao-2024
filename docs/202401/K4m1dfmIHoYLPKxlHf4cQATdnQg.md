# 设计师下班后做小红书，一年涨粉1.3w，副业月入1w+，设计插画类账号的变现历程

> 来源：[https://fm1y82mmpu.feishu.cn/docx/K4m1dfmIHoYLPKxlHf4cQATdnQg](https://fm1y82mmpu.feishu.cn/docx/K4m1dfmIHoYLPKxlHf4cQATdnQg)

大家好呀，我是十一，

去年我和队友一起运营了一个小红书插画类账号，一年做到了粉丝量1.3w，月入1W+，

我是设计师，队友是插画师，我们两个目前都是自由职业，

这也是目前我们做到的第一个运营比较稳定的插画设计类账号，

在这之前我们在做账号方面，做过不少尝试也走过不少弯路，算是总算积累出一些手感，所以想好好复盘一下。

我想我们生财里也有不少设计、插画类的圈友，我深知大多数同行，都是更擅长执行而疏于运营，所以分享出来，希望能够提供一些思路给大家。

当然，除了顺利的部分之外，我们也有遇到一些卡点～

一个是想到这些都是可供大家参考或避雷的点，另外也愿意与大家一起讨论解决。

# 账号运营概况

去年10月到现在，账号涨粉1.3w+

通过接单与线上插画课程，做到月入1w+

# 账号的定位与内容

关于内容，这里提供四个比较重要的建议，也是我们在「确认账号内容与定位」这个过程中，一些行之有效的经验。

## 1、内容要结合账号定位，先想好变现方向再反推内容；

虽然很多有账号经验的朋友嘴皮子都说烂了，但现实中我遇到的很多人还是完美避开了这一点，所以结合自己的实践，还是想跟大家强调，「想好变现方向」，真的非常重要。

以前我们瞎发乱发的时候，只想做“接单”，但是接单接什么方向，画什么内容，完全没有头绪，再一看同类比较喜欢的插画类账号，基本都是以自己的日常创作为主，很好看，很小众，很艺术，所以就想自己也随便画，结合一些热点话题创作一下，结果就是石沉大海，越做越没信心。

如果想变现，就要想好变现方向，然后往这方向去思考内容，如果想「先涨粉后变现」，我经历过的无论是个人还是公司，都很难有结果。

·

设计、插画类的变现，比较常见的有三种，

接单、做课、卖资源。

·

在单纯想接单的实践失败后，我们想到做课。

做课是我们一直想做的，之前主业比较忙的时候，这个想法就一直萦绕在脑海，但是没有行动。

队友作为插画师，有比较丰富的商业插画经验，另外从小专业学画画，基础比较夯实，讲课还是比较擅长的。

另外，小红书内容是否能火，存在很多不确定性，但是，我们可以去试着提高一些可能性。最容易被赞藏的内容，往往就是非常利他的干货内容。比如我自己业余做读书账号，如果是发单本书内容，那被赞藏的不确定性就很高，但如果我做某个主题类的书单（比如「设计灵感」「哲学入门」「摇滚乐书单」），因为有比较明显的利他属性，那么数据大概率就会好很多，那我做主题书单，就能提高被赞藏的概率。

既然目标是做课程，那我们就围绕课程生产一些利他的内容。

当我们想好变现方向，去反推内容，其实就很清晰了。

## 2、找到适合自己的爆款内容；

具体我们如何找到「爆款内容」的呢。

最开始我想的是，把课程大纲列出来，根据里面的内容拆解，去更新内容就行了，类似于五官画法、人体比例、小动物画法等等，每个主题，还可以细分成更多的内容。

这个方向好像可行，但问题是，对于我们这个略显懒散的二人团队来说，还是任务有点重😂因为一口气做完一门课程，其实还好，但是一边做课程，一边做内容，那这个时间就拉长很多，更新频率也不一定跟得上。

我们开始狂刷内容，做更多的调研，迅速确定了一个主题，是「适合新手的插画风格推荐」

内容就是推荐和解析一些国外插画师的作品。

队友负责找插画师，我负责内容设计和编排，这也是我作为设计师比较擅长的，努力一下做到日更基本没问题。

这个系列，利他性很强，美观度很高，我们自己做的过程中都能学到东西。

另外这个主题发出来，就是针对我们想招生的人群「插画新手」，也能顺便调研一下大家的风格喜好。

同理，我觉得换成别的行业的账号，或许也可以尝试一下这个模式。

将「拆解优秀同行的内容」做成内容，来同时达到利他属性、人群调研，以及积累账号更新的手感。

就这样，第一篇就获得了千赞，在发了大概10篇左右内容的时候，账号就涨了几千个粉丝（具体多少我不记得了）。

并且因为有「解析」的内容，就比较偏教学方向一点，这时候就已经有粉丝在问有没有课程了。

这算是我们“起号”的一个方式，不能说没有运气成分，但是我相信这个思路，是可以增加一些起号出爆款的可能性的。

## 3、通过内容的数据反馈，确定教学方向；

刚才那点提到，「适合新手的插画风格推荐」这个主题发出来，就是针对我们想招生的人群「插画新手」，也能顺便调研一下大家的风格喜好。

内容的数据反馈是很直观的，赞藏最多的就是大家最感兴趣的。

大家最感兴趣的风格，就是我们的教学方向。

这里有个前提就是，我们过去的插画与设计，大多是商单，属于什么风格都可以做，什么风格都能画，但是到了自己做，没有甲方需求的情况下，画什么反倒成了一个问题。而做风格解析，再看大家的反馈，就成为了很好的锚定目标的方式。

后来根据教学中的学员反馈，以及粉丝评论，我们又逐渐开始加入一些大家呼声较高新的课程，比如小动物课，以及之后要上的小物件课程、婚礼插画课等等，这些都是通过反馈得来的。

当然，我们选择积极的反馈，去行动去落实，但如果是一些消极、恶意的、除了你自己其实没有人在乎的评论，就不要去回复了。

## 4、如果更新频率不够高，就要有节奏地去宣传。

这个意思是，当我们没办法创作足够多的内容，那就把内容攒一攒，放到招生期间发。

为什么特意提这点，因为这其实也是我们的一个弱势。

插画设计类创作，短时间能出的内容量是有限的，因为还要上课，就很难做到日更。

所以我们只能，画完先不着急发，平时能做到周更两篇就可以了，到了招生期间，才有足够素材日更，多一些曝光。

# 设计插画类账号的接单

我想大多数设计师、插画师还是希望能通过公域实现接单的，但是苦于没有方向，不知道怎么起步。

所以也分享一下我们这个账号「接单」的几个方向。

通过这几个方向，陆续开启约稿之后，约稿的作品就正好作为小红书的内容发布，

这样下来接单反哺内容，形成了一个良性循环，算是比较省力的一点。

## 1、婚礼插画

必须要感谢我的好伙伴@雪姨！

正好也是我们在起号的阶段，雪姨找我给她的朋友画了一组「婚礼插画」，

当时我们就结合准备教学的风格给她画了，效果很不错，后来我们发到账号上，这个内容获得了2000+的赞。之后就有不少人咨询我们婚礼插画的约稿。

这个市场需求是非常大的。而且咨询者基本都是新人自己或者要送朋友，比起价格，他们更注重质量，这就比小红书里大量的低价市场要好很多。

## 2、公司员工头像

另外，雪姨又给我转介绍了生财的伙伴，为他们公司的员工画头像作为礼物。（这样的领导和公司氛围也太好了吧）

## 3、红包封面

红包封面因为是定制加上时效性，（加上过年了）我们就没有太来得及发小红书，但是去年的春节期间确实还是画了一些的，正好最近也快过年了，大家也可以参考一下。

我们在春节期间开始给朋友定制红包封面，当时正好有视频号赠额，今年可能也会有，大家可以留意一下。

当时我们提供了「动态涂鸦」以及「人物插画」两种风格，并且针对兔年主题提供了可选文案，以及操作步骤。

视频号发红包封面引流效果是非常好的，所以需求肯定有，我们没有公域宣传，主要是朋友转介绍来，这样春节期间，在比较佛系的情况下，大概接了十来个红包封面这样。

红包封面的实操，我看了生财有很多厉害的帖子，去年到今年一直在看！非常推荐大家做。

## 4、宠物插画

宠物插画的需求，也是从婚礼插画里衍生出来的。

因为找我们画婚礼插画的新人，很多都加了自己毛孩子。

这时候开始有不少粉丝和学员，说小动物画的很可爱，以及“不会画小动物”，提议我们开小动物插画的课程。

顺理成章地我们开始发一些小动物的作品，最近也陆续接到了一些宠物插画的订单。

发现很多来约稿的人，他们给自己家的宠物，都在不停地约稿，用插画的方式记录自己的宠物，所以这也是比较容易有复购和转介绍的。

约稿的人，会把这些插画做成头像，或者给小动物印卡片、做些小徽章，小配件，都很有意义，我们画的过程也很开心（因为自己也很喜欢小动物）。

# 转化与报价

以上是这段时间我们公域私域接单的几个方向，当然还有一些别的，比如全家福，比如活动海报，动漫、艺人周边等需求，相对少一些，就不一一列举了。

分享一下我们目前的项目报价，大家可以参考，但是如果自己做的话还是要根据自己的时间成本、预期收益来定。

因为我们平时的“主业”都是以商单为主，报价是对公的，比如一张海报平均是四五千的报价，一张插画，品牌/公司有这个预算，是因为它的商业价值，要用于宣传，要露出，要延展。

但是放到小红书，放到个人单上，这个报价就不适用了，毕竟大家约稿只是自用，所以不能一概而论。

小红书上的总体报价，对商业插画/设计师是偏低的，但是好处是接稿自由，约稿也是一个双选的过程，基本对方看中了你的风格，要的就是类似的东西，所以沟通都还比较顺畅。

大家可以参考，我们调整后的价位，是我们接公司商单的1/4—1/3的样子，这个报价大家也可以结合自己的商稿报价来定。

### 1、接稿的四个方向，报价如下

婚礼插画（或任意双人情侣图）：半身400，全身600

个人头像：200/人

红包封面：200

宠物插画：50—100/张

### 2、课程报价，目前有3个档的课程

1、录播课：99

录播课是长期招生，课程一共10节，一次打包给学员，留10张作业。

点评作业是直播，每周直播1-2次，每次大概2小时。

2、短期的直播课：199

指的是头像、小动物等，时长在一个月内的专项训练，

包含大概10-12次直播，以及一周1-2次的点评，也是每次大概2小时。

3、年课：549

为期一年的综合训练，每月一个主题，每月的第一周上三节直播课，剩下的三周布置和点评作业。

点评的直播大概在一周一次。

这三个档的课程，在市面上价位是偏低的，因为第一年做我们还是没什么概念，不太敢定价。

在这一年做课的过程中，对自己的课程也更加有信心了，我们确实是掏心窝子地在教学，加上陆续也咨询了一些经常报课的学员建议，之后都会涨价。

### 3、转化：

我们正式招生和接稿是今年3月份启动，到年底9个月，这个号月均收入稳定在1w+。

（具体是，专门用于约稿和招生的微信小号目前是加了三百多个人，因为都是小红书询价后才加的微信，所以基本都是确认要下单的，加微信的人中大概是80%的下单率，课程+约稿一起，均价在400）

# 卡点与避雷：

这个账号目前作为副业来讲，还是比较适合的，做自己擅长的领域，基本在我们的舒适圈内。

但是最近也就卡在这里了，以及中途遇到过不少小问题，我们也分享出来，

一个是提供一些注意事项给大家，另外，也希望能寻求一些新的解法，欢迎大家一起讨论。

### 第一，目前我们90%的转化，都来源于小红书。

而小红书的流量，是不稳定的。

前期我们起号的流量不错，然后从双十一起到现在，流量是有明显下滑的。

完全依托于一个平台的流量，这个我们就很容易焦虑。

加上小红书需要引流到微信，这个就有风险。

目前的打算是，做两个小号，把内容分一下，

一个是视频内容，以教学视频为主，除了小红书，也将内容分发到抖音和视频号

另外拿一个小号来做客服招生。

### 第二，教学平台的不稳定（CCTALK）

之前决定用CCTALK是真的觉得，它的操作、教学流程，都是用起来最顺畅，最适合我们课程的，加上很多有名气的插画师都在上面开课，当时虽然已经有一些传言，但是我们天真地觉得，这么多大V在上面，应该不至于。

事实就是我们确实太天真了……刚入驻不久就遇到了网传的问题，提现困难，客服消失。一次结算到提现基本都要一个多月（还是在不停催客服和尝试举报的情况下），所以上了两个月之后我们也决定转战。

目前我们是用【企业微信】直播，然后在【知识星球】给大家布置作业，用下来还是很顺畅的。

但用【企业微信】的时候，用了一段时间，微盘内存满了，只有认证公司扩容，前后又花了一个多月时间。

认证成功前，不得不把所有视频压缩一遍重新上传，最大化地利用微盘空间，好在在新课开启之前顺利解决，否则又要面临让大家换平台看课程的问题，会非常影响学员体验。

### 第三，线下化的目标

目前是希望能在明年实现课程线下化的，做成画室+工作室的模式。

第一，对我们来说，线上教学，客单价低，如果要做高客单价的课程，需要更长的时间打磨以及团队运营，目前我们二人公司，比较难承载。

线下教学，客单价会高很多，另外教学的内容也可以更专业，更体系化，这个是我们擅长的。

第二，弥补线上流量问题。

线下能有多种引流方式（除了小红书抖音之外，想做美团点评，以及线下活动等等）

然后，目前的线下课基本都集中在晚上和周末，其它的时间我们是没太利用好的，所以也想通过线下把白天的时间更充分地利用起来，

第三，因为我们主业也是做商业插画与设计，所以想有一个场地承载更多的创作和可能性。

当然，这又面临了许许多多的挑战，开店的预算，场地的选择，装修，店铺的运营宣传等等，这些会需要可能小半年的时间。启动的话我也希望将来能分享出来。

# 总结

为什么要写这一篇，

因为我能感受到，从事设计、插画行业，真的是一个有非常多可能性，也会经历非常多困惑的过程。

很多人都想做自己想做的创作，但是不知道从何开始，或者难以变现。

我们之前尝试过很多，做潮玩，做动画，做周边，都没有做起来。现在回想起来，大多数卡在，时间成本太高，对市场不够了解，自己不够坚持，以及正反馈不够。

摸索了这么长时间，终于有了一个成型的项目。

这个项目有顺利的部分，也有很多不足，所以分享出来，

欢迎一起讨论，如果能给大家有一点启发就更好啦。
