# 生财小白经过6个月的迷茫期，进入生财后，成功转型月入2w+

> 来源：[https://rxxy4o2wwcf.feishu.cn/docx/QuHYdyjapoaF71xwRqfc8nXrnPc](https://rxxy4o2wwcf.feishu.cn/docx/QuHYdyjapoaF71xwRqfc8nXrnPc)

谨以此篇送给未跑通闭环的家人，这是我过去一年的心路历程，希望能对正在路上的你有帮助，大家好，我是徐子然，一个做了10年销售的打工人，2023年真的是徐子然脱胎换骨，改头换面的一年

再此感谢@梁靠谱，感谢@李千万，感谢@芷蓝

因为2023年，我正式开始了我的副业之路，说起来是一路坎坷，2022年10月份我从北京辞职，结束了我长达6年的北漂生活，回到了老家云南，完成了我的第一件人身大事—结婚

2023年我们正式开始在互联网上找副业，通过公众号了解到学科项目，加了一个老师，付了699，正式开始学习，在这期间，我承认我是幸运的，用了一周时间跑通了闭环，然后我就开始买手机，办电话卡，开始批量起号，随着每天发的笔记的增加，引流的人数也由1个变成了10个，变成了20个，所以收入也得到了提升，从每天几十元变成了每天几百元，最高单日变现3000+

当然，这也让我尝到了副业的甜头，但是好景不长，3月中下旬开始，账号频繁被限流，我只有频繁地注销账号再重新注册，这期间，来来回回我至少注销了30多个小红书账号，依然无果，让我陷入焦虑迷茫的状态

因为限流太严重，我又开始了频繁找项目，看到别人说考证的资料比学科资料更加好卖，加上那位老师的朋友圈，每天都是各种应酬和高端商务局，成交截图，客户好评，而且年纪轻轻就买了保时捷718，我承认我心动了，于是花了999买了她的课程，但是我进来之后，发现这些课程和她说的并不相符，问问题也回复特别慢，基本几小时才回复一次，这就让我很难受

所以只能自己摸索着去做，在做的过程中，某一天我突然刷到一条笔记，这张图片好熟悉，我立马打开对方朋友圈一看，图片一模一样，在看定位，头像，昵称也不相符，我不确定，我也不敢相信

![](img/c517dbd84fd5cf28fe9528a806f87d0c.png)

于是我又搜了下同款，我有找到了另外一张图片，我死心了，原来对方买的保时捷是在小红书上买的

![](img/529be74b0b234847648cf5dc4bc96f6a.png)

这让我再次陷入了迷茫，4月中旬，我通过鱼丸拿到了3天的体验卡，然后花了2000多加入了生财有术，进到生财，每天看精华帖，很多都是月入10w+的大佬，甚至还有很多00后，我又陷入了焦虑和迷茫，甚至感觉到自卑和无助，不知道该何去何从

伴着焦虑的心情，终于迎来了生财的航海，因为那段时间，我的朋友圈几乎都在卖小红书店铺的课程，都说是2023年的趋势，进去就能赚钱，利润都在50%以上，我一看生财也有小红书店铺的航海，就很开心

但是离航海开始还有10天左右的时间，等待的过程是漫长的，加上朋友圈的爆单截图，我非常的焦虑，所以在焦虑的情况下，我又花了3680买了一个小红书店铺的课程，所以放弃了参加航海，为了更快拿到结果，我和老婆2个人，一次性开了5个店铺，天天刷对标，刷同行，选品，剪视频，发视频，下单，发货

从信心满满到垂头丧气，也不是不出单，一天有3到5单，因为客单价低，所以利润就比较少，每天几十元的利润，然后问对方应该如何去精进视频，对方告诉我，多发，持续发，问她怎么选品，告诉我去拼多多和淘宝选品，每次问问题，都得不到答案，慢慢地我也就不问了，直到有一天，她给我发信息：宝，我看你某某商品出了20单，接下来就是继续发视频，堆量

所以这就是私教吗？看着他们的朋友圈，天天有人爆单，我不甘心，为什么别人可以，而我不行呢？耳饰，乳帖，儿童玩具，厨房用品，解压小玩具，汽车用品，百货都去尝试了

每天有两三单，给你希望，又让你绝望，每个品都能出单，就是无法打造爆品，我也不知道问题出在哪里，也不知道该去哪里找答案，应该如何去精进，就这样半死不活地熬了半年，这期间，我们俩无数次怀疑，是不是自己不适合这条路？还是踏踏实实找个班去上，混吃等死吧，但是我就是不甘心，我和别人到底差了什么？

9月底迎来了生财新一期的航海，我一次性报了3条船，小红书店铺，视频号带货和选品特训，每天按照航海手册和学习的资料去执行，我报着试试的心态，开始了视频号之路，我没想过会出单，甚至选品都是随机的，每天发3个视频，第三天无意间打开了视频号，看到今日数据：17单

我靠，出单了，就这样，我们放弃了小红书，来到了视频号，一路摸爬滚打，踩了很多坑，也走了很多弯路，也喜提了很多个限流和橱窗被封，当然比起做小红书的时候，收入也有所增长

直到10月23日通过梁靠谱，加入了玩赚合伙人，命运的齿轮开始转动，在靠谱的指导下，行动力加强了，目标也明确了，咔咔就是干，月收入稳定在1w左右

直到12月1日，我通过生财，看了李千万老师的文章，添加了李千万老师，命运的齿轮再次转动，我加入了李千万老师的视频号私教，在李千万老师的指导下，我的视频号得到了突破

实现了单日收入3000+，一周8000+，15天破万，单月收入2w+，当然这仅仅是开始，未来我会突破月入5w+，月入10w+，路虽远，行则将至，最后，我想说，对于新手小白来说，最重要的不是什么项目是风口，而是踏踏实实地去学习，去执行，在执行中找反馈，今天我们所踩的每一个坑，再未来都会变成我们的经验，把经验分享出去的过程，就是变现的过程

![](img/1dbfce09f209aeef84fd503932190ca9.png)

电影《误杀》中有那么一句台词：如果你看过1000部以上的嫌疑电影，你就会发现这个世界上根本没有任何离奇事情，同理，只要你仔细研究过1000个你所在行业的案例，你就会发现所有问题都不是问题，只要你拆解过1000个以上的项目，你就会发现，所有项目都是想通的

感谢大家，未来我们一起生财有术，一路同行，希望你能早日找到适合自己的副业项目，跑通闭环，拿到结果，期待你的报喜
