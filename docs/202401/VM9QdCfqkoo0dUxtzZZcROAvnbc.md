# 年报（图文详解）---小副业 副本

> 来源：[https://p0xfhvmcoj.feishu.cn/docx/VM9QdCfqkoo0dUxtzZZcROAvnbc](https://p0xfhvmcoj.feishu.cn/docx/VM9QdCfqkoo0dUxtzZZcROAvnbc)

1，打开网站：https://www.gsxt.gov.cn/corp-query-homepage.html

2，选择企业信息填报，每年1-6月份年审上一年的执照

个体和公司年审步骤都是一样的

![](img/321c1466e78bd0aeee57ff71a2a7b593.png)

3，选择执照所在地

![](img/d610fccb92bfa4d17a25afaec346a2dd.png)

4，登陆，选择个体经营者登陆，然后按照信息填写即可！

也可以选择电子营业执照登陆，微信搜索小程序：电子营业执照，打开小程序扫码登录。也是方便简洁，主要是看怎么登录方便怎么来，多研究一下就行。

![](img/aaa054409424828cee2e67cfdbef9929.png)

5，选择年报填写，跳弹窗，选择：已阅

![](img/95988aa5a1848f36f45c27e1e8afc19c.png)

6，填写年报内容，自行根据实际情况填写

![](img/05eb35fb494992c0a85ec2b4d24a6215.png)

7，保存并公示

![](img/631492920336db8c58a9925939490715.png)

8,答题，答案再最下面

![](img/c93b71a5ee77ddb180e5085cb6f6f0c1.png)

9，公示成功

![](img/4b34f62446f6d823f15dc69d7fbbde0c.png)

小副业，也可以出去外面接单，帮别人年报！几分钟完事。价格10-200不等。
