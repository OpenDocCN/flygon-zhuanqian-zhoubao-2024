2 如何选择和申请广告主

2.1 如何申请广告主

我们通过上方导航栏，点击find Advertisers，可以进入选择广告主的页面。

在广告主集合页面，你可以通过左边的侧边栏，选择你感兴趣的类目或者广告主所属的国家，也可以通过关键词，搜索感兴趣的广告主。

选择好你希望合作的广告主以后，点击右边的“Apply to Program",就可以申请权限了。

所有通过你申请的广告主可以在左边侧边栏的“my advertisers(active)"找到。

审核中的在左边侧边栏的“Pending applications”里。

2.2 如何判断广告主的质量

cj的广告主有两个很重要的指标，分别是 network Earning 和 EPC。

这两个指标越高，则侧面说明这个广告主的质量也越高。

2.2.1 什么是EPC

Epc的全称是：Earnings Per Click (EPC)，每个联盟都有这个指标，但是这里要强调一下，每个联盟关于这个指标的解释可能不一样。

比如 cj 的 epc 是 每一百个点击产生的佣金数额，有的联盟则是每一个点击产生的佣金数额。

这一定要明确好，因为这个指标是我们判断offer质量最直接的指标，特别是你使用付费广告推offer的话，指标越高，则说明产生佣金的能力越强，转化率也越好。

2.2.2 什么是Network Earning

这是一个相对评级，指的是该广告主在一个周期下，支付佣金的情况，它支付的越多，则评级越高，侧面说明了和它合作的推广商赚的也越多。

3 cj联盟的付款政策

标准情况下，cj是次月付款，根据广告主和你账号的情况，你将在次月的20号或者次月的30号收到你上个月的佣金款。

比如你1月份的佣金，会在2月份的10号锁定，第一批收款是在2月的20号左右，第二批是在2月的月底付款。

绝大部分cj的广告主是默认次月10号锁定佣金的（如下图），付款就遵循以上所说的时间表。

如果锁定佣金的时间不是10号，则需要锁定以后，再进入付款周期，这个要看广告主具体的锁定时间。

四 广告主

1 广告主推广条款

我们申请到广告主以后，就是要看一下广告主的推广条款，包括它的佣金是多少，转化方式是什么，允许和不允许的推广方式。

就比如1Password 这个广告主，它给的条款是，当有一个新用户通过你的推荐链接进来注册了一个免费的账号，会给你2美金的佣金，如果它后续付费了，则再给你付费额的25%。

（部分广告主会给出达量奖，为了鼓励你更多的推广它的offer，比如当你每月的佣金达到1000美金，会额外再给你500刀，诸如此类，也会在 Program Terms这个页面说明）

2 如何拿广告主的推广链接和素材

当我们被广告主批准以后，要如何拿推广链接呢？

选择你的广告主，点击getlinks，会进入广告主推广素材的集合页

在这个页面，你可以看到广告主预设好的各种广告图和专属于你这个账号的推广链接。

你可以根据你想放置广告素材的位置选择相应大小的素材。

找到你想要的素材，点击右边的“<>”

这里的click URL 就是专属于你这个账号的推荐链接，所有通过此链接进入广告主的转化动作，佣金都将记录在你的这个cj账号里。

3 Deeplink 工具

广告主在cj给的推荐链接，基本都是跳转到主页或者他们某个推广页面的。

但是我们作为推广者，有些时候，是希望用户访问广告主的某个特定页面的，这样能减少用户的动作，让他们直入主题。

比如我推沃尔玛的affiliate link，用户通过搜索“吹风机”访问了我们的推荐链接以后到达沃尔玛主页，它还得自己去找“吹风机”，这样无形之中就会增加用户的跳出率。

如果我们要用户搜索“吹风机”点的affiliate推荐链接能直接到达沃尔玛的吹风机买卖页面该怎么办。

这里就要用到联盟提供的deeplink工具，这个工具可以给广告主的任何页面生成推荐链接，那么用户点击这个生成的新链接，跳转的就是你希望他到达的页面，同时还能记录你的推荐佣金。

点击Link tools 就能看到cj的deeplink生成工具了，使用方式请看红框，把cj的deeplink 插件拖拽到你chrome的导航栏，然后访问广告主任何你想生成链接的页面，就可以生成推荐链接了。

这是一家保险公司，在cj上的offer，是一个询盘给20美金，很典型的Cpl （ cost per lead )
