## (316 赞)如果我觉得什么事情对公司特别重要，我就一定亲自上去把它干成不重要

作者： 亦仁

日期：2024-01-04

最近最有启发的一句话，来自一位年利润大几千万的老板：

如果我觉得什么事情对公司特别重要，我就一定亲自上去把它干成不重要。现在，我觉得人力部门特别重要，明年就准备专门投时间进去，把这个部门干的没那么重要。

* * *

评论区：

W Z : 老板的核心就是让所有人所有人都正常就可以运行下去并取得不错业绩，特别依赖谁都不好
老船长李阳 : 是不是可以理解成:重要说明不可缺少，说明存在问题，或者有需求没满足，不重要就是解决了这个节点的问题。
徐蒙徐大爷 : 市场发现你这个企业很重要（不可或缺），就会冲一波上去把你干得不重要。
一个道理。
三亿 : 没有谁是不可替代的，所以发明了轮值岗位。
王先森 : 不想有软肋，有种一将功成万骨枯的感觉
阿怀 : 乍一看，不理解干成不重要，还以为有语病😳
仔细琢磨后，发现讲得好有道理，发现 SOP 就是来做这个干成不重要的
糖糖 : 俞老师：点谁呢？
陈文根 : 费老板，第一要发现问题点在哪，第二，要能解决问题！
