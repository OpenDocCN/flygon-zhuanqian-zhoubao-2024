# 微信红包开放平台

这个平台很好注册，建议大家都去注册一个

然后主要关注这个封面图要求。图片尺寸，大小等。

![](img/307311d9094c8f9ceb4bf0046dcf0c8d.png)

# AI助力

现在市场上ai作图工具已经很多了。我还是使用的传统MJ去做的。大家也可以使用吐司，创客贴，等作图平台去做。

![](img/e5502bc4ab93e3f056c0d406f1eecee4.png)

简单切图出来之后，就可以直接上传到红包开放平台了

上传之后自动显示效果图。

![](img/edaa66d2ee05346269d31c5bb6cbe0a2.png)

# 证明材料

目前AI作图证明材料比较简单 ，直接贴生成图就行了

也就是直接上传这张图。

![](img/1df28eedb43b99b0bb84217d5c953c4f.png)

# 提交审核

证明材料上传完成之后，就可以提交审核了。

如果证明材料过多记得打包成压缩包。

![](img/cbbc05c542483ffc9215d75d656bd956.png)

# 引流：

设置领取时间，这样用户就会关注你的公众号，到时间来领取。

而序列号可以放在公众号文章，抖音，小红书等平台。毕竟占便宜，肯定会有很多人来领的。
