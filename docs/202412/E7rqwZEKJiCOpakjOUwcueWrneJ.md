# 写作变现，公众号爆文帮你拿回掌控权

今天分享的第一部分，就是我猜：有些朋友可能看着其他航海的朋友开始赚钱了，但是公众号还是0或者个位数的收益，觉得公众号已经不值得做了，真的是这样吗？

开卷大学毕业以来，走过传统的写作变现路径，也在不同的公司靠文字赚过钱。所以，今天先和你分享写作变现的传统路径，来对比看看，公众号爆文，是不是真的“不行了”？

传统写作变现路径，开卷最开始是写片段，千字20，后来能独立成稿，能拿到千字200，再后来，能独立出一篇深度观察，能拿千字2000。

但是，不管收益多少，这个路径都十分煎熬。因为你会遇到千奇百怪的需求，遇到各种意想不到的推进中的波折，甚至还有客户在发表之前，临时决定不打广告了，推进了半个月的内容打了水漂。

做的时间久了，开卷总有一种被别人扼住命运咽喉的感觉，不可控的因素实在太太太多了，而且大部分是人为因素。做了几年，除了收益，几乎是风过无痕，没有个人资产的积累。

但是，咱们做公众号爆文，只要流量主这个项目还在，就意味着有需求也有市场，只需要研究自己擅长什么，以及当下的平台算法规则。

上次写完公众号复盘帖之后，有位圈友没有参加爆文航海，参考我的思路，没多久就出了单篇500的收益来找我报喜，20天之后，实现了日入1666又来找我报喜。可见，摸透了思路，做爆文号真的不难。

对开卷而言，之所以坚持做这个项目，不仅是我跑出了百万爆款，加速我达成全网千万阅读的目标，更重要的是，不管是写作变现的路径，还是写作变现的收益，公众号爆文让我相信，普通人可以完全把写作变现的主动权牢牢抓在自己手里。

所以也特别推荐，想把写作变现的主动权掌握在自己手里的圈友，把这条路坚持走下去。

# 自媒体变现，普通人弯道超车的最短路径

今天分享的第二部分，就是我猜：有朋友可能到了现在还没什么水花，就想换项目或者换平台，我知道有些朋友着急，但是你先别急，可以看看开卷的踩坑经历。

开卷全职自媒体之后，做过小红书、抖音图文带货和知乎带货。说实话，水花都不大。

后来我认真复盘了一下，小红书、抖音和知乎，普通人没有自己的产品，基本上都是先靠内容涨粉，然后接广或者做课程变现。

如果接广，一定会面临开卷前面说过的和客户沟通需求反复拉扯的问题，以及目前最大的问题：很多客户没有投广预算了。

如果你从0-1做个人IP，更需要全方位的商业闭环能力，少了哪个环节，其他的都是白费力气。

如果公众号爆文没做起来，可能在内容层面还有提升的空间，换到其他平台，同样需要面临内容的考验。

你可以考虑一下：你想因为内容没做好，就要一直换平台吗？你想因为不会做内容，而一直冒着风险抄袭搬运吗？

一旦你把公众号爆文项目跑通，有了你的项目基本盘，再去做IP，通过其他平台放大，也能更轻松。公众号爆文，很有可能是你做互联网项目挖到的第一桶金。

# 从爆文到IP，一通百通的底层逻辑

今天分享的第三部分，就是我猜：会不会有些朋友和我一样，是冲着想做IP类账号来参加的这期航海？

因为有些朋友，有自己的业务，需要积累个人资产，扩大影响力，也有些朋友，分销别人的产品，需要公众号这个相对其他平台有优势的渠道。

所以这里，开卷也来分享一下自己从爆文号做到IP号，3个亲测有效的底层逻辑，都是我偷偷总结出的小诀窍，希望能帮到你。

1.  选题定生死，秘诀是“取交集”

一个简单粗暴的找选题的方法，就是你找到3-5个你这个领域的优质对标，用“爱小助”或者其他能拉取公众号全部文章的工具，直接把标题先导出出来。

然后在这3-5份标题文档里面“取交集”，同一个爆款选题，有些作者会反复写，同一领域不同作者都会争先恐后地写，这两个交集取好了，找到高频出现的标题，能覆盖这个领域绝大部分的爆款。

把你筛出来的至少30个标题，一天写一篇，先干他一个月，不管写的有没有大佬那么牛，先发出去，以周为单位复盘，慢慢调整。

1.  内容定人群，秘诀是“偷梁换柱”

做IP号之后，我发现有些专业大佬，对爆文标题不屑一顾，感觉是很浅显的套路，但是又心底里偷偷羡慕别人的流量，这里，开卷给大家分享3个我看到的灵活变通就能改写成自己领域的爆文选题。

参考这种思路，你会发现，爆文标题+定向内容，真的很香！

选题①——布局2025：

这个选题，年末年初都非常火，像成长博主、减肥博主、养生博主都能用。

开卷写了一篇《布局2025：普通人的顶级发财清单（建议收藏）》，这个选题是“布局2025”和“清单体”的结合。

换到其他领域，是不是也可以写：布局2025：普通人的顶级自律/减肥/成长/圆梦……清单（建议收藏），把标题关键词改成你的用户需求关键词，把文章的核心内容，改成和你的领域相关的内容，就可以写类似的文章。

选题②——业务模式拆解

开卷写过一篇《一个人就能闷声发大财的高利润生意》，这篇文章就是把高利润生意的模式拆解出来，让人能读到干货，有信息增量，而不是通篇硬广。

如果你有自己的成熟的业务模式，或者分销别人的产品，也可以通过拆解别人的业务，写类似的选题。

选题③——需求合集

相信不少圈友看过《不想上班，一个人也能闷声赚大钱的6个野路子》这种文章吧？有没有可能，咱们如果是做知识付费产品分销，就是把自己的产品，包装成这6个野路子之一呢？

这样，咱们赚的不是纯流量和搞噱头的钱，而是赚的精准用户咨询项目的钱，是不是瞬间豁然开朗？

类似的选题思路大家可以多多琢磨和研究，是换一些领域关键词或者换个方式思考，只要能自圆其说且不违和，都可以试试，万一试出自己的爆款套路了，不就赚大发了吗？

1.  情绪定爆款，秘诀是“逐步调整”

最后想特别说一下情绪，因为这个点用好了，可能让你的文章成为爆款，用得不好，你的流量主文章可能被投诉没收收益，也有可能IP号文章完全起不来。

如果你逻辑思维强大，文章极其克制，在某些领域可能会比较吃亏。

尤其是，写情感故事、人生哲理、社会热点相关的文章，如果没有加入情绪，就像没有灵魂，这也是为什么有些人仿写爆文，却不温不火。

调整的方式，要么去刻意训练情绪表达，要么去做不需要太多情绪参与的文章，比如技术类、严肃科普类。

如果你表达情绪的能力很好，也要注意，因为如果你写的是负面事件还特别能煽动情绪，可能会有风险。

开卷之前写过一篇偏负面的热点，没多久就被系统删文了；目前在做的IP号，这个方向在今年7月就尝试过，一写就噶，现在想起来，可能也是当时仿写的一些文章消极负面的情绪过重。

关于情绪，开卷的建议是，如果你要用情绪出爆款，最好是选择相对正能量或者治愈的切入点，哪怕最开始的话题引入是悲伤的，但是要能转过来就没什么太大问题，如果情绪激烈又消极负面，风险需要琢磨把控一下。
