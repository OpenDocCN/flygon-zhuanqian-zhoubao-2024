# 创业的两个阶段：找项目与持续摇人

## (35 赞)创业的两个阶段：找项目与持续摇人

作者： 勤学

日期：2024-12-18

阶段一 找项目 （1）项目能正常跑通 （2）项目存在高坡厚雪 （3）项目特性和自己的盖洛普花色比较匹配 阶段二 持续摇人 （1）阶段一跑通项目时需要摇人
（2）放大项目需要摇人 （3）同一个项目存在不同的发展分支，想要做好任意一个分支，都需要摇到对应的人。 1、摇人的两种方式 （1）看精华帖
（2）参加星球线下活动 2、摇人的标准 （在自己想要突破的项目发展分支） 同一个项目，可以闲鱼获客、可以小红书获客、可以天猫获客、可以深入做后端供应链。。
如果自己当前阶段想要在闲鱼分支获得突破，那么就需要找到在闲鱼分支做项目月入 5 万+的人。 找别的分支的大佬作用有限，除非自己大换方向。 3、大佬在哪里
大佬在深广杭。 什么叫大佬？做项目稳定年入 100 万以上。 所以，不论一个人身处哪里，在项目开始阶段，可以囿于本地，充分利用精华帖、手册、地区聚会，跑通项目。
但是在项目放大阶段，毫无疑问是要勤于跑深广杭参加高质量线下活动的。 否则，很难摇到自己想要突破的项目分支领域的对应高手。 做作业容易，还是抄作业容易？
一个人在项目没有跑通时，是很容易当韭菜的。 随着自己这颗韭菜被越割越粗壮，项目跑通了、能够识别好课了。
“一年六位数的知识付费”会成为向大佬看齐的方向。我在很多位大佬圈友的精华帖中看到过这句话。 需要解决一个问题 →
从星球几万人中摇到对应的人（我基本只买精华帖圈友的课） → 付一笔钱 → 问题顺利解决，上台阶 所以，随着创业从阶段一迈向阶段二，这 3 者会同步提高。
宇宙有大小之分，但每一个宇宙的突破，都需要以年为单位。 （1）闲鱼宇宙 （2）小红书宇宙 （3）视频号宇宙（公众号、短视频、直播三位一体）
我认为，上述 3 个宇宙是从易到难的。 就像雷军做小米手机，米聊不敌微信，错过了一个宇宙。 我们在专注上述其中一个宇宙的时候，也必然会错过其他宇宙的重要窗口期。
目前来看，这是一件没有办法的事情。 好消息是，通关一个宇宙之后（），通关下一个宇宙就会顺利很多、快速很多。
就像雷军在金山苦干 16 年，才终于使其勉强上市，但市值远低于 BAT；
小米手机十年（2010.4-2021.3），出货量全球第三。小米公司成立 9 年即进入世界 500 强； 3 年时间，小米汽车问世，惊艳众人。
金山时期、小米手机、小米汽车，刚好对应闲鱼宇宙、小红书宇宙、视频号宇宙。 没有金山时期的苦熬，哪有后续小米的辉煌。
同理，当我们做项目时，不论我们最先上手的是哪个平台，都需要像打上甘岭一样，务求通关。 我猜测，说不定这就是 上班的时候有人管，人是靠外力推动才去干活。
创业者万事自己安排，每天醒来全靠兴趣驱动。 在阶段一的时候，会过手很多项目，有些项目只要下功夫做，确实能看到别人赚到钱，奈何自己不喜欢。
到了阶段二就会知道，只要试的多了（把星球的项目挨个试一遍、实在不行多买几个 1 万人以上的星球），总是能试出自己喜欢的项目、试出和自己盖洛普相匹配的项目发展路线、用一种顺着自己人格发展的方式把项目做起来。
在阶段一的时候，人是放弃自我的。星球推荐什么项目、什么项目好赚钱，我就做什么项目。 但是，对自我的放弃并没有带来项目的跑通，反而处处碰壁。
到了阶段二，对自我内在的追寻、对个人兴趣的投入，反而成为了项目推进的核动力。 人和项目获得了统一，项目发展成为了个人意志、精神世界生长的显化。
所以，阶段一测项目的本质就是测出自己喜好的过程。 阶段二就是放大自己的喜好、为自己的喜好投入更多的时间、的过程。 创业也可以每天都很快乐。
