# 四、赛道分析和自我定位

1.  为什么需要赛道分析

第一：在当今内容创作的世界中，时间成本无疑是最大的支出。因此，深入分析当前热门赛道，将有助于我们事半功倍，避免走冤枉路。

第二：YouTube上的创作内容每天以惊人的速度增加，海量作品让新创作者在一开始就找到一个创新赛道变得异常困难。因此，最有效的方式是：首先大胆尝试，迅速上手，然后根据数据反馈不断优化，逐步摸索出属于自己的细分领域。在这个过程中，积累经验与技巧是成功的关键。

第三：出海创作的最终目标是盈利。在这一点上，明智的投入产出比分析显得尤为重要。我们需要仔细评估所选择赛道的潜在收益，并对自己的时间和成本投入设定合理的预期。通过这样的方式，我们不但能够确保资源的有效利用，还能为实现梦想与盈利铺平道路。

可灵AI：国产，有免费次数（免费要排队），收费便宜，效果较好

海螺AI：国产，有免费次数（排队不要紧），收费便宜，效果不如可灵AI，可用于低成本验证

RUNWAY：海外，收费较贵，某宝可搜共享账号，体验下来效果一般，可能是提示词不对。

建议前期先白嫖免费次数跑通，熟练之后我还是固定采用gpt生词+即梦AI（包月69 会员无水印 可选参考图解决一致性问题）+海螺AI（包月79 会员无水印）的方式建立工作流，免费的是最贵的，时间成本也是成本，大量的生图重试以及排队流程是一笔不小的时间开销。

第一个视频我大概剪了十个小时左右（加上看生财攻略、探索AI工具、反复生图、文档记录的时间），因为是图生视频，所以比起图片故事还多一步生成视频的工作。

确定工作流后第二个视频大概只需要3-4个小时（开了会员之后不需要去除水印、排队、以及切换不同工具），极大的提高了效率。
