**  **

**创业公司在低增长时期的领导力核心元素，主要包括守、攻、正三个方面。**

**首先，“守” 代表生存能力。**

对于创业公司来说，确保基本盘的稳定是第一要务。这包括业务的正常运转和团队的和谐协作。基本功的扎实程度，例如产品的质量和组织的效率，也直接影响着公司的生存能力。

更重要的是，公司要有正向现金流和盈利能力。这就像盖房子的基石，没有稳固的财务基础，再宏伟的商业蓝图也只是空中楼阁。创业者需要有稳健的财务管理能力，确保每一笔支出都能带来相应的回报，同时保持足够的现金储备以应对突发情况。

**其次，“攻” 代表破局能力。**

在市场环境不断变化的情况下，创业公司不能固步自封，需要有敏锐的产业升级能力。结合科技与国资的力量，可以为公司带来更多的资源和发展机遇。

同时，创业者需要具备破局能力，利用如 AI、出海等策略寻找新的增长点，避免在市场的 “下沉”
中被淘汰。例如，许多互联网公司在国内市场饱和后，通过出海战略在东南亚、非洲等地找到了新的用户群体，实现了业务的二次增长。这种破局能力需要领导者有长远的眼光和敢于冒险的精神。

**最后，“正” 代表沟通能力。**

在创业过程中，合规性是不可逾越的红线。领导者需要确保公司的运营符合法律法规，避免因违规带来的巨大风险。同时，创始人的 IP
塑造和与共情大众的沟通能力也非常重要。

一个有魅力的创始人可以吸引更多的人才和资源，而与大众的有效沟通可以帮助公司更好地了解市场需求，树立良好的品牌形象。例如，一些创业者通过社交媒体分享创业故事和理念，赢得了用户的信任和支持。

总之，创业公司的领导力是一个综合性的能力体系，需要在守、攻、正三个方面都有所建树。创业者不仅要像守城的将军一样，稳固公司的基本盘，确保生存；还要像攻城的勇士一样，敢于突破，寻找新的发展机遇；更要像正直的使者一样，合规经营，与大众保持良好的沟通。只有这样，创业公司才能在低增长时期乃至更长的商业周期中，保持领导力，实现可持续发展。
