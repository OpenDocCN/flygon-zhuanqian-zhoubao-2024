# 什么是解说类账号？

它是通过原视频，以及拼接不同素材的视频，在加上自己二次创作的解说词，融合在一起形成新的视频这种风格，均属于解说类。那它很明显的特点，就是要说出来，讲出来。短平快，三秒内拉住你的停留，并且通过吸引人的画面，消耗掉你的60秒，甚至更多的60秒。

解说账号它的领域很广，科普解说，游戏攻略解说，爆款视频解说，动物解说，动漫解说，电影解说等等这些类目都属于这种类别。

# 科普解说：（拼接类解说）

其实说科普解说比较笼统，很多人会认为，科普解说是科学科普，地理人文这种讲解的视频，其实不是，如果我们换个概念来理解的话，它就是十万个为什么，你没见过的东西，现象，事物，我都可以解说出来。

但如果在创作者的角度上来看，我更容易把它称呼为拼接类解说。

那么为什么叫这个名字？

因为它是通过当下这个视频的选题，采集围绕这个选题不同的素材，通过拼接，放在一起的手法，这种我把它称为拼接类解说，那么便于理解，它也属于科普解说。

那么它的操作手法，就是围绕选题，找到和选题强相关的素材，通过转场和音效最后合成的视频。

当你理解了这个概念，你可能现在就想跃跃欲试，别急，因为这个板块，也分为正练和逆练，需要的素材，也不一样，所以，先耐心看下去，你在去施展拳脚。

拼接类解说的好处很明显，它基本上不会出现0播，低播的情况，当你完成了标准化，各个视频都是万播，除非你的选题，素材清晰度，展现画面吸引度，文案，太差了，不然万播就是标准值。但也有缺点，那就是它太稳重了，天天万播，一点波澜都没有，只是说不定哪天你的一个月前视频会被挖坟，然后跑的很高，属于单兵作战比较强劲的那种。这种类型，对比后面要讲解的爆款视频解说（原视频解说）而言，流量会低一些，但胜在稳定，后面会讲为什么。

## 正练：

拼接类解说的正练比较麻烦，因为你要先确定选题，在去围绕选题去寻找素材，通过选题写出来初稿，然后通过改文模版二次进化文案，然后配音，剪辑。在这块儿上，确实不如逆练方便一些，因为逆练，你只需要洗稿，寻找不同一个选题视频素材，在剪辑即可。但，还是会讲。以下几种方法来完成正练。

# 爆款视频解说：（原视频解说）

爆款视频解说也叫原视频解说，在操作手法上，是以爆款视频深度二创去重，或者一个博主的多个视频剪辑在一起的创作手法。它的好处是你借鉴的都是爆款的视频，经过你的二创去重以后再次跑了一遍，那你的流量也会相对来更加的好，但它的缺点也很明显，当你遇到这个小气的博主，他可能会过来diss你，版权警告你，要你赔偿删掉视频之类的情况。所以在我们选择爆款视频的时候，要么不以人物为主体，要么以国内的爆款素材为主体，这样避免了很多的风险。

爆款视频解说在素材收集以及剪辑手法上和科普解说非常类似，只是一些小细节略微有差异。所以重复的内容就不在讲了，主要讲一下不同区别的操作。大概是正练的描述手法，以及文案模版，和剪辑的时候不同的操作细节。

# 二创本源-站外素材获取

我们做的是Shorts，它属于YouTube，那除了这里，其他的平台都属于站外，国内抖音，快手，海外推特，INS，TIKTOK，都是你可以获取到的视频来源，你要建立自己的素材库，比如，做爆款视频解说，你就要抢爆款视频的二创首发机会，做拼接类解说，你就要收集同类型题材的素材。

相信我，初期很难，当你构建好这些体系，制作这两种赛道的内容，简直不要太简单。

那么，这里讲解一下大概步骤：

先建立平台采集的根据地，比如，我先锚定TIKTOK，推特，INS，这三个平台，复选国内抖音，快手。国内的素材大部分加入的文字很多，你在去重去水印方面很麻烦。

这里的分支要先了解赛道属性，爆款视频解说，它很多和事物，人物相关，那么使用海外的人物视频素材做解说就会面临侵权的风险，你的视频爆炸了，遇到小气的他会过来找你的麻烦，你会收到版权警告删除，找你赔偿等等。那我如果做爆款视频解说，我优先选择国内的爆款视频素材，这样能降低更多的风险。

对于科普解说而言，拼接类素材去重，合理利用版权基本上是允许的，所以基本上不会有任何的版权风险，获取素材的源头就可以首选海外平台。
