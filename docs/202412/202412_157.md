首先自我介绍一下我是卢卡卡，三期大航海的短视频教练，2018 年做抖音，对于垂类知识 ip 打造有一定经验，19 年做法律垂类科普账号，做出过百万粉丝账号

宠物科普类账号就是我此前一年多主要做板块，一年多孵化出多个抖音宠物科普账号，宠物知识讲解员是目前的主要账号。

也做过电影知识科普，和一些泛知识科普，都有几十万粉丝

今天主要是分享一下我做账号的方法论，以及自己打造 ip 的认知，这篇文章主要分享给小白，讲的稍微浅显一点，后续有机会再出一个文章讲讲，我是如何把科普账号做到百万粉丝的。
