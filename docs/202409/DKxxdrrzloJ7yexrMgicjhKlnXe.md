## 制作简易版扫雷游戏

相信windows上的扫雷游戏大家都不陌生，规则并不复杂、但有一定的益智性，我就以这个游戏作为案例，向大家展示我开始使用cursor时如何复现这个游戏。

下面分为2个部分，1.1使用的是ChatGPT-3.5模型，借助了我一丢丢的开发经验，让AI使用Qt这个可视化编程工具来实现的，但是也可以说是硬莽了。1.2则使用了claude-3.5-sonnet，并没有限定python使用什么库，这时AI选择了pygame库。

下面开始展示。

```
请使用Qt制作一个4*4的扫雷游戏，这个游戏有一个二维界面，这个扫重游戏的要素如下:
1.地雷：在网格中随机分布的地雷，在这个4*4的游戏中，共存在3个随机分布的地雷。
2.扫雷：玩家使用鼠标左键来打开格子，若格子是地雷则游戏结束，若该格子不是地雷，则该格子会显示数字：1）当格子非边缘格子时，数字为该格子周围8个格子中地雷的数量，2）当格子为边缘格子时，检查4*4范围内周围格子中地雷的数量；
3.当玩家点击完所有非地雷格子，则玩家胜利，游戏结束。
```

# 二、cursor的安装、注册（需魔法、使用的全过程软件有报错优先考虑魔法代理问题）

## 安装与创建账号

官网地址：https://www.cursor.com/

![](img/d3a0436841973512ad8b499b1c676227.png)

先点击右上角的Download下载安装包到本地；

下载过程中点击右上角【Sign In】按钮进入登录页面，可以选择用谷歌或者github登录，也可以点击登录页面最下面的【Sign up】进行账号注册，如下图左侧所示，下图右侧是注册页面，First name和Last name随便填，不重要。

![](img/8a871d7a50b3e087ceea215fe4fee89b.png)

![](img/d74113028e560f822882523f86c5b830.png)

注册完成后，安装包也下载完成了，双击【Cursor Setup .exe】安装即可。

安装后后在客户端点击Sign In然后跳转到登录页面，输入刚才注册的账号密码就可以，编辑器风格默认VS code，开发大佬自行选择Sublime或其他，不太熟悉编程的用默认即可。

和ChatGPT一样，Cursor有免费版和Pro版（20刀），只要有邮箱就可以注册，而免费版可以白嫖2周Pro（所以你懂，但是不提倡圈友打信息差去卖给那些不会注册的人）。

![](img/f9cf9a72b56ab1497e21fada94d5c53e.png)

## 如何更高效的使用cursor

### 程序功能性能要求描述

*   在向 Cursor 提问时，尽量具体和明确，明确你需要的以及不需要的，罗列你需要的所有目标以及你想通过什么方式抵达目标；

*   不提倡模糊的描述，比如：“我需要帮助写一个程序”、“帮我写一个爬虫”；

*   直接描述这个功能例如：“我需要一个函数来计算斐波那契数列的第n项”；

*   如何罗列目标和方式：帮我写一个爬虫，爬取公众号（链接）的文章，以及描述清楚要爬取的文章的特点，并且提供给AI想要以什么方式保存爬取的信息，如excel，excel需要存储那几列信息。

### 提供上下文的约束

*   指定你正在使用的编程语言，例如：“我在使用Python。”

*   如果适用，提及你正在使用的框架或库，例如：“我在使用Django框架。”

*   描述你尝试解决的问题的背景，例如：“我正在开发一个Web应用，需要实现用户认证功能。”

### 提供代码片段

*   提供已经在尝试的代码片段，不必完整，例如：

```
python
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```

*   解释你的代码尝试做什么，以及你遇到的问题。

### 错误调试需要详细描述error信息

*   如果你在调试代码，提供具体的错误信息，例如：“当我运行这段代码时，它抛出了一个‘IndexError’。”

*   描述你观察到的行为，例如：“程序在处理大于30的输入时会崩溃。”

*   明确你期望的结果，例如：“我希望程序能够无错误地计算出斐波那契数列的第100项。”

*   一般而言，将终端或网页上返回的error直接喂给AI效果更好。
