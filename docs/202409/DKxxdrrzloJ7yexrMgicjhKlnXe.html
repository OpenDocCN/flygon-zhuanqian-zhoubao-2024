首先简单介绍一下cursor的功能。

Cursor的主要功能：

  • 代码生成：Cursor能够根据开发者的指令或代码上下文自动生成代码，节省编写基础代码的时间。
  • 智能补全：提供更精准的代码补全建议，帮助开发者快速完成代码编写。
  • Bug检测与修复：AI能够识别代码中的错误并提供修复建议，减少调试时间。
  • 代码解释：当开发者对某段代码有疑问时，Cursor可以提供代码的解释和功能说明。
  • 文档生成：如README文件的自动生成，帮助开发者快速整理项目信息。
  • 代码优化建议：分析现有代码，提供优化建议以提升代码质量和性能。
  • 多语言支持：支持多种编程语言，为不同领域的开发者提供帮助。

Cursor的适用人群：

  • 编程新手：刚开始学习编程的，Cursor的AI辅助可以帮助他们快速掌握编程技巧，提供代码示例和解释。
  • 专业开发者：经验丰富的开发者可以利用Cursor的AI功能来提高编码效率，自动生成代码片段，检测和修复Bug。
  • 学生和教师：在学习编程或教授编程课程的过程中，Cursor可以作为一个强大的工具，帮助理解和实践编程概念。
  • 数据科学家和分析师：在进行数据分析和机器学习项目时，Cursor可以帮助快速编写和测试代码。
  • 自由职业者和创业者：需要快速开发软件原型或产品的个人，可以使用Cursor加速开发过程，例如出海工具站、公众号爆文爬虫、小红书图文等。

安装插件

2.1 安装中文插件

在Cursor窗口左侧点击Extensions按钮，在搜索框输入Chinese，然后按需安装/Install 简体或繁体中文，安装完成后点击Restart重启Cursor就可以切换中文啦。

2.2 安装编译插件——python版

还是Extensions位置，在搜索框输入python，然后和刚才安装Chinese插件的方法一样，当前是>3.7版本的。除了基础的python，还有很多python相关的插件，以及C、C++等语言，后面自行探索即可。

cursor设置

在cursor右上角点击设置按钮，然后在Features中把Composer设为enabled，这里编程会更加自动化；

在Beta中，把LONG CONTEXT CHAR（BETA）设为enabled，这样就可以支持200K的长文本啦。

如何更高效的使用cursor

程序功能性能要求描述

  • 在向 Cursor 提问时，尽量具体和明确，明确你需要的以及不需要的，罗列你需要的所有目标以及你想通过什么方式抵达目标；
  • 不提倡模糊的描述，比如：“我需要帮助写一个程序”、“帮我写一个爬虫”；
  • 直接描述这个功能例如：“我需要一个函数来计算斐波那契数列的第n项”；
  • 如何罗列目标和方式：帮我写一个爬虫，爬取公众号（链接）的文章，以及描述清楚要爬取的文章的特点，并且提供给AI想要以什么方式保存爬取的信息，如excel，excel需要存储那几列信息。

提供上下文的约束

  • 指定你正在使用的编程语言，例如：“我在使用Python。”
  • 如果适用，提及你正在使用的框架或库，例如：“我在使用Django框架。”
  • 描述你尝试解决的问题的背景，例如：“我正在开发一个Web应用，需要实现用户认证功能。”

提供代码片段

  • 提供已经在尝试的代码片段，不必完整，例如：
python
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
  • 解释你的代码尝试做什么，以及你遇到的问题。

错误调试需要详细描述error信息

  • 如果你在调试代码，提供具体的错误信息，例如：“当我运行这段代码时，它抛出了一个‘IndexError’。”
  • 描述你观察到的行为，例如：“程序在处理大于30的输入时会崩溃。”
  • 明确你期望的结果，例如：“我希望程序能够无错误地计算出斐波那契数列的第100项。”
  • 一般而言，将终端或网页上返回的error直接喂给AI效果更好。

以上仅为一个简单的科普演示，大家做爬虫、做工具站、科学计算、绘图等均可自己定制更高级的功能，也可以导入相关库以及使用conda管理python。
