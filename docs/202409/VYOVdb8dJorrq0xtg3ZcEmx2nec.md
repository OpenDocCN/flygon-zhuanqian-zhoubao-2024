# 体制内做公众号副业变现10w+的方法路径

# 一、个人背景介绍。

我本人的本职工作是一名国企金融从业人员。副业在2022年开启，因为未婚单身，又处在被催婚的年纪，所以身边的已婚未婚的都挺喜欢找我咨询情感问题，吐槽或者抱怨都有。

就经常在公众号写一些相亲日记一样的内容，后来自学了一些排版和写作方法，慢慢积累了几千粉丝，但我不懂得如何变现，也不知道有什么变现路径。

# 二、在生财遇到转折

因为机缘巧合2023年8月看到了一篇关于整理的生财有术“投资亏钱”的合集，找到了这个组织，添加了鱼丸的微信，加入了星球。跟着就参加了八月的航海，跟着公众号爆文航海的实战，学到了很多方式方法，实战结束后我原本流量主收益每个月仅有两三百提升至600元每个月，后来两千多每个月，现在每个月流量主收益平均每月1千多。

当然我不是做专业爆文的，当时我也只有一个账号，我个人认为公众号作为副业还是稳定发展比较好，追求热点和爆文只靠流量主收益，这个收入太过于单一了，而且流量也不是很稳定，需要多个账号一起跑，普通上班族把这个作为副业，很难批量化起号，和上一些高效率的批量化工具。

普通上班族每个月副业稳定月入5000+，我觉得是一个很不错的方式，一个账号稳定耐心的做下去不用两年绝对可以实现，但很多人也许坚持不到收益的时候就放弃了。

# 三、我的公众号收入模式介绍

## 流量主收益：

前面也说了我的一部分收入来源于，流量主收益，这块总的来说现在变现了2w多。

付费文章收益：因为我的账号粉丝比较垂直，全部都是内容涨粉，没有做过任何引流，粉丝粘性很强，都是25-35的情感粉，女粉占大多数。

我平时会写一些付费文章，和付费合集，目前的频率是每周一篇，价格有9.9，19.9还有29.9，根据我的创作难度和花费时间来定价，付费文章字数也比较多，干货也很多，也广受好评。平均每月付费文章收入1000元左右。

## 打赏收入：

打赏收入微不足道，每个月也有点，几十块到一百多不等。

## 私域转化收入：

做情感账号，粉丝也比较稳定，平均阅读量没有推荐的情况下每篇文章也有1500-2000的阅读量。所以做了情感咨询，按小时收费，每小时收费199，同时也推出了半年度咨询和年度咨询，费用均过千。这块总的收入没有计算过，有时候支付宝有时候微信收款，应该也有1w多收入。

这里说一下，做咨询是一件挺消耗能量的事情，情感咨询大多数都是挺复杂和负能量的，心态不好的还是别给别人做咨询了。

私域除了转化咨询服务，我个人也推出了自己写的脱单手册，今年6月份开始推出的，这块的收入已经超过3w元。

## 星球收入：

我一开始做了个女性成长社群，是免费的，但免费的活跃度很差劲，每次就我一个人在群里更新和分享资源，一些电子书和我自己购买的情感课程。

一百多人的社群活跃度非常差，可能免费的东西别人都不珍惜。我觉得没有什么价值感，加上主业也挺忙，副业刚顾得上，就不想运营社群了，就解散了，这个月刚推出付费星球，定价59，目前推出还没到一周，收益已经超过500元了，我个人觉得还可以。

我相信后期会越来越好的。

## 接广告收入：

目前公众号广告报价每条1200，陆陆续续也接了不少广告，之前是打算每周接一个的，但大多数广告主都不走互选，每次说保证安全，但大多数都会违规，所以大家做账号的，接广告要小心，别人说安全不一定是真的是，反正你违规了他们也没损失，不会管你账号死活的。

这块的接广收入预估也有1.5w+。

### 接广告注意事项：

1、避开互选接广告存在一定的风险，如果你的账号有稳定收入，不建议为了那点钱冒风险，虽然复制粘贴就一千多收入到账了。但长远来看不利于账号发展。

而且每次接广告也会影响你的账号流量，甚至多接几次就没有平台推荐了。

我接的广告基本有一半都会违规，可见风险很高。哪怕你已经删了那个广告推文，过几天腾讯还会判定的账号违规，一般违规原因都是过度营销，骚扰用户。

2、坚决不能接的广告，会违规甚至封号的：封建迷信算命的、医疗类的，减肥丰xiong黑五类的，代发文章特别是足球类（涉及du博）,金融投资下载软件类的，兼职副业类的（容易有诈骗）。

3、关于广告报价，我没看过有什么官方报价方式，我报价根据粉丝量的10%，或者按照自己的文章平均阅读量0.5-1元/个来报，你也可以试探广告商你的报价区间在哪个位置，比如你的粉丝1000，你可以报1000，1200，1500，这样试探，看最后成交在哪个区间，建议往高了报，反正每次都会被砍价的。

# 四、收入护城河

我个人认为每个人可以建立属于自己的收入护城河。

## 1.保底收入：

比如我的本职工作是很稳定，保底收入，覆盖我的贷款和生活支出没有问题，没啥压力。

## 2.发展收入：

比如我的公众号收入就是我的发展收入，这部分收入会有轻微浮动，总的来说算稳定；

可以完全存起来，作为自己的保障和对自己的投资发展基金，甚至抗风险资金。

当然没找到新项目有新收入的情况下，可以对原本有盈利的项目做大做强，做精做优，进一步增加收入。就像亦仁说的任何一个小项目只要你跑通了最小的mvp,都可以放大，能赚到1块钱的项目就可以赚到100w,关键在怎么放大。我非常认可这个理论，这个我自己也在探索中。

## 3.开发新收入：

如果你有一份稳定工作和收入，还有一个已经稳定运营的项目带给你增值收入，还有余力的情况下，可以适当拓展一下新的项目，比如小红书店铺很稳定出单了，可以拓展一下视频类的。

# 五、做账号建议

## 上班族副业建议

我建议打算做副业的上班族尽量集中时间和精力去做一个人可以操作的项目，特别是结合你的工作和兴趣爱好，这样不耽误主业，也不耽误生活，轻松赚钱。

像我之前尝试过写热点，追流量，但我不喜欢关注那些民生或者八卦，写的也不是很顺手，但写情感文章得心应手，做自己擅长的事不费力又能赚钱，就轻松很多。写一篇热点文的时间可以产出两篇情感文，那我宁愿做擅长的事。

我主要说的是做公众号变现的路径和经验分享，我的方法比较适合有耐心能坚持的人，可能不太适合赚快钱的。

做账号很慢，需要认真做内容，我的号也是日更状态而且都是手写。

如果你是上班族，公众号可以算一个不错的副业，你也可以学习我的收入模式。上班别人摸鱼你可以利用这个时间搜集素材写稿。

## 案例参考

你做的是职场类的账号，你可以引流私域做职业咨询，面试辅导，简历修改这样的服务，根据耗时难度定价，甚至售卖一些ppt模板，职场小技巧，和领导同事打交道话术这类书籍。

如果你做的是穿搭账号，可以引流私域，做穿搭指导服务，形象改造，甚至做一些饰品珠串的售卖转化。

如果你平时喜欢国学玄学占星，你可以引流私域做占星和看八字的转化，也可以售卖一些水晶，有趋吉避凶寓意的手串。

任何类型和赛道的账号，都可以找到私域变现的方式方法。

如果是上班族做副业，不建议免费引流，泛粉，白嫖粉太多，不仅会耽误你很多时间，转化困难还影响你心情，因为一些人的三观和厚脸皮，让你震惊。

一个人做副业，精力有限就服务好那些真诚认可你的粉丝，付费意愿强的，既然你的私域是为了转化变现，可以从低价引流开始，比如9.9的产品，起码别人有付费意识，不是纯白嫖。

经常有人会觉得我的号是一个团队或者公司运营的，有的人加过来甚至会当我是客服，趾高气扬的，给我整无语了。

还有那种加微信不备注说话也不回复的，建议直接删除，不然后续你也会发现这种人也很无耻，一个人做私域就做精准人群，没必要浪费时间搞泛粉，转化概率低难度大。
