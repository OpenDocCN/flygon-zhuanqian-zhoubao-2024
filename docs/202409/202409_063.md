# 公众号隐藏新功能：怎么无限注册公众号？

## (72 赞)公众号隐藏新功能：怎么无限注册公众号？

作者： 格子

日期：2024-09-04

**各位圈友好，我是格子，聚焦于公众号流量主和小红书商单。**

这两天意外发现公众号一个隐藏的新功能：**居然可以无限注册公众号了！**

我们之前是一个身份证只能注册一个账号，你再多的微信号手机号都没用，但是这次，只要一个手机号居然就能注册新号了。

![](img/b8d4331788cbcc25dc63faee66d7b012.png)

**就我测试下来的，一个手机号只能注册一个未实名的公众号，但是也有同行说可以注册 1-12 个号不等！具体到底多少，大家去测试了的可以评论区反馈下。**

当然了，这种注册的号也是有一定的局限性，因为实名制度上还是只能一个身份证实名一个号，那多出来的公众号是没法实名的。

目前不能实名的账号不能开通赞赏、流量主等功能，那就意味着这样的账号想要靠流量主挣钱不可能。

**但是呢，这号能正常发布文章，可以矩阵化做 ip 引流到私域做转化。**

这个是目前我能想到的变现途径，当然大家有更好的思路，欢迎补充。

不过有机会多注册号，还是可以先注册放那里的，毕竟这应该是腾讯新开放的功能，说不定过段时间，会优化，也许又注册不了这么多账号了！

另外就是，先注册好放那里，也许未来平台放开实名认证，或者说改规则，依然还是能做公众号流量主变现的。

**一个手机号能注册一个未实名的公众号，那想要多公众号就多个手机号即可，另外就是一个微信只能运营 5 个公众号，如果这个微信满了，可以换一个微信哦。**

![](img/95d3888e0cb68f6abc5cbe0122d2e8aa.png)

**接下来就写个保姆级的操作流程，看看怎么无限注册公众号。**

首先我们要先去手机的应用市场下载订阅号助手，这个手机要登录一个实名认证号的微信哈。

以我这个手机为例，我本身是已经注册了一个公众号的，我先把公众号退出，然后点击微信登录，右上角会出现注册订阅号：

![](img/f932710a936e80f7dbb04821edb119c7.png)

如果是用当下这个手机注册就选择快速注册，如果是其他手机号注册，就点其他的，接下来就是修改信息：

![](img/2e25c8af76efc3418f19de149c306945.png)

已经注册完成开始创作了，这个号就是全新的：

![](img/49d728370d390a3d93905d422e974ba6.png)

接下来我们再多注册几个，还是一样的先退出登录，重复前面的流程，**会显示一个手机号只能注册一个未实名的，那我们换个手机号进行就可以了**，看，我一下子多注册了两个：

![](img/d377ae9004c1c473ceff18a894143a77.png)

像我本身就做小红书，手机号非常多，起码有 20 多个，那我可以注册 20 多个未实名的公众号了。

**注意：注册的时候，如果手机上登录了一个未实名的微信，是不能注册公众号的，系统会要求你必须先实名微信再注册，所以我们注册新公众号的时候，一定要登录实名的微信哦。**

**另外还有一个弊端就是，如果你是用你实名微信注册的公众号，是不能再认证别人的身份证的，你要是想实名认证的话，必须把前面一个实名的公众号注销掉！**

![](img/227e313480ac102e9f1d1e25c7c679ac.png)

**对了关于这个点，刚刚测试了下，其实也有办法，就是把这个新注册的公众号的管理员换成别人，再实名验证即可。比如我们爸妈微信下没有注册公众号，可以登录公众号网页版，把管理员换成爸妈，实名验证他们的就行了！**

不过现在能注册，就多注册几个放那，看看后面平台是不是会改规则，说不定哪天就用上了，尤其是做 ip 的号主，本来也不靠流量主变现，直接矩阵化 ip 去获客也不错！

最近，公众号规则不断调整，比如去年推出小绿书，今年 ip 内容可以进流量池，前段时间说是可以推广短剧了，都是平台在释放他们看重公众号的信号。

**如果说什么项目能长期做，我想一定是公众号，因为它背靠微信，只要微信在，公众号就不大可能砍掉，而其他项目、平台都有可能会倒，微信接下来很长一段时间都不会倒。**

对于做公众号的人来说，平台的每一次调整其实都是利好，所以加油干吧。
